Платформа

Система A2v10 реалізована з використанням технологічного стека .NET Framework корпорації Microsoft.

В якості СУБД використовується Microsoft SQL-Server або хмарний сервіс Azure SQL Database (service).

Серверна частина системи реалізована як web-застосунок на платформі Microsoft ASP.NET.MVC. Підтримується розгортання системи в хмарному сервісі Microsoft Azure як Web App Service.

Для реализації клиєнтської частини web-програми використовується JavaScript фреймворк з відкритим сирцевим кодом Vue.js. Клієнтська частина реалізована як SPA-додаток (Single Page Application). Підтримуються всі сучасні браузери (Edge, Chrome, FireFox, Opera, Safari і т.п.). Браузер Internet Explorer НЕ підтримується.

Для побудови звітів і друкованих форм застосовується звітна система Stimulsoft Reports.Web.
