Платформа

Система A2v10 реализована з використанням технологического стека .NET Framework корпорации Microsoft.

В качестве СУБД використовується Microsoft SQL-Server або облачный сервис Azure SQL Database (service).

Серверна часть системы реализована як web-приложение на платформе Microsoft ASP.NET.MVC. Підтримується развертываение системы в облачном сервисе Microsoft Azure як Web App Service.

Для реализации клиентской части web-програми використовується JavaScript фреймворк с открытым исходным кодом Vue.js. Клієнтская часть реализована як SPA-приложение (Single Page Application). Поддерживаются всі современные браузеры (Edge, Chrome, FireFox, Safari). Браузер Internet Expolorer НЕ підтримуєся.

Для построения звітів і печатных форм применяется звітная система Stimulsoft Reports.Web.
