Свойства и методы корневого объекта

Модель на клиенте

Корневой объект расширяется следующими служебными свойствами и методами:

Свойства

readonly $ready: boolean

true, если в данный момент НЕ выполняется никаких запросов к серверу. Может использоваться для запрета/разрешения элементов интерфейса при выполнении серверных команд. 

readonly $template: Template

Ссылка на объект шаблона (Template).

Методы

$defer(handler:()=>any): void

Выполняет функцию (handler) после завершения текущего цикла событий.
