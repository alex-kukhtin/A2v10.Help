Властивості і методи кореневого об'єкта

Модель на клієнті

Кореневий об`єкт розширюється наступними службовими властивостями і методами:

Властивості

readonly $ready: boolean

true, якщо в даний момент НЕ виконується жодних запитів до серверу. Може використовуватися для заборони/дозволу елементів інтерфейса при виконанні серверних команд. 

readonly $template: Template

Посилання на об`єкт шаблона (Template).

Методи

$defer(handler:()=>any): void

Виконує функцію (handler) після завершения поточного циклу подій. 

$revalidate(elem: IElement, rule: string): void

Примусово викликає асінхронні валідатори для элементу.

Аргументи:

  • elem - елемент, для якого має бути виконано валідатор.
  • rule - текстове ім'я валідатора.
