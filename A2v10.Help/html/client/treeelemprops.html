Властивості и методы елементу дерева

Модель на клиенте

Каждый елемент дерева расширяется следующими служебными властивостіми и методами:

Властивості

readonly $expanded: boolean

Елемент развернут.

Методы

Опис

Разворачивает елемент дерева (якщо он был свернут). Роботает как для статических, так и для динамических деревьев.

Возвращает:

  • Промис со ссылкой на массив развернутых дочерних елементів.

Методы

Аргументы:

  • path - массив, который определяет путь в дереве, начиная с корня. Массив може содержать любые елементи.
  • predicate - функция, яка буде викликатися для поиска заданного елементу в дереве. Должна повернути true, якщо елемент дерева item соответствует елементу в массиве val. Аргументы предиката:
    • item: ITreeElement - елемент дерева.
    • val: any - елемент з масива path.

Опис

Выделяет в дереве елемент по заданному пути. Має сенс тільки для динамических деревьев. В процессе роботи метод разворачивает все промежуточные узлы.

Возвращает:

  • Промис со ссылкой на выделенный елемент дерева.

Более подробно работа с деревьями описана в разделе об иерархических моделях даних.
