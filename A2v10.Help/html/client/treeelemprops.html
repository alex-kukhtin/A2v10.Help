Свойства и методы элемента дерева

Модель на клиенте

Каждый элемент дерева расширяется следующими служебными свойствами и методами:

Свойства

readonly $expanded: boolean

Элемент развернут.

Методы

Описание

Разворачивает элемент дерева (если он был свернут). Работает как для статических, так и для динамических деревьев.

Возвращает:

  • Промис со ссылкой на массив развернутых дочерних элементов.

Методы

Аргументы:

  • path - массив, который определяет путь в дереве, начиная с корня. Массив может содержать любые элементы.
  • predicate - функция, которая будет вызываться для поиска заданного элемента в дереве. Должна вернуть true, если элемент дерева item соответствует элементу в массиве val. Аргументы предиката:
    • item: ITreeElement - элемент дерева.
    • val: any - элемент из массива path.

Описание

Выделяет в дереве элемент по заданному пути. Имеет смысл только для динамических деревьев. В процессе работы метод разворачивает все промежуточные узлы.

Возвращает:

  • Промис со ссылкой на выделенный элемент дерева.

Более подробно работа с деревьями описана в разделе об иерархических моделях данных.
