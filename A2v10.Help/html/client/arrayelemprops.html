Свойства и методы элемента массива

Модель на клиенте

Каждый элемент массива расширяется следующими служебными свойствами и методами:

Свойства

readonly $parent: IElementArray

Ссылка на родительский массив (IElementArray). 

$selected: boolean

Элемент выделен. 

$checked: boolean

Элемент отмечен. Коллекцию отмеченных элементов можно получить с помощью свойства $checked родительского массива.

Методы

$remove(): void

Удаляет элемент из родительского массива.
