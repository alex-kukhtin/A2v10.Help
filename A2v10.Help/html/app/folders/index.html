Специальные папки приложения

Папка приложения может включать специальные папки. Их имена начинаются с символа "_".

Специальные папки

  • _assets - общие элементы приложения.
  • _layout - переопределение основных страниц приложения.
  • _emails - шаблоны писем.
  • _files - статические файлы для загрузки.
