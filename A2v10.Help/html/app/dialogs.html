Диалоги (dialogs)

model.json

Объект dialogs описывает диалоги, возможные в текущей конечной точке. Имена свойств объекта представляют собой имена диалогов. Каждое свойство представляет собой объект следующей структуры: 

{
  "source": "",
  "schema": "",
  "model": "",
  "index": true|false,
  "copy": true|false,
  "template": "",
  "view": "",
  "parameters": {}
}

Свойства объекта диалога

Имя Тип Описание
source String Необязательно. Источник данных. Если не указан, берется из родительского элемента
schema String Необязательно. Имя sql-схемы. Если не указана, берется из родительского элемента
model String Имя модели. Если не указано, берется из родительского элемента
index Boolean Показывает ли диалог индексную модель. Для индексных моделей выполняется хранимая процедура с суффиксом '.Index', для обычных - с суффиксом '.Load'
copy Boolean Является ли диалог диалогом копирования. В этом случае выполняется хранимая процедура с суффиксом '.Copy'. Копирование в индексных диалогах не поддерживается
template String Путь к файлу шаблона для диалога. Подробнее о шаблонах
view String Путь к файлу представления для диалога
parameters Object Объект, статические параметры, которые передаются в хранимую процедуру загрузки модели. Имя свойства является именем параметра, значение - значением параметра
