Всплывающие окна (popups)

model.json

Всплывающее окно может как встраиваться в разметку, так и иметь внешнее загружаемое содержимое. Во втором случае оно имеет собственную модель и собственный шаблон.

Объект popups описывает всплывающие окна, возможные в текущей конечной точке. Имена свойств объекта представляют собой имена окон. Каждое свойство представляет собой объект следующей структуры:

Свойства объекта для всплывающего окна

Имя Тип Описание
source string Необязательно. Источник данных. Если не указан, берется из родительского элемента.
schema string Необязательно. Имя sql-схемы. Если не указана, берется из родительского элемента.
model string Имя модели. Если не указано, берется из родительского элемента.
index boolean Показывает ли окно индексную модель. Для индексных моделей выполняется хранимая процедура с суффиксом '.Index', для обычных - с суффиксом '.Load'.
template string Путь к файлу шаблона для окна. Подробнее о шаблонах.
view string Путь к файлу представления для всплывающего окна. Как правило, корневым элементом разметки окна будет элемент Popup.
parameters object Объект, статические параметры, которые передаются в хранимую процедуру загрузки модели. Имя свойства является именем параметра, значение - значением параметра.
