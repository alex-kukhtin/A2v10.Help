Спливаючі вікна (popups)

model.json

Спливаюче вікно може як вбудовуватися в розмітку, так і мати зовнішній вміст, що завантажується. В останньому випадку воно має власну модель і власний шаблон.

Об'єкт popups описує спливаючі вікна, можливі в поточній кінцевій точці. Імена властивостей об'єкта являють собою імена вікон. Кожна властивість являє собою об`єкт наступної структури:

Властивості об'єкта для спливаючого вікна

Им'я Тип Опис
source string Не обов'язково. Джерело даних. Якщо не указан, береться з батьківського елементу.
schema string Не обов'язково. Им'я sql-схеми. Якщо не указана, береться з батьківського елементу.
model string Им'я моделі. Якщо не вказано, береться з батьківського елементу.
index boolean Чи показує окно індексну модель. Для індексних моделей виконується збережена процедура з суфіксом '.Index', для звичайних - з суфіксом '.Load'.
template string Шлях до файлу шаблона для вікна. Детальніше про шаблони.
view string Шлях до файлу виду для спливаючого вікна. Як правило, кореневим елементом розмітки вікна буде елемент Popup.
parameters object Об'єкт, статичні параметри, які передаються в збережену процедуру завантаження моделі. Им'я властивості є ім'ям параметра, значення - значенням параметра.
