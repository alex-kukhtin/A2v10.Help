Елемент MessageReport

Надсилання сообщений

Представляет собой звіт, который буде сформирован і вложен в повідомлення электронной почты.

Властивість содержимого: Model

Властивості

Им'я Тип Опис
Report String Всегда привязка з моделі даних. Поле основної моделі (не моделі звіта!), которое должно містити текстовое описание звіта (тип поля - nvarchar(max)).
Name String Им'я звіта (вложения). Якщо не вказано, буде иметь значення "report". Кроме того, якщо не вказано розширення, то автоматически добавляется розширення .pdf.
Model MessageReportModel Опис моделі даних для построения звіта.
