Элемент MessageReport

Отправка сообщений

Представляет собой отчет, который будет сформирован и вложен в сообщение электронной почты.

Свойство содержимого: Model

Свойства

Имя Тип Описание
Report String Всегда привязка из модели данных. Поле основной модели (не модели отчета!), которое должно содержать текстовое описание отчета (тип поля - nvarchar(max)).
Name String Имя отчета (вложения). Если не указано, будет иметь значение "report". Кроме того, если не указано расширение, то автоматически добавляется расширение .pdf.
Model MessageReportModel Описание модели данных для построения отчета.
