Елемент MessageReport

Надсилання повідомлень

Являє собою звіт, який буде сформирований і вкладений в повідомлення електронної пошти.

Властивість вмісту: Model

Властивості

Им'я Тип Опис
Report String Завжди прив'язка з моделі даних. Поле основної моделі (не моделі звіта!), яке повинно містити текстовий опис звіту (тип поля - nvarchar(max)).
Name String Им'я звіту (вкладення). Якщо не вказано, буде мати значення "report". Крім того, якщо не вказано розширення, то автоматично додається розширення .pdf.
Model MessageReportModel Опис моделі даних для побудови звіта.
