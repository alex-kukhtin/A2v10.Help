Елемент MessageReportModel

Надсилання повідомлень

Представляє опис моделі даних для побудови звіта для відправки з повідомленням.

Властивість вмісту: Parameters

Властивості

Им'я Тип Опис
Source String Джерело даних для завантаження моделі.
Schema String Схема для завантаження моделі.
Model String Им'я моделі. Для завантаження моделі буде викликана процедура [{Schema}].[{Model}.Load] в джерелі даних {Source}.
Parameters Словник MessageReportParam Параметри для моделі звіта.
