Элемент MessageReportModel

Отправка сообщений

Представляет описание модели данных для построения отчета для отправки с сообщением.

Свойство содержимого: Parameters

Свойства

Имя Тип Описание
Source String Источник данных для загрузки модели.
Schema String Схема для загрузки модели.
Model String Имя модели. Для загрузки модели будет вызвана процедура [{Schema}].[{Model}.Load] в источнике данных {Source}.
Parameters Словарь MessageReportParam Параметры для модели отчета.
