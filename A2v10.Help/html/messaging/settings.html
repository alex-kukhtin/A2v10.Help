Налаштування може виконуватися як стандартними средствами .NET (<mailSettings>), так і елементом mailSettings в разделе <appSettings>.

Формат json-текста для <appSettings> полностью повторяет стандартний формат настроек .NET.

Зверніть увагу!, Якщо задан метод доставки 'SpecifiedPickupDirectory' (використовується для тестирования, без реальной отправки повідомлення), то обов'язково потрібно задать папку, куда будуть помещаться тестовые повідомлення (pickupDirectoryLocation).
