Привязки

Для того, чтобы в свойствах можно было использовать значения "снаружи", используются привязки. Привязка это наименование свойства, заключенное в двойные квадратные, фигурные или круглые скобки.

Привязка Тип Описание
[[param]] Параметры команды Значение из параметров команды. Обратите внимание, что это не свойство parameters, а параметры, с которыми команда вызвана с клиента.
{{path}} Свойства модели Значение из модели. Path это путь в модели данных.
((value)) Окружение Значение внешнего окружения. Возможны следующие значения:
  • ((Host)) - имя хоста из файла конфигурации. Используется для построения ссылок в теле письма.
  • ((UserHost)) - имя хоста пользовательского приложения (сайта) из файла конфигурации. Используется для построения ссылок в теле письма..
  • ((ProcessId)) - только для сообщений из бизнес-процессов. Идентификатор текущего процесса.
  • ((ProcessId)) - только для сообщений из бизнес-процессов. Идентификатор текущей задачи.
