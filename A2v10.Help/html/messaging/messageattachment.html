Элемент MessageAttachment

Отправка сообщений

Представляет собой вложение в сообщение электронной почты.

Свойства

Имя Тип Описание
Data Stream Всегда привязка из модели данных. Поле модели, к которому привязывается это свойство должно быть типа массив байтов (vabinary).
Name String Имя вложения. Чтобы клиент смог правильно распознать тип файла вложения, добавьте к имени расширение (например .pdf).
Mime String Mime-тип содержимого вложения.
