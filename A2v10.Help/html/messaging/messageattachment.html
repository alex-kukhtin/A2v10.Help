Елемент MessageAttachment

Надсилання сообщений

Представляет собой вложение в повідомлення электронной почты.

Властивості

Им'я Тип Опис
Data Stream Всегда привязка з моделі даних. Поле моделі, к которому привязывается це властивість повинно бути типа масив байтов (vabinary).
Name String Им'я вложения. Чтобы клиент смог правильно распознать тип файла вложения, добавьте к имени розширення (наприклад .pdf).
Mime String Mime-тип содержимого вложения.
