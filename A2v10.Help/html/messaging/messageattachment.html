Елемент MessageAttachment

Надсилання повідомлень

Являє собою вкладення в повідомлення електронної пошти.

Властивості

Ім'я Тип Опис
Data Stream Завжди прив'язка з моделі даних. Поле моделі, до якого прив'язується ця властивість повинно бути типу масив байтів (vabinary).
Name String Ім'я вкладення. Щоб клієнт зміг правильно розпізнати тип файлу вкладення, додайте до імені розширення (наприклад .pdf).
Mime String Mime-тип вмісту вкладення.
