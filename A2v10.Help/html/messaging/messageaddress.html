Элемент MessageAddress

Отправка сообщений

Представляет адрес для сообщений.

Конвертер: поддерживается преобразование из строки текста как в одно сообщение, так и в коллекцию сообщений. Строка текста в этом случае считается адресом (свойство Address).

Свойства

Имя Тип Описание
Address String Адрес сообщения.
DisplayName String Отображаемое имя.
Phone String Номер телефона. Для е-mail не используется.
