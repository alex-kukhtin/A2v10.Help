Елемент MessageAddress

Надсилання сообщений

Представляет адрес для сообщений.

Конвертер: підтримуєся преобразование з строки текста як в одно повідомлення, так і в коллекцию сообщений. Рядок текста в этом случае вважається адресом (властивість Address).

Властивості

Им'я Тип Опис
Address String Адрес повідомлення.
DisplayName String Отображаемое имя.
Phone String Номер телефона. Для е-mail не використовується.
