Елемент MessageAddress

Надсилання повідомлень

Представляє адресу для повідомлень.

Конвертер: підтримується перетворення з рядку текста як в одне повідомлення, так і в коллекцию повідомлень. Рядок тексту в цьому випадку вважається адресою (властивість Address).

Властивості

Им'я Тип Опис
Address String Адреса повідомлення.
DisplayName String Ім'я, що відображається.
Phone String Номер телефону. Для е-mail не використовується.
