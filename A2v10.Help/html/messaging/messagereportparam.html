Елемент MessageReportParam

Надсилання повідомлень

Являє собою параметр для моделі даних звіту.

Колекція параметрів являє собою словник з текстовим ключем. Им'я параметру задається атрибутом x:Key

Властивості

Им'я Тип Опис
Value Object Зазвичай прив'язка. Значення параметру. Зверніть увагу, що прив'язка працює відносно основної моделі повідомлення (не моделі звіта!).
