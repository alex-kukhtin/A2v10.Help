Элемент MessageReportParam

Отправка сообщений

Представляет собой параметр для модели данных отчета.

Коллекция параметров представляет собой словарь с текстовым ключом. Имя параметра задается атрибутом x:Key

Свойства

Имя Тип Описание
Value Object Обычно привязка. Значение параметра. Обратите внимание, что привязка работает относительно основоной модели сообщения (не модели отчета!).
