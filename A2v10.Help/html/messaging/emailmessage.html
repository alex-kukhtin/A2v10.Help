Елемент EmailMessage

Надсилання сообщений

Представляет собой сообщение электронной почты.

Наследует: TemplatedMessage

Властивість содержимого: Body

Властивості

Им'я Тип Опис
Body String Тіло повідомлення. Якщо задано властивість BodyTemplate, то значення властивості игнорируется.
Subject String Тема повідомлення.
BodyTemplate String Им'я файла с шаблоном повідомлення.
From MessageAddress Отправитель. Може бути пустым. Якщо не вказано, використовується з настроек почты.
To Коллекция MessageAddress Получатели повідомлення.
СС Коллекция MessageAddress Копия (Carbon Copy).
Bcc Коллекция MessageAddress Скрытая копия (Blind Carbon Copy).
Attachments Коллекция MessageAttachment Вложения.
Reports Коллекция MessageReport Звіти.
