Элемент EmailMessage

Отправка сообщений

Представляет собой сообщение электронной почты.

Наследует: TemplatedMessage

Свойство содержимого: Body

Свойства

Имя Тип Описание
Body String Тело сообщения. Если задано свойство BodyTemplate, то значение свойства игнорируется.
Subject String Тема сообщения.
BodyTemplate String Имя файла с шаблоном сообщения.
From MessageAddress Отправитель. Может быть пустым. Если не указано, используется из настроек почты.
To Коллекция MessageAddress Получатели сообщения.
СС Коллекция MessageAddress Копия (Carbon Copy).
Bcc Коллекция MessageAddress Скрытая копия (Blind Carbon Copy).
Attachments Коллекция MessageAttachment Вложения.
Reports Коллекция MessageReport Отчеты.
Свойства базового элемента

Обратите внимание! Все текстовые свойства поддерживают привязки.
