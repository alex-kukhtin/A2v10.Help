Шаблоны

Шаблон представляет собой объект JavaScript, который определяет поведение модели данных. Общая структура шаблона:

Свойства объекта шаблона

Имя Тип Описание
options options Объект, описывающий опции настройки шаблона.
defaults defaults Объект, описывающий правила инициализации модели.
properties properties Объект, описывающий дополнительные свойства, которые будут добавлены в модель.
validators validators Объект, описывающий правила проверки достоверности для элементов модели.
events events Объект, описывающий обработчики событий модели.
commands commands Объект, описывающий команды для модели.
delegates delegates Объект, описывающий делегаты (функции обратного вызова, callback functions), необходимые для работы модели.
