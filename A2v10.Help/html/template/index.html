Шаблони

Шаблон являє собою об`єкт JavaScript, который определяет поведение моделі даних. Общая структура шаблона:

Властивості об'єкта шаблона

Им'я Тип Опис
options options Об'єкт, що описує опции настройки шаблона.
defaults defaults Об'єкт, що описує правила инициализации моделі.
properties properties Об'єкт, що описує додаткові властивості, які будуть додані в модель.
validators validators Об'єкт, що описує правила перевірки достоверности для елементів моделі.
events events Об'єкт, що описує обработчики событий моделі.
commands commands Об'єкт, що описує команди для моделі.
delegates delegates Об'єкт, що описує делегаты (функции обратного вызова, callback functions), необходимые для роботи моделі.
