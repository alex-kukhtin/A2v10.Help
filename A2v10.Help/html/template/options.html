Шаблони. Опции

Опции шаблона задают поведение всей моделі.

Синтаксис

Властивості об'єкта options

  • noDirty - не взводить флаг изменений $isDirty при изменениях моделі даних.
  • bindOnce - масив рядків, определяющий какие корневые властивості моделі будуть связываться однократно.
  • persistSelect - масив рядків, определяющий масиви моделі, для которых потрібно сохранять позицию виділеного елементу в процесі сохранения або перезавантаження моделі.
  • skipDirty - масив рядків, определяющий пути к властивостім моделі, при изменении которых не потрібно взводить флаг изменений $isDirty.

Подробнее

Флаг $isDirty відповідає за вывод повідомлення о необхідности зберігти модель після внесения изменений, а також за поведение некоторых елементів керування. Наприклад кнопка, связанная с командой Save буде автоматично разрешаться після изменения моделі. Якщо опция noDirty установлена, то флаг $isDirty взводиться не буде.

Властивість bindOnce дозволяє визначити корневые властивості моделі, які будуть связываться с даними тільки при первой завантаженні моделі. При дальнейшей перезавантаженні (наприклад командой Reload або після сохранения моделі) переприв`язки цього властивості не буде. Частіше за всі це використовується для прив`язки елементів справочников.

Зверніть увагу! В масиве bindOnce можна указывать тільки корневые властивості моделі (то є тільки те, в іменах которых нет точки).

Властивість persistSelect дозволяє визначити масиви в моделі, для которых буде сохраняться позиция виділеного елементу ($selected). Позиция буде сохраняться при збереженні моделі (наприклад командой Save) и при ее перезавантаженні (наприклад командой Reload).
