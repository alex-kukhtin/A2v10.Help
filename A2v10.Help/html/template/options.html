Шаблони. Опции

Опции шаблона задают поведение всей моделі.

Синтаксис

Властивості об'єкта options

  • noDirty - не взводить флаг изменений $isDirty при изменениях моделі даних.
  • bindOnce - масив рядків, определяющий какие корневые властивості моделі будуть связываться однократно.
  • persistSelect - масив рядків, определяющий масивы моделі, для которых потрібно сохранять позицию виділеного елементу в процессе сохранения або перезавантаження моделі.
  • skipDirty - масив рядків, определяющий пути к властивостім моделі, при изменении которых не потрібно взводить флаг изменений $isDirty.

Подробнее

Флаг $isDirty відповідає за вывод повідомлення о необходимости зберігти модель після внесения изменений, а також за поведение некоторых елементів керування. Например кнопка, связанная с командой Save буде автоматически разрешаться після изменения моделі. Якщо опция noDirty установлена, то флаг $isDirty взводиться не буде.

Властивість bindOnce дозволяє визначити корневые властивості моделі, які будуть связываться с данными тільки при первой завантаженні моделі. При дальнейшей перезавантаженні (наприклад командой Reload або після сохранения моделі) перепривязки цього властивості не буде. Частіше за всі це використовується для привязки елементів справочников.

Зверніть увагу! В масиве bindOnce можна указывать тільки корневые властивості моделі (то есть тільки те, в іменах которых нет точки).

Властивість persistSelect дозволяє визначити масивы в моделі, для которых буде сохраняться позиция виділеного елементу ($selected). Позиция буде сохраняться при сохранении моделі (наприклад командой Save) и при ее перезавантаженні (наприклад командой Reload).
