Шаблони

Раздел defaults дозволяє установить начальные значення для новой моделі. Новой вважається модель с нулевым (пустым) идентификатором главного об'єкта.

Об'єкт, що описує начальные значення являє собою обычный об`єкт JavaScript. Імена властивостей представляют собой шлях до елементу даних в моделі (к тому властивості, значення якого надо установить).
