Шаблоны

Раздел defaults позволяет установить начальные значения для новой модели. Новой считается модель с нулевым (пустым) идентификатором главного объекта.

Объект, описывающий начальные значения представляет собой обычный объект JavaScript. Имена свойств представляют собой путь к элементу данных в модели (к тому свойству, значение которого надо установить).
