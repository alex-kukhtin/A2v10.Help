Платформа А2v10

Платформа А2v10 представляет собой инструментальное средство, предназначенное для разработки прикладных решений, связанных с автоматизацией учета и управления предприятием.

Справочная система А2v10 рассчитана на тех, кто разрабатывает, устанавливает, настраивает и обслуживает прикладные решения, построенные на этой программной платформе. Обратите внимание, что в этой справочной системе НЕ содержится информация для конечных пользователей.

Справочная система А2v10 включает:

  • Описание принципов построения платформы.
  • Описание объектной модели.
  • Справочная информация о свойствах/методах/событиях.
  • Описание правил построения структуры базы данных.
