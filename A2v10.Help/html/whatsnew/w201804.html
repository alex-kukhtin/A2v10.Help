Выпуск 3 - версии 7150-7172 (апрель 2018)

Все выпуски| « Предыдущий выпуск| Следующий выпуск »

Общие

  • Добавлена возможность изменения пароля самим пользователем.
  • Добавлена возможность загружать пользовательские стили.

Модель данных

  • В метаданные модели добавлена длина текстовых полей. В связанных элементах управления эта длина учитывается автоматически.

Общие модули

  • В модуль std:utils добавлен метод fromJson.
  • В объект date модуля std:utils добавлен метод diff.

Программирование клиента

  • Для всех массивов данных добавлено свойство $selectedIndex.

Xaml

  • Добавлен элемент Image.
  • В элемент TextBox добавлено свойство UpdateTrigger.
  • В элемент DatePicker добавлено свойство Direction.
  • В элемент DataGridColumn добавлено свойство If.
  • Исправлена ошибка, из-за которой неправильно работало свойство Show элемента Tab (в случае связанного свойства).
  • В элемент Block добавлены свойства Height, Scroll, Border, Align.
  • В Bind добавлены свойства Mask, UpdateAfter.
  • В элемент Selector добавлены свойства ItemsPanel, PanelPlacement.
