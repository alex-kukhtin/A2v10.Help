Выпуск 6 - версии 7237-7264 (июль 2018)

Все выпуски| « Предыдущий выпуск| Следующий выпуск »

Исправлено

  • Исправлена ошибка, из-за которой для команды Save в привязке BindCmd не работало свойство ValidRequired.
  • Теперь при изменении свойств объекта Query перезаполнение модели происходит автоматически.

Общие

  • Добавлена возможность работы с загружаемыми файлами формата *.xlsx.
  • Добавлена возможность работы с REST api.

Xaml

  • В элемент Radio добавлено свойство Style. Теперь можно отображать радиокнопки в виде флажков.
  • В элемент TableColumn добавлено свойство If.
  • В элемент Popover добавлены свойства Width, Top.
  • В привязку Bind добавлено свойство NegativeRed.
  • В элемент Partial добавлено свойство Background.
  • В элемент Control (а значит во все элементы управления) добавлены свойства Popover и TestId.
