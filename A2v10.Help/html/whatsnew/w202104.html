Выпуск 32 - версии 7751-7770 (Апрель 2021)

Xaml

  • В элемент DataGrid добавлено свойство AutoSelect.
  • В элемент TextBox добавлено свойство ShowClear.
  • В элемент List добавлено свойство BorderStyle.

Программирование клиента

  • В объект валидатора templateValidatorObj добавлено свойство applyIf, которое дает возможность задавать функцию, указывающую нужно ли использовать валидатор.

Прикладные приложения

  • В работу с файлами добавлена возможность указания локали разбираемого файла. Управляется свойством locale.
  • В контроллер IController добавлен метод $focus.

Справка

  • Добавлено описание элементов Partial, FlexList, FileImage.
  • Добавлено описание работы с изображениями в моделях данных.
