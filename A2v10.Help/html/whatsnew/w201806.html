Выпуск 5 - версии 7200-7236 (июнь 2018)

Все выпуски| « Предыдущий выпуск| Следующий выпуск »

Исправлено

  • Исправлена ошибка, из-за которой для команды Select в привязке BindCmd неправильно работало свойство ValidRequired.
  • Исправлена ошибка, из-за которой не работало свойство Export в привязке BindCmd.
  • Исправлена ошибка, из-за которой не рабала привязка Value в объекте FilterItem.

Общие

  • При регистрации пользователей добавлено протоколирование отправки e-mail.
  • Добавлен механизм формирования xml-файлов по заданной схеме.
  • В файле app.json теперь можно задавать свойства для панели "Обратная связь".

Файлы приложения

  • В разделе reports файла model.json свойство name теперь может содержать макросы для доступа к свойствам модели данных ({{...}}).

Шаблоны

  • Добавлено событие Model.saved.
  • В события *.change добавлен аргумент oldValue (предыдущее значение).
  • Валидаторы теперь могут быть просто функциями.

Xaml

  • Добавлен элемент GridGroup.
  • В элемент List добавлены свойства Select и Height.
  • В элемент ListItem добавлено свойство Footer.
  • В элемент Block добавлено свойство Color.
  • В элемент TaskPad добавлены свойства Background, Collapsible.
  • Элемент TagLabel теперь поддерживает новый стиль LightGray.
  • Свойство Mask объекта Bind теперь поддерживает связывание.
  • В привязку BindCmd добавлены команды Navigate, Help, Download.
  • Добавлены новые иконки ArrowUpGreen, ArrowDownRed.
