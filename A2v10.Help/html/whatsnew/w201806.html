Выпуск 5 - версії 7200-7236 (июнь 2018)

Всі випуски| « Попередній випуск| Наступний випуск »

Исправлено

  • Исправлена ошибка, из-за которой для команди Select в привязке BindCmd неправильно работало властивість ValidRequired.
  • Исправлена ошибка, из-за которой не работало властивість Export в привязке BindCmd.
  • Исправлена ошибка, из-за которой не рабала прив'язка Value в об'єкті FilterItem.

Общие

  • При регистрации пользователей добавлено протоколирование відправки e-mail.
  • Добавлен механізм формуваня xml-файлів по заданной схеме.
  • В файле app.json теперь можна задавати властивості для панели "Обратная связь".

Файли програми

  • В разделе reports файла model.json властивість name теперь може містити макроси для доступа к властивостім моделі даних ({{...}}).

Шаблони

  • Добавлено событие Model.saved.
  • В события *.change добавлен аргумент oldValue (предыдущее значення).
  • Валидаторы теперь можуть бути просто функціями.

Xaml

  • Добавлен елемент GridGroup.
  • В елемент List додані властивості Select і Height.
  • В елемент ListItem добавлено властивість Footer.
  • В елемент Block добавлено властивість Color.
  • В елемент TaskPad додані властивості Background, Collapsible.
  • Елемент TagLabel теперь підтримує новий стиль LightGray.
  • Властивість Mask об'єкта Bind теперь підтримує связывание.
  • В привязку BindCmd додані команди Navigate, Help, Download.
  • Добавлены новые иконки ArrowUpGreen, ArrowDownRed.
