Компонент Slot

Все компоненты

Представляет собой элемент, который устанавливает контекст данных и область видимости для вложенных компонентов.

Наследует: UIElementBase

Свойство содержимого: Children

Свойства

Имя Тип Описание
Children UIElementCollection Свойство содержимого. Коллекция дочерних элементов.
Scope Object Всегда привязка (Bind). Контекст данных.
Fallback UIElementBase Элемент, который будет отображаться в случае, если контекст данных пустой.
