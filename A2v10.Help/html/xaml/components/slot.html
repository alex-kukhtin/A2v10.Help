Компонент Slot

Все компоненты

Представляет собой елемент, который устанавливает контекст даних и область видимости для вложенных компонентов.

Наследует: UIElementBase

Властивість содержимого: Children

Властивості

Им'я Тип Опис
Children UIElementCollection Властивість содержимого. Коллекция дочерних елементів.
Scope Object Всегда привязка (Bind). Контекст даних.
Fallback UIElementBase Елемент, который буде отображаться у випадку, якщо контекст даних пустой.
