Компонент Slot

Всі компоненти

Являє собою елемент, який встановлює контекст даних і область видимості для вкладених компонентів.

Наслідує: UIElementBase

Властивість вмісту: Children

Властивості

Ім'я Тип Опис
Children UIElementCollection Властивість вмісту. Колекція дочірніх елементів.
Scope Object Завжди прив'язка (Bind). Контекст даних.
Fallback UIElementBase Елемент, який буде отображатися у випадку, якщо контекст даних пустий.
