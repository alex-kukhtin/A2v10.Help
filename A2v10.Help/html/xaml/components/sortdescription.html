Определение сортировки для коллекции

Все компоненты

Определяет ім'я властивості и порядок сортировки для коллекции. Має сенс тільки для коллекций, обрабатываемых на клиенте.

Властивості

Им'я Тип Опис
Property String Им'я властивості для сортировки
Dir SortDirection Направление сортировки. Можливі значения:
  • Asc - (за замовчанням). По возрастанию.
  • Desc - по убыванию.
