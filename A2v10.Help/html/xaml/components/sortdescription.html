Определение сортировки для коллекции

Все компоненты

Определяет имя свойства и порядок сортировки для коллекции. Имеет смысл только для коллекций, обрабатываемых на клиенте.

Свойства

Имя Тип Описание
Property String Имя свойства для сортировки
Dir SortDirection Направление сортировки. Возможные значения:
  • Asc - (по умолчанию). По возрастанию.
  • Desc - по убыванию.
