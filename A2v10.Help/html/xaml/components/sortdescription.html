Визначення сортування для колекції

Всі компоненти

Визначає ім'я властивості і порядок сортування для колекції. Має сенс тільки для колекцій, що обробляються на клієнті.

Властивості

Ім'я Тип Опис
Property String Ім'я властивості для сортування
Dir SortDirection Напрямок сортування. Можливі значення:
  • Asc - За зростанням (за замовчанням).
  • Desc - За зменшенням.
