Компонент CollectionView

Все компоненты

Являє собою елемент, который дозволяє работать с коллекциями. Обеспечивает розбиття на сторінки, сортировку і фільтрацію. Колекція може оброблятися як на сервере, так і на клієнті.

Внутри елементу обработанная коллекция доступна по ссылке Parent. При серверной обробці требуется специальная поддержка на уровне моделі даних.

Наследует: UIElementBase

Властивість вмісту: Children

Властивості

Ім'я Тип Опис
Children UIElementCollection Властивість вмісту. Колекція дочірніх елементів.
ItemsSource Array Завжди прив'язка - Bind. Джерело даних для роботи с коллекцией.
RunAt RunMode Режим роботи с коллекцией. Можливі значення:
  • Client - (за замовчанням). Колекція обробляється на клієнті.
  • Server - коллекция обробляється на сервере.
  • ServerUrl - коллекция обробляється на сервере. Дополнительно в url додаються параметри запиту для обробки коллекции.
Filter FilterDescription Определение фильтров.
PageSize Int32? Размер страницы. Тільки для роботи на клієнті.
Sort SortDescription Определение сортировки. Тільки для роботи на клієнті.
FilterDelegate String Делегат для фильтрации. Тільки для роботи на клієнті.
Властивості базового елементу

Компонент має властивості, к которым можуть обращаться дочірні компоненты. Такое обращение осуществляется через префікс Parent.

Внешние властивості

Ім'я Тип Опис
ItemsSource Array Часть отображаемой коллекции. Содержит дані выбранной страницы в правильно отсортированном виде.
Filter Object Фильтр коллекции. Структура об'єкта соответствует определению фильтра FilterDescription.
Pager Object Джерело для связывания со властивістю Source об'єкта Pager.

Зверніть увагу! Компонент CollectionView дуже тесно взаимодействует с об'єктами, поддерживающими работу с коллекциями (зокрема DataGrid). Например, щелчок по заголовку колонки автоматично меняет сортировку в родительском компоненте. Никаких додаткових дій не требуется.
