Компонент CollectionView

Все компоненты

Представляет собой элемент, который позволяет работать с коллекциями. Обеспечивает разбиение на страницы, сортировку и фильтрацию. Коллекция может обрабатываться как на сервере, так и на клиенте.

Внутри элемента обработанная коллекция доступна по ссылке Parent. При серверной обработке требуется специальная поддержка на уровне модели данных.

Наследует: UIElementBase

Свойство содержимого: Children

Свойства

Имя Тип Описание
Children UIElementCollection Свойство содержимого. Коллекция дочерних элементов.
ItemsSource Array Всегда привязка - Bind. Источник данных для работы с коллекцией.
RunAt RunMode Режим работы с коллекцией. Возможные значения:
  • Client - (по умолчанию). Коллекция обрабатывается на клиенте.
  • Server - коллекция обрабатывается на сервере.
  • ServerUrl - коллекция обрабатывается на сервере. Дополнительно в url добавляются параметры запроса для обработки коллекции.
Filter FilterDescription Определение фильтров.
PageSize Int32? Размер страницы. Только для работы на клиенте.
Sort SortDescription Определение сортировки. Только для работы на клиенте.
FilterDelegate String Делегат для фильтрации. Только для работы на клиенте.
Свойства базового элемента

Компонент имеет свойства, к которым могут обращаться дочерние компоненты. Такое обращение осуществляется через префикс Parent.

Внешние свойства

Имя Тип Описание
ItemsSource Array Часть отображаемой коллекции. Содержит данные выбранной страницы в правильно отсортированном виде.
Filter Object Фильтр коллекции. Структура объекта соответствует определению фильтра FilterDescription.
Pager Object Источник для связывания со свойством Source объекта Pager.

Обратите внимание! Компонент CollectionView очень тесно взаимодействует с объектами, поддерживающими работу с коллекциями (в частности DataGrid). Например, щелчок по заголовку колонки автоматически меняет сортировку в родительском компоненте. Никаких дополнительных действий не требуется.
