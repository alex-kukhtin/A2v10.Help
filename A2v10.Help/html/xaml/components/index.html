Компоненты

Компоненты не имеют внешнего представления и используются для изменения поведения других компонентов.

Компоненты

  • CollectionView - представление коллекции.
  • Confirm - подтверждение.
  • FilterDescription - определение фильтра.
  • Slot - Блок с привязкой к элементу.
  • SortDescription - Определение сортировки.
  • Switch - Выбор разметки в зависимости от условия.
  • Validator - представление валидатора.
