Компоненти

Компоненти не мають зовнішнього представлення і використовуються для зміни поведінки інших компонентів.

Компоненти

  • Case - елемент компонента Switch.
  • CollectionView - вид колекції.
  • Confirm - підтвердження.
  • Else - елемент компонента Switch.
  • FilterDescription - визначення фільтра.
  • Slot - Блок з прив'язкою до елементу.
  • SortDescription - Визначення сортування.
  • Switch - Вибір розмітки в залежності від умови.
  • Validator - вид валідатора.
