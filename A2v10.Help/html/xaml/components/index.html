Компоненты

Компоненты не мають зовнішнього виду і використовуються для змінення поведения других компонентов.

Компоненты

  • CollectionView - вид коллекции.
  • Confirm - подтверждение.
  • FilterDescription - визначення фильтра.
  • Slot - Блок з прив'язкою к елементу.
  • SortDescription - Определение сортировки.
  • Switch - Выбор розмітки в залежності від условия.
  • Validator - вид валідатора.
