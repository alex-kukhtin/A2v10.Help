Компонент Component

Всі компоненти

Дозволяє вставити в розмітку користувацький компонент та встановити для нього контекст даних.

Щоб компонент можна було додати в розмітку, він має бути доступним. Словник компонентів задається властивостю Components для контейнера RootContainer.

Наслідує: UIElementBase

Властивості

Ім'я Тип Опис
Scope Object Завжди прив'язка (Bind). Контекст даних.
Name String Ім'я комопоненту. Це ключ x:Key зі словника компонентів.
Властивості базового елементу
