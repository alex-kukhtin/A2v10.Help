Перечисление AutoSelect

Все перечисления

Задает автоматическое выделение элементов в контейнере после загрузки/обновления.

Используется: DataGrid, List, TreeView.

Значения

Значение Описание
None По умолчанию. Не выделять элемент.
FirstItem Выделять первый элемент.
LastItem Выделять последний элемент.
ItemId Выделять элемент с заданным идентификатором (идентификатор берется из url страницы). Не поддерживается для DataGrid.
