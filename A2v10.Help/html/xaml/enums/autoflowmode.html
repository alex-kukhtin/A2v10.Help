Перелік AutoFlowMode

Всі переліки

Задає режим автоматичного заповнення сітки

Значення

Значення Опис
Default За замовчанням. Не вказано.
Row По рядках.
Column По стовпчиках
RowDense По рядках з ущільненням (порядок елементів може змінюватися)
ColumnDense По стовпчиках з ущільненням (порядок елементів може змінюватися)
