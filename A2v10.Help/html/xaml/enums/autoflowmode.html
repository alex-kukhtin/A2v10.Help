Перечисление AutoFlowMode

Все перечисления

Задает режим автоматического заполнения сетки

Значения

Значение Описание
Default По умолчанию. Не указано.
Row По строкам.
Column По колонкам
RowDense По строкам c уплотнением (порядок элементов может меняться)
ColumnDense По колонкам с уплотнением (порядок элементов может меняться)
