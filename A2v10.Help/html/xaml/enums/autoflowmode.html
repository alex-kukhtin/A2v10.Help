Перелік AutoFlowMode

Всі перечисления

Задает режим автоматичного заполнения сітки

Значення

Значення Опис
Default За замовчанням. Не указано.
Row По рядокм.
Column По стовпчикм
RowDense По рядокм c уплотнением (порядок елементів може змінюватися)
ColumnDense По стовпчикм з уплотнением (порядок елементів може змінюватися)
