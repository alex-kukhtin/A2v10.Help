Перелік AutoFlowMode

Все перечисления

Задает режим автоматического заполнения сетки

Значення

Значення Опис
Default За замовчанням. Не указано.
Row По строкам.
Column По колонкам
RowDense По строкам c уплотнением (порядок елементів може меняться)
ColumnDense По колонкам с уплотнением (порядок елементів може меняться)
