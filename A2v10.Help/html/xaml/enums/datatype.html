Перечисление DataType

Все перечисления

Задает тип данных.

Используется: Bind

Значения

Значение Описание
String По умолчанию. Строка.
Date Дата (без времени).
DateTime Дата и время.
Number Число.
Currency Сумма.
Boolean Логическое значение.
Object Объект.
Period Период. Специальный объект, состоящий из двух дат: начало и конец периода.
