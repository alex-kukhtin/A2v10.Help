Перечисление ListStyle

Все перечисления

Задает стиль элемента List

Значения

Значение Описание
List По умолчанию. Элементы размещаются сверху вниз.
TwoColumnsGrid Элементы размещаются в таблице с двумя колонками одинаковой ширины.
ThreeColumnsGrid Элементы размещаются в таблице с тремя колонками одинаковой ширины.
Underlined Элементы размещаются сверху вниз. После каждого элемента выводится горизонтальная линия.
Chat Элементы размещаются в виде чата. Элементами списка в этом случае чаще всего будут элементы ChatItem.
