Перелік AlignItems

Всі переліки

Задає вирівнювання елементів.

Використовується: StackPanel, Toolbar, FlexList.

Значення

Значення Опис
Default За замовчуванням. Baseline.
Baseline По базовій лінії тексту.
Center По центру.
Start По початку осі (ліворуч для горизонтальних панелей, вверх для вертикальних).
End По закінченню осі (праворуч для горизонтальних панелей, донизу для вертикальних).
Stretch Розтягувати на всю ширину/висоту контейнера.
