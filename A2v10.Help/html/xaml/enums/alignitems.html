Перелік AlignItems

Всі перечисления

Задает вирівнювання елементів.

Використовується: StackPanel, Toolbar, FlexList.

Значення

Значення Опис
Default За замовчанням. Baseline.
Baseline По базовой линии текста.
Center По центру.
Start По началу оси (ліворуч для горизонтальных панелей, вверх для вертикальных).
End по концу оси (праворуч для горизонтальных панелей, внз для вертикальных).
Stretch Растягивать на всю ширину/высоту контейнера.
