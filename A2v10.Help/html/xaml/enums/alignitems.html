Перечисление AlignItems

Все перечисления

Задает выравнивание элементов.

Используется: StackPanel, Toolbar, FlexList.

Значения

Значение Описание
Default По умолчанию. Baseline.
Baseline По базовой линии текста.
Center По центру.
Start По началу оси (влево для горизонтальных панелей, вверх для вертикальных).
End по концу оси (вправо для горизонтальных панелей, вниз для вертикальных).
Stretch Растягивать на всю ширину/высоту контейнера.
