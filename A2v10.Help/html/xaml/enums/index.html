Перечисления

  • AlignItem - вертикальное вирівнювання в контейнере.
  • AlignItems - вирівнювання дочірніх елементів.
  • AutoFlowMode - режим автоматичного заполнения.
  • AutoSelectMode - автоматическое выделение елементу.
  • BackgroundStyle - стиль фона.
  • BorderStyle - стиль рамки.
  • ButtonStyle - стиль командной кнопки.
  • ColumnBackgroundStyle - цвет фона для колонки.
  • ColumnControlType - тип елементу керування для колонки таблиці.
  • ControlSize - розмір елементу керування.
  • DataType - тип даних.
  • Icon - иконки.
  • JustifyItems - распределение дочірніх елементів.
  • ListStyle - стиль відображення списка.
  • Orientation - направление розміщення елементів.
  • RadioButtonStyle - режим відображення радивікнопки.
  • ShadowStyle - стиль тіні елементу
  • TextAlign - горизонтальное вирівнювання.
  • TextColor - цвет текста.
  • UpdateTrigger - режим оновлення моделі.
  • VerticalAlign - вертикальное вирівнювання.
  • WrapMode - режим переноса.
