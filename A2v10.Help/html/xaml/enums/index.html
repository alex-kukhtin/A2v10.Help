Перечисления

  • AlignItem - вертикальное выравнивание в контейнере.
  • AlignItems - выравнивание дочерних элементов.
  • AutoFlowMode - режим автоматического заполнения.
  • AutoSelectMode - автоматическое выделение элемента.
  • BackgroundStyle - стиль фона.
  • BorderStyle - стиль рамки.
  • ButtonStyle - стиль командной кнопки.
  • ColumnBackgroundStyle - цвет фона для колонки.
  • ColumnControlType - тип элемента управления для колонки таблицы.
  • ControlSize - размер элемента управления.
  • DataType - тип данных.
  • Icon - иконки.
  • JustifyItems - распределение дочерних элементов.
  • ListStyle - стиль отображения списка.
  • Orientation - направление размещения элементов.
  • RadioButtonStyle - режим отображения радиокнопки.
  • ShadowStyle - стиль тени элемента
  • TextAlign - горизонтальное выравнивание.
  • TextColor - цвет текста.
  • UpdateTrigger - режим обновления модели.
  • VerticalAlign - вертикальное выравнивание.
  • WrapMode - режим переноса.
