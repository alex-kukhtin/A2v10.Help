Переліки

  • AlignItem - вертикальне вирівнювання в контейнері.
  • AlignItems - вирівнювання дочірніх елементів.
  • AutoFlowMode - режим автоматичного заповнення.
  • AutoSelectMode - автоматичне виділення елементу.
  • BackgroundStyle - стиль фону.
  • BorderStyle - стиль рамки.
  • ButtonStyle - стиль командної кнопки.
  • ColumnBackgroundStyle - колір фону для стовпчика.
  • ColumnControlType - тип елементу керування для стовпчика таблиці.
  • ControlSize - розмір елементу керування.
  • DataType - тип даних.
  • Icon - іконки.
  • JustifyItems - розподіл дочірніх елементів.
  • ListStyle - стиль відображення переліку.
  • Orientation - напрямок розміщення елементів.
  • RadioButtonStyle - режим відображення радіокнопки.
  • ShadowStyle - стиль тіні елементу
  • TextAlign - горизонтальне вирівнювання.
  • TextColor - колір тексту.
  • UpdateTrigger - режим оновлення моделі.
  • VerticalAlign - вертикальне вирівнювання.
  • WrapMode - режим перенесення.
