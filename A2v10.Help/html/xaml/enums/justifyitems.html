Перелік JustifyItems

Все перечисления

Задает распределение елементів.

Використовується: StackPanel, FlexList.

Значення

Значення Опис
Default За замовчанням. Start.
Start С начала.
Center По центру.
End С конца
SpaceAround Равномерно по ширине блока (пространство перед первым и после последнйого елементу буде дорівнює половине расстояния между елементуми).
SpaceEvenly Равномерно по ширине блока (все елементи получают одинаковое пространство вокруг).
SpaceBetween Равномерно по ширине блока (первый в начале, последний в конце).
