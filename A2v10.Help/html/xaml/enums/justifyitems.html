Перечисление JustifyItems

Все перечисления

Задает распределение элементов.

Используется: StackPanel, FlexList.

Значения

Значение Описание
Default По умолчанию. Start.
Start С начала.
Center По центру.
End С конца
SpaceAround Равномерно по ширине блока (пространство перед первым и после последнего элемента будет равно половине расстояния между элементами).
SpaceEvenly Равномерно по ширине блока (все элементы получают одинаковое пространство вокруг).
SpaceBetween Равномерно по ширине блока (первый в начале, последний в конце).
