Перечисление AlignItem

Все перечисления

Задает выравнивание в контейнере

Используется: Grid

Значения

Значение Описание
Default По умолчанию. Не указано. Поведение зависит от контейнера
Start С начала (по основной оси контейнера)
End С конца (по основной оси контейнера)
Top По верхнему краю
Bottom По нижнему краю
Middle По центру
Center По центру
Stretch Растягивать на всю ширину/высоту контейнера.
Baseline По базовой линии текста.
