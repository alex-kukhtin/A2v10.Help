Перелік AlignItem

Всі переліки

Задає вирівнювання в контейнері

Використовується: Grid

Значення

Значення Опис
Default За замовчанням. Не вказано. Поведінка залежить від контейнера
Start З початку (по основній осі контейнера)
End З кінця (по основній осі контейнера)
Top По верхньому краю
Bottom По нижньому краю
Middle По центру
Center По центру
Stretch Розтягувати на всю ширину/висоту контейнера.
Baseline По базовій лінії тексту.
