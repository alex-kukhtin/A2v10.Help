Перелік AlignItem

Все перечисления

Задает выравнивание в контейнере

Використовується: Grid

Значення

Значення Опис
Default За замовчанням. Не указано. Поведінка зависит від контейнера
Start С начала (по основної оси контейнера)
End С конца (по основної оси контейнера)
Top По верхнему краю
Bottom По нижнему краю
Middle По центру
Center По центру
Stretch Растягивать на всю ширину/высоту контейнера.
Baseline По базовой линии текста.
