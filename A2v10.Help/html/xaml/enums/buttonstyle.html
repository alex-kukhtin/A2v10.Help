Перечисление ButtonStyle

Все перечисления

Задает стиль командной кнопки. Конкретный вид и цвет кнопки зависит от темы пользовательского интерфейса.

Используется: Button.

Значения

Значение Описание
Default По умолчанию. Серая.
Primary Основная кнопка. Темно-синяя.
Danger, Error, Red Ошибка. Красная.
Success, Green Успех. Зеленая.
Warning, Orange Предупреждение. Оранжевая.
Info, Cyan Информация. Голубая.
Outline Только контур.
