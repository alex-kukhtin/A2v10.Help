Задает режим оновлення значення моделі

Значення

Значення Опис
Default За замовчанням. Blur
Blur Значення в моделі оновлюється тільки при потере фокуса елементом або при нажатии клавиши Enter.
Input Значення оновлюється по мере введення символов в поле введення.
