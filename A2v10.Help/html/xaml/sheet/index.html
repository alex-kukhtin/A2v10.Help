Елементи таблиць

  • Sheet - таблиця
  • SheetCell - клітинка таблиці.
  • SheetCellGroup - група клітинок таблиці.
  • SheetColumn - стовпчик таблиці.
  • SheetColumnGroup - група стовпчиків таблиці.
  • SheetGroupCell - клітинка таблиці для елементів групування.
  • SheetPage - друкована сторінка.
  • SheetRow - рядок таблиці.
  • SheetSection - секція таблиці.
  • SheetTreeSection - секція таблиці для ієрархічних даних.
  • SheetAutoGenerate - опис автоматичної генерації.
