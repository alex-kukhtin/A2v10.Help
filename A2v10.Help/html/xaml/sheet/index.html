Елементы таблиц

  • Sheet - таблиця
  • SheetCell - ячейка таблиці.
  • SheetCellGroup - группа ячеек таблиці.
  • SheetColumn - колонка таблиці.
  • SheetColumnGroup - группа колонок таблиці.
  • SheetGroupCell - ячейка таблиці для елементів группировки.
  • SheetPage - печатная сторінка.
  • SheetRow - рядок таблиці.
  • SheetSection - секция таблиці.
  • SheetTreeSection - секция таблиці для ієрархічних даних.
  • SheetAutoGenerate - описание автоматической генерации.
