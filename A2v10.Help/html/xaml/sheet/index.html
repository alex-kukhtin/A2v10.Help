Элементы таблиц

  • Sheet - таблица
  • SheetCell - ячейка таблицы.
  • SheetCellGroup - группа ячеек таблицы.
  • SheetColumn - колонка таблицы.
  • SheetPage - печатная страница.
  • SheetRow - строка таблицы.
  • SheetSection - секция таблицы.
  • SheetTreeSection - секция таблицы для иерархических данных.
