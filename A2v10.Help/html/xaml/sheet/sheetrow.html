Элемент SheetRow

Все элементы таблиц

Представляет собой строку таблицы.

Наследует: UIElement : UIElementBase

Свойство содержимого: Cells

Свойства

Имя Тип Описание
Cells Коллекция SheetCell Коллекция ячеек в строке.
RowStyle RowStyle Стиль отображения строки. Возможные значения
  • Default - (по умолчанию) обычная строка.
  • Title - Заголовок отчета.
  • Parameter - Параметр отчета.
  • LastParameter - Последний параметр в списке параметров.
  • Header - Строка заголовка.
  • LightHeader - Строка заголовка другого стиля.
  • Footer - Строка подвала.
  • Total - Строка итогов.
  • NoBorder - Строка с ячейками без рамок.
Align TextAlign? Выравнивание по горизонтали для всех ячеек строки. Может быть переопределено в каждой ячейке индивидуально.
Mark Object Только привязка. Маркировка строки. Подробности ниже.
Свойства базового элемента

Маркировка строк

Выражение привязки для свойства Mark должно возвращать текстовую строку, которая представляет собой дополнительный css-класс, добавляемый к строке. В теме по умолчанию поддерживаются следующие варианты маркировки строк:

"danger", "error", "red" Содержимое строки
"warning", "orange" Содержимое строки
"success", "green" Содержимое строки
"info", "cyan" Содержимое строки
"gray" Содержимое строки

Обратите внимание! Маркировка просто добавляе css-класс к строке. Это означает, что вы не ограничены только указанными вариантами. Например, если свойство привязки вернет "danger bold", то строка будет маркирована красным фоном и будет показываться полужирным шрифтом.
