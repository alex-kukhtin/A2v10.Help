Елемент SheetRow

Всі елементи таблиць

Являє собою рядок таблиці.

Наслідує: UIElement : UIElementBase

Властивість вмісту: Cells

Властивості

Ім'я Тип Опис
Cells Колекція SheetCell Колекція клітинок в рядку.
RowStyle RowStyle Стиль відображення рядку. Можливі значення
  • Default - (за замовчуванням) Звичайний рядок.
  • Title - Заголовок звіту.
  • Parameter - Параметр звіту.
  • LastParameter - Останній параметр в переліку параметрів.
  • Header - Рядок заголовку.
  • LightHeader - Рядок заголовку іншого стилю.
  • Footer - Рядок футера.
  • Total - Рядок підсумків.
  • NoBorder - Рядок з клітинками без рамок.
Align TextAlign? Вирівнювання по горизонталі для всіх клітинок рядку. Може бути перевизначено в кожній клітинці індивідуально.
Mark Object Тільки прив'язка. Маркування рядку. Подробиці нижче.
Властивості базового елементу

Маркування рядків

Вираз прив`язки для властивості Mark повинен повертати текстовий рядок, який являє собою додатковий css-клас, що додається до рядку. В темі за замовчуванням підтримуються наступні варіанти маркування рядків:

"danger", "error", "red" Вміст рядку
"warning", "orange" Вміст рядку
"success", "green" Вміст рядку
"info", "cyan" Вміст рядку
"gray" Вміст рядку

Зверніть увагу! Маркування просто добавляє css-клас до рядку. Це означає, що ви не обмежені тільки вказаними варіантами. Наприклад, якщо властивість прив`язки поверне "danger bold", то рядок буде маркований червоним фоном і буде відображатися напівжирним шрифтом.
