Елемент SheetRow

Все елементи таблиц

Являє собою строку таблиці.

Наследует: UIElement : UIElementBase

Властивість вмісту: Cells

Властивості

Ім'я Тип Опис
Cells Колекція SheetCell Колекція ячеек в строке.
RowStyle RowStyle Стиль відображення строки. Можливі значения
  • Default - (за замовчанням) обычная рядок.
  • Title - Заголовок звіта.
  • Parameter - Параметр звіта.
  • LastParameter - Последний параметр в списке параметров.
  • Header - Рядок заголовка.
  • LightHeader - Рядок заголовка другого стиля.
  • Footer - Рядок подвала.
  • Total - Рядок итогов.
  • NoBorder - Рядок з ячейками без рамок.
Align TextAlign? Выравнивание по горизонтали для всех ячеек строки. Може бути переопределено в каждой ячейке индивидуально.
Mark Object Тільки привязка. Маркировка строки. Подробности ниже.
Властивості базового елементу

Маркировка строк

Выражение прив`язки для властивості Mark повинно возвращать текстовую строку, яка являє собою додатковий css-класс, добавляемый к строке. В теме за замовчанням поддерживаются наступні варианты маркировки строк:

"danger", "error", "red" Вміст строки
"warning", "orange" Вміст строки
"success", "green" Вміст строки
"info", "cyan" Вміст строки
"gray" Вміст строки

Зверніть увагу! Маркировка просто добавляе css-класс к строке. Це означає, що вы не ограничены тільки вказаними вариантами. Наприклад, якщо властивість прив`язки вернет "danger bold", то рядок буде маркирована красным фоном і буде показываться полужирным шрифтом.
