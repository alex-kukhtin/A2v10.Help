Элемент SheetGroupCell

Все элементы таблиц

Представляет собой ячейку для отображения элементов управления группировкой в таблице.

Ячейка не имеет содержимого, вместо этого она отображает элементы управления группировкой в таблице. Внешний вид зависит от темы пользовательского интерфейса.

Использование такой ячейки имеет смысл только для строк в иерархических секциях SheetTreeSection. Во всех остальных случаях ячейка просто будет пустой.

Наследует: SheetCell : UIContentElement : UIElementBase
