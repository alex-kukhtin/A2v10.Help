Элемент SheetCellGroup

Все элементы таблиц

Представляет собой коллекцию ячеек таблицы. Может спользоваться в любом месте, где допустимо использование элемента SheetCell.

Свойство содержимого: Cells

Свойства

Имя Тип Описание
Cells Коллекция SheetCell Коллекция ячеек в строке.
ItemsSource Array Всегда привязка (Bind). Источник данных для получения дочерних ячеек.

Описание

Обычно этот элемент используется для совместной работы с перекрестными моделями данных.
