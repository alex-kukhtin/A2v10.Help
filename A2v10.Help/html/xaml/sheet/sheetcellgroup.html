Елемент SheetCellGroup

Все елементи таблиц

Являє собою коллекцию ячеек таблиці. Может спользоваться в любом месте, де допустимо использование елементу SheetCell.

Властивість вмісту: Cells

Властивості

Им'я Тип Опис
Cells Колекція SheetCell Колекція ячеек в строке.
ItemsSource Array Завжди прив'язка (Bind). Джерело даних для отримання дочірніх ячеек.

Опис

Зазвичай цей елемент використовується для совместной роботи с перехресними моделями даних.
