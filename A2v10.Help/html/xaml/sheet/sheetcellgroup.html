Елемент SheetCellGroup

Всі елементи таблиц

Являє собою колекцію клітинок таблиці. Может спользоваться в любом месте, де допустимо використання елементу SheetCell.

Властивість вмісту: Cells

Властивості

Ім'я Тип Опис
Cells Колекція SheetCell Колекція клітинок в рядку.
ItemsSource Array Завжди прив'язка (Bind). Джерело даних для отримання дочірніх клітинок.

Опис

Зазвичай цей елемент використовується для совместной роботи з перехресними моделями даних.
