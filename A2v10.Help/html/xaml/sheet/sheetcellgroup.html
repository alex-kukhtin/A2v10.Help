Елемент SheetCellGroup

Всі елементи таблиць

Являє собою колекцію клітинок таблиці. Може використовуватися в будь-якому місці, де припустиме використання елементу SheetCell.

Властивість вмісту: Cells

Властивості

Ім'я Тип Опис
Cells Колекція SheetCell Колекція клітинок в рядку.
ItemsSource Array Завжди прив'язка (Bind). Джерело даних для отримання дочірніх клітинок.
If Boolean? Включати елемент в потік рендеринга.

Опис

Зазвичай цей елемент використовується для спільної роботи з перехресними моделями даних.
