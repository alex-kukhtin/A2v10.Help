Элемент SheetPage

Все элементы таблиц

Представляет собой печатную страницу для таблицы Sheet. Используется для визуального отображения печатных страниц на экране.

Наследует: Container : UIElement : UIElementBase

Свойство содержимого: Children

Свойства

Имя Тип Описание
Orientation PageOrientation Ориентация страницы. Возможные значения:
  • Portrait - по умолчанию. Портретная.
  • Landscape - ландшафтная.
Если указано свойство PrintPage, то свойство игнорируется.
PageSize Size? Размер страницы. Если не указан - то формат А4. Обратите внимание, что печать страницы с указанными размерами может быть проблематичной. Свойствами печатной страницы можно управлять более гибко с помощью свойства PrintPage.
PrintPage PrintPage Описание печатной страницы.
