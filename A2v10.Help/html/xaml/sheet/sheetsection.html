Елемент SheetSection

Все елементи таблиц

Являє собою секцию таблиці.

Секция використовується у випадку, якщо потрібно связывать дані з джерелом даних. Джерело задається прив'язкою ItemsSource. Строки секции будуть повторены для кожного елементу масива - джерела.

Якщо в теле таблице безпосередньо задаются строки, то вони попадают в секцию, яка створюється неявно. Джерело даних для такой секции задать невозможна.

Наследует: UIElement : UIElementBase

Властивість вмісту: Children

Властивості

Ім'я Тип Опис
Children Колекція SheetRow Колекція рядків в секции.
ItemsSource Array Завжди прив'язка (Bind). Джерело даних для отримання дочірніх елементів.
