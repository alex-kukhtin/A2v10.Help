Елемент SheetSection

Все елементи таблиц

Представляет собой секцию таблиці.

Секция використовується у випадку, якщо потрібно связывать дані с джерелоом даних. Джерело задається привязкой ItemsSource. Строки секции будуть повторены для каждого елементу масива - джерела.

Якщо в теле таблице непосредственно задаются строки, то вони попадают в секцию, яка создается неявно. Джерело даних для такой секции задать невозможна.

Наследует: UIElement : UIElementBase

Властивість содержимого: Children

Властивості

Им'я Тип Опис
Children Коллекция SheetRow Коллекция рядків в секции.
ItemsSource Array Всегда привязка (Bind). Джерело даних для отримання дочірніх елементів.
