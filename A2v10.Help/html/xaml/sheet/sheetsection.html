Елемент SheetSection

Всі елементи таблиць

Являє собою секцію таблиці.

Секція використовується у випадку, якщо потрібно зв'язувати дані з джерелом даних. Джерело задається прив'язкою ItemsSource. Рядки секції будуть повторені для кожного елементу масиву - джерела.

Якщо в тілі таблиці безпосередньо задаються рядки, то вони потрапляють в секцію, яка створюється неявно. Джерело даних для такої секції задати неможливо.

Наслідує: UIElement : UIElementBase

Властивість вмісту: Children

Властивості

Ім'я Тип Опис
Children Колекція SheetRow Колекція рядків в секції.
ItemsSource Array Завжди прив'язка (Bind). Джерело даних для отримання дочірніх елементів.
