Элемент SheetSection

Все элементы таблиц

Представляет собой секцию таблицы.

Секция используется в случае, если нужно связывать данные с источником данных. Источник задается привязкой ItemsSource. Строки секции будут повторены для каждого элемента массива - источника.

Если в теле таблице непосредственно задаются строки, то они попадают в секцию, которая создается неявно. Источник данных для такой секции задать невозможно.

Наследует: UIElement : UIElementBase

Свойство содержимого: Children

Свойства

Имя Тип Описание
Children Коллекция SheetRow Коллекция строк в секции.
ItemsSource Array Всегда привязка (Bind). Источник данных для получения дочерних элементов.
