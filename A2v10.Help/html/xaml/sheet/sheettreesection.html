Элемент SheetTreeSection

Все элементы таблиц

Представляет собой секцию для размещения иерархических данных. Связывается с источником данных рекурсивно.

Предназначено для связывания с иерархическими моделями или моделями с группировкой. Такие модели фактически являются рекурсивными.

Наследует: SheetSection : UIElement : UIElementBase
