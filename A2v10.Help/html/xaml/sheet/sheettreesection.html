Елемент SheetTreeSection

Всі елементи таблиц

Являє собою секцию для розміщення ієрархічних даних. Связывается з джерелом даних рекурсивно.

Предназначено для зв'язування з иерархическими моделями або моделями з групуванням. Такие моделі фактично є рекурсивными.

Наслідує: SheetSection : UIElement : UIElementBase
