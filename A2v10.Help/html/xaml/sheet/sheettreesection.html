Елемент SheetTreeSection

Всі елементи таблиць

Являє собою секцію для розміщення ієрархічних даних. Зв'язується з джерелом даних рекурсивно.

Призначено для зв'язування з ієрархічними моделями або моделями з групуванням. Такие моделі фактично є рекурсивними.

Наслідує: SheetSection : UIElement : UIElementBase
