Елемент SheetColumnGroup

Все елементи таблиц

Являє собою коллекцию колонок таблиці. Может спользоваться в любом месте, де допустимо использование елементу SheetColumn.

Властивість вмісту: Columns

Властивості

Им'я Тип Опис
Columns Колекція SheetColumn Колекція колонок в строке.
ItemsSource Array Завжди прив'язка (Bind). Джерело даних для отримання дочірніх колонок.

Опис

Зазвичай цей елемент використовується для совместной роботи с перехресними моделями даних.
