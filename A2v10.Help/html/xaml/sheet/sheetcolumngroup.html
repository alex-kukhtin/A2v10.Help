Елемент SheetColumnGroup

Всі елементи таблиц

Являє собою колекцію стовпчиків таблиці. Может спользоваться в любом месте, де допустимо використання елементу SheetColumn.

Властивість вмісту: Columns

Властивості

Ім'я Тип Опис
Columns Колекція SheetColumn Колекція стовпчиків в рядку.
ItemsSource Array Завжди прив'язка (Bind). Джерело даних для отримання дочірніх стовпчиків.

Опис

Зазвичай цей елемент використовується для совместной роботи з перехресними моделями даних.
