Элемент SheetColumnGroup

Все элементы таблиц

Представляет собой коллекцию колонок таблицы. Может спользоваться в любом месте, где допустимо использование элемента SheetColumn.

Свойство содержимого: Columns

Свойства

Имя Тип Описание
Columns Коллекция SheetColumn Коллекция колонок в строке.
ItemsSource Array Всегда привязка (Bind). Источник данных для получения дочерних колонок.

Описание

Обычно этот элемент используется для совместной работы с перекрестными моделями данных.
