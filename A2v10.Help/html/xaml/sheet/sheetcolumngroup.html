Елемент SheetColumnGroup

Все елементи таблиц

Представляет собой коллекцию колонок таблиці. Может спользоваться в любом месте, где допустимо использование елементу SheetColumn.

Властивість содержимого: Columns

Властивості

Им'я Тип Опис
Columns Коллекция SheetColumn Коллекция колонок в строке.
ItemsSource Array Всегда привязка (Bind). Джерело даних для отримання дочірніх колонок.

Опис

Зазвичай этот елемент використовується для совместной роботи с перехресними моделями даних.
