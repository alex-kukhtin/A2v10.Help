Елемент SheetCell

Всі елементи таблиць

Являє собою клітинку таблиці.

Наслідує: UIContentElement : UIElementBase

Властивості

Ім'я Тип Опис
ColSpan Int32? Об'єднання стовпчиків.
RowSpan Int32? Об'єднання рядків.
Bold Boolean? Напівжирний шрифт. Якщо не вказано, наслідується від батьківського рядку.
Italic Boolean? Курсивний шрифт. Якщо не вказано, наслідується від батьківського рядку.
Underline Boolean Підкреслений текст.
Vertical Boolean Вертикальна орієнтація тексту в клітинці. Автоматичне перенесення не підтримується. Для перенесення на інший рядок додайте в текст "\n".
Align TextAlign Горизонтальне вирівнювання тексту.
VAlign VerticalAlign Вертикальне вирівнювання тексту.
GroupIndent Boolean Відступ для ієрархічних елементів. Використовується у випадку, якщо клітинка знаходиться в секції SheetTreeSection.
DataType DataType Тип даних. Використовується у випадку, якщо в клітинці нема прив`язки, а при експорті потрібно встановити заданий тип даних.
MinWidth Length Мінімальна ширина клітинки.
CssClass String Класи css, які потрібно додати до клітинки.
Fill Bind Завжди прив'язка. Колір для заливки фону клітинки. Використовується для заливки фоном, колір якого залежить від даних. Використовує css-властивість background-color. Несумісно з властивістю CssStyle.
CssStyle Bind Завжди прив'язка. Правила css для клітинки. Властивість повинна повернути об'єкт. Ім'я властивості - властивість css, значення - відповідне значення. В іменах властивостей можна використовувати camelCase нотацію, наприклад, властивість background-color можна представити як backgroundColor. Несумісно з властивістю Fill.
