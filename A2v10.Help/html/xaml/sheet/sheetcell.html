Элемент SheetCell

Все элементы таблиц

Представляет собой ячейку таблицы.

Наследует: UIContentElement : UIElementBase

Свойства

Имя Тип Описание
ColSpan Int32? Объединение колонок.
RowSpan Int32? Объединение строк.
Bold Boolean? Полужирный шрифт. Еслие не указан, наследуется от родительской строки.
Italic Boolean? Курсивный шрифт. Еслие не указан, наследуется от родительской строки.
Underline Boolean Подчеркнутый текст.
Vertical Boolean Вертикальная ориентация текста в ячейке. Автоматический перенос не поддерживается. Для переноса на другую строку включите в текст "\n".
Align TextAlign Горизонтальное выравнивание текста.
VAlign VerticalAlign Вертикальное выравнивание текста.
GroupIndent Boolean Отступ для иерархических элементов. Используется в случае, если ячейка находится в секции SheetTreeSection.
