Элемент SheetCell

Все элементы таблиц

Представляет собой ячейку таблицы.

Наследует: UIContentElement : UIElementBase

Свойства

Имя Тип Описание
ColSpan Int32? Объединение колонок.
RowSpan Int32? Объединение строк.
Bold Boolean? Полужирный шрифт. Если не указан, наследуется от родительской строки.
Italic Boolean? Курсивный шрифт. Если не указан, наследуется от родительской строки.
Underline Boolean Подчеркнутый текст.
Vertical Boolean Вертикальная ориентация текста в ячейке. Автоматический перенос не поддерживается. Для переноса на другую строку включите в текст "\n".
Align TextAlign Горизонтальное выравнивание текста.
VAlign VerticalAlign Вертикальное выравнивание текста.
GroupIndent Boolean Отступ для иерархических элементов. Используется в случае, если ячейка находится в секции SheetTreeSection.
DataType DataType Тип данных. Используется в случае, если в ячейке нет привязки, а при экспорте нужно установить заданный тип данных.
MinWidth Length Минимальная ширина ячейки.
CssClass String Классы css, которые нужно добавить к ячейке.
Fill Bind Всегда привязка. Цвет для заливки фона ячейки. Используется для заливки фоном, цвет которого зависит от данных. Использует css-свойство background-color. Не совместимо со свойством CssStyle.
CssStyle Bind Всегда привязка. Правила css для ячейки. Свойство должно вернуть объект. Имя свойства - свойтсво css, значение - соответствующее значение. В именах свойств можно использовать camelCase нотацию, например свойство background-color можно представить как backgroundColor. Не совместимо со свойством Fill.
