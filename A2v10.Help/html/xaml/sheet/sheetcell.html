Елемент SheetCell

Все елементи таблиц

Являє собою ячейку таблиці.

Наследует: UIContentElement : UIElementBase

Властивості

Ім'я Тип Опис
ColSpan Int32? Объединение колонок.
RowSpan Int32? Объединение строк.
Bold Boolean? Полужирный шрифт. Якщо не указан, наследуется від родительской строки.
Italic Boolean? Курсивный шрифт. Якщо не указан, наследуется від родительской строки.
Underline Boolean Подчеркнутый текст.
Vertical Boolean Вертикальная ориентация текста в ячейке. Автоматический перенос не підтримується. Для переноса на іншу строку включите в текст "\n".
Align TextAlign Горизонтальное выравнивание текста.
VAlign VerticalAlign Вертикальное выравнивание текста.
GroupIndent Boolean Отступ для ієрархічних елементів. Використовується у випадку, якщо ячейка знаходиться в секции SheetTreeSection.
DataType DataType Тип даних. Використовується у випадку, якщо в ячейке нет прив`язки, а при експорте потрібно встановити заданий тип даних.
MinWidth Length Минимальная ширина ячейки.
CssClass String Классы css, які потрібно додати к ячейке.
Fill Bind Завжди привязка. Цвет для заливки фона ячейки. Використовується для заливки фоном, цвет якого зависит від даних. Использует css-властивість background-color. Не совместимо со властивістю CssStyle.
CssStyle Bind Завжди привязка. Правила css для ячейки. Властивість повинно повернути об'єкт. Ім'я властивості - свойтсво css, значення - соответствующее значення. В іменах властивостей можна використовувати camelCase нотацию, наприклад властивість background-color можна представить як backgroundColor. Не совместимо со властивістю Fill.
