Елемент SheetCell

Всі елементи таблиц

Являє собою клітинку таблиці.

Наслідує: UIContentElement : UIElementBase

Властивості

Ім'я Тип Опис
ColSpan Int32? Об'єднання стовпчиків.
RowSpan Int32? Об'єднання рядків.
Bold Boolean? Напівжирний шрифт. Якщо не указан, наслідується від батьківської рядки.
Italic Boolean? Курсивный шрифт. Якщо не указан, наслідується від батьківської рядки.
Underline Boolean Подчеркнутый текст.
Vertical Boolean Вертикальная ориентация текста в ячейці. Автоматичний перенесення не підтримується. Для переноса на іншу строку включите в текст "\n".
Align TextAlign Горизонтальне вирівнювання текста.
VAlign VerticalAlign Вертикальне вирівнювання текста.
GroupIndent Boolean Отступ для ієрархічних елементів. Використовується у випадку, якщо клітинка знаходиться в секции SheetTreeSection.
DataType DataType Тип даних. Використовується у випадку, якщо в ячейці нет прив`язки, а при експорте потрібно встановити заданий тип даних.
MinWidth Length Мінімальна ширина клітинки.
CssClass String Классы css, які потрібно додати к ячейке.
Fill Bind Завжди прив'язка. Колір для заливки фона клітинки. Використовується для заливки фоном, колір якого залежить від даних. Использует css-властивість background-color. Не совместимо со властивістю CssStyle.
CssStyle Bind Завжди прив'язка. Правила css для клітинки. Властивість повинно повернути об'єкт. Ім'я властивості - свойтсво css, значення - відповідне значення. В іменах властивостей можна використовувати camelCase нотацию, наприклад властивість background-color можна представить як backgroundColor. Не совместимо со властивістю Fill.
