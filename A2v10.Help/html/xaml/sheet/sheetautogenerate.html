Элемент SheetAutoGenerate

Все элементы таблиц

Объект, который описывает правила генерации таблицы из данных.

Конвертер: поддерживается преобразование из строки текста в объект, где заданная строка является свойством PropertyName, а Mode = FromDataModel.

Свойства

Имя Тип Описание
Mode SheetAutoGenerateMode Режим автогенерации. Возможные значения:
  • FromDataModel - по умолчанию. Из модели данных. Должно быть задано свойство PropertyName.
PropertyName String Имя свойства модели, которое должно представлять коллекцию для генерации таблицы.
