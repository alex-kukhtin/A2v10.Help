Елемент SheetAutoGenerate

Всі елементи таблиць

Об'єкт, який описує правила генерації таблиці з даних.

Конвертер: підтримується перетворення з рядку тексту в об'єкт, де заданий рядок є властивістю PropertyName, а Mode = FromDataModel.

Властивості

Ім'я Тип Опис
Mode SheetAutoGenerateMode Режим автогенерації. Можливі значення:
  • FromDataModel - за замовчанням. З моделі даних. Повинно бути задано властивість PropertyName.
PropertyName String Ім'я властивості моделі, яке повинно представляти колекцію для генерації таблиці.
