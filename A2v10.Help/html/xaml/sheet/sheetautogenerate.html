Елемент SheetAutoGenerate

Все елементи таблиц

Об'єкт, который описує правила генерации таблиці з даних.

Конвертер: підтримуєся преобразование з строки текста в об'єкт, где заданная строка является властивістю PropertyName, а Mode = FromDataModel.

Властивості

Им'я Тип Опис
Mode SheetAutoGenerateMode Режим автогенерации. Можливі значения:
  • FromDataModel - за замовчанням. Из моделі даних. Повинно бути задано властивість PropertyName.
PropertyName String Им'я властивості моделі, которое должно представлять коллекцию для генерации таблиці.
