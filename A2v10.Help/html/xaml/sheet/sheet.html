Елемент Sheet

Всі елементи таблиц

Являє собою таблицю в стиле электронных таблиц. Таблица оптимизирована для простий выгрузки в Excel.

Наслідує: UIElement : UIElementBase

Властивість вмісту: Sections

Властивості

Ім'я Тип Опис
Sections SheetSections Набор секций таблиці. (SheetSection)
Columns SheetColumnCollection Стовпчики таблиці. (SheetColumn)
Header SheetRows Строки заголовка. (SheetRow)
Footer SheetRows Строки футера. (SheetRow)
Compact Boolean Компактный стиль відображення таблиці.
Hover Boolean Виділення рядки при наведенні курсора миші.
Striped Boolean Розфарбування таблиці "у смужку" з виділенням парних і непарних рядків.
GridLines GridLinesVisibility Відображення сітки. Можливі значення:
  • None - за замовчанням.
  • Horizontal - горизонтальні лінії.
  • Vertical - вертикальні лінії.
  • Both - сітка.
AutoGenerate SheetAutoGenerate Автоматическая генерация таблиці. Дозволяє сгенерувати таблицю у відповідності до текущими даними.
