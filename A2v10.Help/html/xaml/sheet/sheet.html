Елемент Sheet

Все елементи таблиц

Являє собою таблицю в стиле электронных таблиц. Таблица оптимизирована для простий выгрузки в Excel.

Наследует: UIElement : UIElementBase

Властивість вмісту: Sections

Властивості

Ім'я Тип Опис
Sections SheetSections Набор секций таблиці. (SheetSection)
Columns SheetColumnCollection Колонки таблиці. (SheetColumn)
Header SheetRows Строки заголовка. (SheetRow)
Footer SheetRows Строки футера. (SheetRow)
Compact Boolean Компактный стиль відображення таблиці.
Hover Boolean Выделение строки при наведении курсора мыши.
Striped Boolean Раскраска таблиці "в полоску" с выделением четных і нечетных строк.
GridLines GridLinesVisibility Отображение сетки. Можливі значення:
  • None - за замовчанням.
  • Horizontal - горизонтальные линии.
  • Vertical - вертикальные линии.
  • Both - сетка.
AutoGenerate SheetAutoGenerate Автоматическая генерация таблиці. Позволяет сгенерировать таблицю у відповідності до текущими даними.
