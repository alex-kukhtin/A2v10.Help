Элемент Sheet

Все элементы таблиц

Представляет собой таблицу в стиле электронных таблиц. Таблица оптимизирована для простой выгрузки в Excel.

Наследует: UIElement : UIElementBase

Свойство содержимого: Sections

Свойства

Имя Тип Описание
Sections SheetSections Набор секций таблицы. (SheetSection)
Columns SheetColumnCollection Колонки таблицы. (SheetColumn)
Header SheetRows Строки заголовка. (SheetRow)
Footer SheetRows Строки футера. (SheetRow)
Compact Boolean Компактный стиль отображения таблицы.
Hover Boolean Выделение строки при наведении курсора мыши.
Striped Boolean Раскраска таблицы "в полоску" с выделением четных и нечетных строк.
GridLines GridLinesVisibility Отображение сетки. Возможные значения:
  • None - по умолчанию.
  • Horizontal - горизонтальные линии.
  • Vertical - вертикальные линии.
  • Both - сетка.
AutoGenerate SheetAutoGenerate Автоматическая генерация таблицы. Позволяет сгенерировать таблицу в соответствии с текущими данными.
