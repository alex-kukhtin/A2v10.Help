Текстові елементи

Текстові елементи (Inlines) використовуються для відображення тексту/зображень.

Абстрактні елементи

  • Inline - рядковий елемент

Текстові елементи

  • Badge - бейджик.
  • Break - перенесення рядку.
  • Html - html-текст.
  • Hyperlink - посилання.
  • Line - горизонтальна лінія.
  • Paragraph - параграф тексту.
  • Popover - посилання для спливаючого елементу.
  • Span - рядковий елемент.
  • SpanIcon - іконка в тексті.
  • SpanSum - рядковий елемент для суми.
  • StaticImage - зображення в тексті.
  • TagLabel - мітка - тег.
  • Text - текст.
