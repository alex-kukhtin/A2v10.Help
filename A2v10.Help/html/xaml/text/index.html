Текстовые елементи

Текстовые елементи (Inlines) використовуються для відображення текста/зображень.

Абстрактные елементи

  • Inline - строчный елемент

Текстовые елементи

  • Badge - бейджик.
  • Break - перенос строки.
  • Html - html-текст.
  • Hyperlink - ссылка.
  • Line - горизонтальная линия.
  • Paragraph - параграф текста.
  • Popover - посилання для всплывающйого елементу.
  • Span - строчный елемент.
  • SpanIcon - иконка в тексте.
  • SpanSum - строчный елемент для суммы.
  • StaticImage - картинка в тексте.
  • TagLabel - метка - тег.
  • Text - текст.
