Текстові елементи

Текстові елементи (Inlines) використовуються для відображення текста/зображень.

Абстрактні елементи

  • Inline - рядковий елемент

Текстові елементи

  • Badge - бейджик.
  • Break - перенос рядки.
  • Html - html-текст.
  • Hyperlink - ссылка.
  • Line - горизонтальная линия.
  • Paragraph - параграф текста.
  • Popover - посилання для всплывающйого елементу.
  • Span - рядковий елемент.
  • SpanIcon - иконка в тексте.
  • SpanSum - рядковий елемент для суммы.
  • StaticImage - картинка в тексте.
  • TagLabel - метка - тег.
  • Text - текст.
