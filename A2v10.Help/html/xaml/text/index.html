Текстовые элементы

Текстовые элементы (Inlines) используются для отображения текста/картинок.

Абстрактные элементы

  • Inline - строчный элемент

Текстовые элементы

  • Badge - бейджик.
  • Break - перенос строки.
  • Html - html-текст.
  • Hyperlink - ссылка.
  • Line - горизонтальная линия.
  • Paragraph - параграф текста.
  • Popover - ссылка для всплывающего элемента.
  • Span - строчный элемент.
  • SpanIcon - иконка в тексте.
  • SpanSum - строчный элемент для суммы.
  • StaticImage - картинка в тексте.
  • TagLabel - метка - тег.
  • Text - текст.
