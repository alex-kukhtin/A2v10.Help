Елемент SpanSum

Всі текстові елементи

Являє собою рядковий текстовий елемент для відображення суммы. Відображає сумму зеленым або червоним кольором) і символ прихода/расхода у вигляді иконки.

Наслідує: Inline : UIElement : UIElementBase

Властивість вмісту: Content

Властивості

Ім'я Тип Опис
Content Object Отображаемая сумма.
Dir Int32 Признак прихода/расхода. Якщо значення больше нуля - приход, менше нуля - расход, дорівнює нулю - приход/расход
Властивості базового елементу

Приклад відображення сумм
