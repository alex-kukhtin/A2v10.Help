Елемент Hyperlink

Всі текстові елементи

Являє собою гіперпосилання.

Наслідує: Inline : UIElement : UIElementBase

Властивість вмісту: Content

Властивості

Ім'я Тип Опис
Content String Текст посилання.
Command BindCmd Команда, яку буде виконано при кліку на гіперпосилання.
Icon Icon Іконка
Size ControlSize Розмір посилання. Можливі значення:
  • Default - За замовчуванням (Normal).
  • Small - зменшений.
  • Normal - звичайний.
  • Large - збільшений.
Style HyperlinkStyle Стиль посилання. Можливі значення:
  • Default - За замовчуванням. Звичайне посилання
  • Popover - посилання для спливаючої підказки. Підкреслюється пунктирною лінією. Конкретний вигляд залежить від теми користувацького інтерфейса.
DropDown UIElementBase Спадний елемент, пов'язаний з посиланням. Зазвичай це елемент DropDownMenu.
HideCaret Boolean Приховувати значок розкриття спадного переліку.
Highlight Boolean Виділяти іконку кольором активного гіперпосилання при наведенні курсора миші. Має сенс тільки для деяких іконок (наприклад Send).
TestId String Ідентифікатор елементу для інструментів тестування.
Hint Popover Спливаюча підказка для посилання. Виводиться у вигляді іконки, вказаної у властивості Icon об'єкта Popover. Якщо іконку не вказано - виводиться .
