Елемент Hyperlink

Все текстові елементи

Являє собою гиперпосилання.

Наследует: Inline : UIElement : UIElementBase

Властивість вмісту: Content

Властивості

Ім'я Тип Опис
Content String Текст посилання.
Command BindCmd Команда, яка буде выполнена при нажатии гиперпосилання.
Icon Icon Иконка
Size ControlSize Размер посилання. Можливі значення:
  • Default - За замовчанням (Normal).
  • Small - уменьшенный.
  • Normal - звичайний.
  • Large - увеличенный.
Style HyperlinkStyle Стиль посилання. Можливі значення:
  • Default - За замовчанням. Обычная ссылка
  • Popover - ссылка для всплывающей подсказки. Подчеркивается пунктирной линией. Конкретный вид зависит від темы пользовательского інтерфейса.
DropDown UIElementBase Выпадающий елемент, связанный з посиланням. Зазвичай це елемент DropDownMenu.
HideCaret Boolean Скрывать значок розкриття выпадающйого списка.
Highlight Boolean Выделять иконку цветом активной гиперпосилання при наведении курсора мыши. Має сенс тільки для некоторых иконок (наприклад Send).
TestId String Ідентифікатор елементу для інструментів тестування.
Hint Popover Всплывающая подсказка для посилання. Выводится у вигляді иконки, указанной у властивості Icon об'єкта Popover. Якщо иконка не указана - выводится .
