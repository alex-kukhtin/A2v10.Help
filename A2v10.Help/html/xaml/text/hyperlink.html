Элемент Hyperlink

Все текстовые элементы

Представляет собой гиперссылку.

Наследует: Inline : UIElement : UIElementBase

Свойство содержимого: Content

Свойства

Имя Тип Описание
Content String Текст ссылки.
Command BindCmd Команда, которая будет выполнена при нажатии гиперссылки.
Icon Icon Иконка
Size ControlSize Размер ссылки. Возможные значения:
  • Default - По умолчанию (Normal).
  • Small - уменьшенный.
  • Normal - обычный.
  • Large - увеличенный.
Style HyperlinkStyle Стиль ссылки. Возможные значения:
  • Default - По умолчанию. Обычная ссылка
  • Popover - ссылка для всплывающей подсказки. Подчеркивается пунктирной линией. Конкретный вид зависит от темы пользовательского интерфейса.
DropDown UIElementBase Выпадающий элемент, связанный со ссылкой. Обычно это элемент DropDownMenu.
HideCaret Boolean Скрывать значок раскрытия выпадающего списка.
Highlight Boolean Выделять иконку цветом активной гиперссылки при наведении курсора мыши. Имеет смысл только для некоторых иконок (например Send).
TestId String Идентификатор элемента для инструментов тестирования.
Hint Popover Всплывающая подсказка для ссылки. Выводится в виде иконки, указанной в свойстве Icon объекта Popover. Если иконка не указана - выводится .
