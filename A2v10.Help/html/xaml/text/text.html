Элемент Text

Все текстовые элементы

Представляет собой фрагмент текста. Поддерживает возможность вставки дополнительных строчных элементов (например Span) непосредственно в текст.

Наследует: UIElement : UIElementBase

Свойство содержимого: Inlines

Свойства

Имя Тип Описание
Inlines InlineCollection Свойство содержимого. Текстовое содержимое. Строка текста, которая может включать другие строчные элементы.
Align TextAlign Выравнивание текста.
Block Boolean Выводить текст в виде блочного, а не строчного элемента.
Small Boolean Уменьшенный шрифт. Конкретный размер шрифта зависит от темы пользовательского интерфейса.
Big Boolean Увеличенный шрифт. Конкретный размер шрифта зависит от темы пользовательского интерфейса.
Size TextSize Размер текста. Возможные значения:
  • Normal - (по умолчанию) нормальный размер текста.
  • Big - увеличенный. (то-же самое что и Small="True")
  • Small - уменьшенный. (то-же самое что и Big="True")
Gray Boolean Серый цвет текста. Конкретный цвет зависит от темы пользовательского интерфейса.
Color TextColor Цвет текста.
