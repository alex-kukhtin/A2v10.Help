Элемент Span

Все текстовые элементы

Представляет собой фрагмент текста.

Наследует: Inline : UIElement : UIElementBase

Свойство содержимого: Content

Свойства

Имя Тип Описание
Content String Свойство содержимого. Текстовое содержимое.
Small Boolean Уменьшенный шрифт. Конкретный размер шрифта зависит от темы пользовательского интерфейса.
Big Boolean Увеличенный шрифт. Конкретный размер шрифта зависит от темы пользовательского интерфейса.
Space SpaceMode Добавлять пробел до или после элемента. Возможные значения:
  • None - (по умолчанию) нет пробелов.
  • Before - пробел перед элементом.
  • After - пробел после элемента.
  • Both - пробел перед и после элемента.
MaxChars Int32 Максимальное количество символов в элементе. Работает только для связанных значений. Если в значении больше символов, то текст обрезается и в конец добавляется многоточие. Не забывайте предоставлять пользователю полный текст (например с использованием подсказки - Tip).
