Елемент Popover

Всі текстові елементи

Являє собою посилання, при щелчке (або наведении курсора миші) на которой показывается всплывающее вікно. Спливаюче вікно може містити як простий текст або другой елемент, так і посилатися на другой url.

Наслідує: Inline : UIElement : UIElementBase

Властивість вмісту: Content

Конвертер: підтримується перетворення з рядки текста і з будь-якого елементу, нанаступного UIElement.

Властивості

Ім'я Тип Опис
Content Object Властивість вмісту. Може бути звичайним текстом або будь-яким елементом.
Url String Url для отримання вмісту з сервера. В цьому випадку всплывающее вікно буде містити собственную модель даних, собственный шаблон и, конечно, саму розмітку.
Text String Текст посилання.
Icon Icon Іконка для посилання.
ShowOnHover Boolean Показувати подсказку не по щелчку, а по наведению курсора миші. Не совместимо з внешним вмістом.
Badge String Текстовий бейджик на елементе. Зазвичай показывается на иконке.
Width Length Ширина спливаючого вікна.
Placement PopupPlacement Положение спливаючого вікна. Можливі значення:
  • TopRight - (за замовчанням) сверху, розгортається праворуч.
  • TopLeft - сверху, розгортається ліворуч.
  • RightBottom - праворуч, розгортається вниз.
  • RightTop - праворуч, розгортається вверх.
  • BottomRight - снизу, розгортається праворуч.
  • BottomLeft - снизу, розгортається ліворуч.
  • LeftBottom - ліворуч, розгортається вниз.
  • LeftTop - ліворуч, розгортається вверх.
Underline PopoverUnderlineMode Стиль подчеркивания посилання при наведенні мыши. Можливі значення:
  • Enable - (за замовчанням) подчеркивать.
  • Disable - не подчеркивать.
  • ShowOnHover - подчеркивать при наведенні мыши.
Background PopoverBackgroundStyle Колір фона спливаючого вікна. Можливі значення (зверніть увагу, що конкретные коліра залежать від поточної теми користувацького інтерфейса):
  • #fffce9 Default - за замовчанням.
  • #fffce9 Yellow - желтый.
  • #edfcfe Cyan - голубой.
  • #f4ffd8 Green - зеленый.
  • #fedbdb Red - красный.
  • #e5f1fe Blue - синий.
  • #ffffff White - белый.
OffsetX Length Горизонтальне смещение спливаючого вікна. Може бути отрицательным.
Властивості базового елементу

Зверніть увагу! Якщо установлено властивість ShowOnHover, то размещать в елементе інші органы керування не має сенса. Всі дорівнює к ним неможна буде отримати доступ. Ведь, як тільки вы уберете курсор мыши з текста, всплывающее вікно исчезнет.

Крім того, властивість ShowOnHover не совместимо з можливістю получать вміст вікна з сервера (зовнішній Url) Це сделано з соображений производительности, щоб не звертатися до серверу при каждом движении мышью.

Результат

  • Слева, розгортається вверх
