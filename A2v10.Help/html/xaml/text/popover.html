Елемент Popover

Все текстовые елементи

Представляет собой ссылку, при щелчке (или наведении курсора мыши) на которой показывается всплывающее окно. Спливаюче вікно може содержать как простой текст або другой елемент, так и ссылаться на другой url.

Наследует: Inline : UIElement : UIElementBase

Властивість содержимого: Content

Конвертер: підтримуєся преобразование з строки текста и з любого елементу, наследующйого UIElement.

Властивості

Им'я Тип Опис
Content Object Властивість содержимого. Може бути простым текстом або любым елементом.
Url String Url для отримання содержимого с сервера. В цьому випадку всплывающее окно буде содержать собственную модель даних, собственный шаблон и, конечно, саму разметку.
Text String Текст ссылки.
Icon Icon Иконка для ссылки.
ShowOnHover Boolean Показывать подсказку не по щелчку, а по наведению курсора мыши. Не совместимо с внешним содержимым.
Badge String Текстовый бейджик на елементе. Зазвичай показывается на иконке.
Width Length Ширина спливаючого вікна.
Placement PopupPlacement Положение спливаючого вікна. Можливі значения:
  • TopRight - (за замовчанням) сверху, разворачивается вправо.
  • TopLeft - сверху, разворачивается влево.
  • RightBottom - справа, разворачивается вниз.
  • RightTop - справа, разворачивается вверх.
  • BottomRight - снизу, разворачивается вправо.
  • BottomLeft - снизу, разворачивается влево.
  • LeftBottom - слева, разворачивается вниз.
  • LeftTop - слева, разворачивается вверх.
Underline PopoverUnderlineMode Стиль подчеркивания ссылки при наведении мыши. Можливі значения:
  • Enable - (за замовчанням) подчеркивать.
  • Disable - не подчеркивать.
  • ShowOnHover - подчеркивать при наведении мыши.
Background PopoverBackgroundStyle Цвет фона спливаючого вікна. Можливі значення (обратите внимание, что конкретные цвета зависят від поточної темы пользовательского интерфейса):
  • #fffce9 Default - за замовчанням.
  • #fffce9 Yellow - желтый.
  • #edfcfe Cyan - голубой.
  • #f4ffd8 Green - зеленый.
  • #fedbdb Red - красный.
  • #e5f1fe Blue - синий.
  • #ffffff White - белый.
OffsetX Length Горизонтальное смещение спливаючого вікна. Може бути отрицательным.
Властивості базового елементу

Зверніть увагу! Якщо установлено властивість ShowOnHover, то размещать в елементе другие органы керування не має смысла. Все дорівнює к ним нельзя буде получить доступ. Ведь, как тільки вы уберете курсор мыши с текста, всплывающее окно исчезнет.

Кроме того, властивість ShowOnHover не совместимо с можливістью получать содержимое окна с сервера (зовнішній Url) Это сделано з соображений производительности, щоб не звертатися до серверу при каждом движении мышью.
