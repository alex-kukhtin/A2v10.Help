Елемент Popover

Всі текстові елементи

Являє собою посилання, при клікуе (або наведенні курсора миші) на яку виводиться спливаюче вікно. Спливаюче вікно може містити як звичайний текст або інший елемент, так і посилатися на інший url.

Наслідує: Inline : UIElement : UIElementBase

Властивість вмісту: Content

Конвертер: підтримується перетворення з рядку тексту і з будь-якого елементу, що наслідує UIElement.

Властивості

Ім'я Тип Опис
Content Object Властивість вмісту. Може бути звичайним текстом або будь-яким елементом.
Url String Url для отримання вмісту з сервера. В цьому випадку спливаюче вікно буде містити власну модель даних, власний шаблон і, звісно, саму розмітку.
Text String Текст посилання.
Icon Icon Іконка для посилання.
ShowOnHover Boolean Показувати підказку не по кліку, а по наведенню курсору миші. Несумісно з зовнішнім вмістом.
Badge String Текстовий бейджик на елементі. Зазвичай показується на іконці.
Width Length Ширина спливаючого вікна.
Placement PopupPlacement Позиція спливаючого вікна. Можливі значення:
  • TopRight - (за замовчанням) зверху, розгортається праворуч.
  • TopLeft - зверху, розгортається ліворуч.
  • RightBottom - праворуч, розгортається вниз.
  • RightTop - праворуч, розгортається вверх.
  • BottomRight - знизу, розгортається праворуч.
  • BottomLeft - знизу, розгортається ліворуч.
  • LeftBottom - ліворуч, розгортається вниз.
  • LeftTop - ліворуч, розгортається вверх.
Underline PopoverUnderlineMode Стиль підкреслення посилання при наведенні миші. Можливі значення:
  • Enable - (за замовчанням) підкреслювати.
  • Disable - не підкреслювати.
  • ShowOnHover - підкреслювати при наведенні миші.
Background PopoverBackgroundStyle Колір фону спливаючого вікна. Можливі значення (зверніть увагу, що конкреті кольори залежать від поточної теми користувацького інтерфейса):
  • #fffce9 Default - за замовчанням.
  • #fffce9 Yellow - жовтий.
  • #edfcfe Cyan - блакитний.
  • #f4ffd8 Green - зелений.
  • #fedbdb Red - червоний.
  • #e5f1fe Blue - синій.
  • #ffffff White - білий.
OffsetX Length Горизонтальне зміщення спливаючого вікна. Може бути від'ємним.
Властивості базового елементу

Зверніть увагу! Якщо установлено властивість ShowOnHover, то розміщувати в елементі інші органи керування нема сенсу. Все одно до них неможна буде отримати доступ: щойно ви приберете курсор миші з тексту, спливаюче вікно зникне.

Крім того, властивість ShowOnHover несумісна з можливістю отримувати вміст вікна з сервера (зовнішній Url) Це зроблено з міркувань продуктивності, щоб не звертатися до серверу при кожному русі курсору миші.

Результат

  • Зверху, розгортається ліворуч

  • Ліворуч, розгортається вверх
