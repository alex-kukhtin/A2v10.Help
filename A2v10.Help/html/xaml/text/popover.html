Елемент Popover

Все текстові елементи

Являє собою посилання, при щелчке (або наведении курсора мыши) на которой показывается всплывающее окно. Спливаюче вікно може містити як простий текст або другой елемент, так і посилатися на другой url.

Наследует: Inline : UIElement : UIElementBase

Властивість вмісту: Content

Конвертер: підтримується перетворення з строки текста і з любого елементу, нанаступного UIElement.

Властивості

Ім'я Тип Опис
Content Object Властивість вмісту. Може бути простым текстом або любым елементом.
Url String Url для отримання вмісту з сервера. В цьому випадку всплывающее окно буде містити собственную модель даних, собственный шаблон и, конечно, саму розмітку.
Text String Текст посилання.
Icon Icon Иконка для посилання.
ShowOnHover Boolean Показывать подсказку не по щелчку, а по наведению курсора мыши. Не совместимо з внешним содержимым.
Badge String Текстовый бейджик на елементе. Зазвичай показывается на иконке.
Width Length Ширина спливаючого вікна.
Placement PopupPlacement Положение спливаючого вікна. Можливі значення:
  • TopRight - (за замовчанням) сверху, розгортаєся вправо.
  • TopLeft - сверху, розгортаєся влево.
  • RightBottom - справа, розгортаєся вниз.
  • RightTop - справа, розгортаєся вверх.
  • BottomRight - снизу, розгортаєся вправо.
  • BottomLeft - снизу, розгортаєся влево.
  • LeftBottom - слева, розгортаєся вниз.
  • LeftTop - слева, розгортаєся вверх.
Underline PopoverUnderlineMode Стиль подчеркивания посилання при наведении мыши. Можливі значення:
  • Enable - (за замовчанням) подчеркивать.
  • Disable - не подчеркивать.
  • ShowOnHover - подчеркивать при наведении мыши.
Background PopoverBackgroundStyle Цвет фона спливаючого вікна. Можливі значення (зверніть увагу, що конкретные цвета залежать від поточної темы користувацького інтерфейса):
  • #fffce9 Default - за замовчанням.
  • #fffce9 Yellow - желтый.
  • #edfcfe Cyan - голубой.
  • #f4ffd8 Green - зеленый.
  • #fedbdb Red - красный.
  • #e5f1fe Blue - синий.
  • #ffffff White - белый.
OffsetX Length Горизонтальное смещение спливаючого вікна. Може бути отрицательным.
Властивості базового елементу

Зверніть увагу! Якщо установлено властивість ShowOnHover, то размещать в елементе інші органы керування не має сенса. Все дорівнює к ним нельзя буде отримати доступ. Ведь, як тільки вы уберете курсор мыши з текста, всплывающее окно исчезнет.

Крім того, властивість ShowOnHover не совместимо з можливістю получать вміст вікна з сервера (зовнішній Url) Це сделано з соображений производительности, щоб не звертатися до серверу при каждом движении мышью.
