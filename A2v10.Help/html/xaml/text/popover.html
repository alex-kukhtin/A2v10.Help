Элемент Popover

Все текстовые элементы

Представляет собой ссылку, при щелчке (или наведении курсора мыши) на которой показывается всплывающее окно. Всплывающее окно может содержать как простой текст или другой элемент, так и ссылаться на другой url.

Наследует: Inline : UIElement : UIElementBase

Свойство содержимого: Content

Конвертер: поддерживается преобразование из строки текста и из любого элемента, наследующего UIElement.

Свойства

Имя Тип Описание
Content Object Свойство содержимого. Может быть простым текстом или любым элементом.
Url String Url для получения содержимого с сервера. В этом случае всплывающее окно будет содержать собственную модель данных, собственный шаблон и, конечно, саму разметку.
Text String Текст ссылки.
Icon Icon Иконка для ссылки.
ShowOnHover Boolean Показывать подсказку не по щелчку, а по наведению курсора мыши. Не совместимо с внешним содержимым.
Badge String Текстовый бейджик на элементе. Обычно показывается на иконке.
Width Length Ширина всплывающего окна.
Placement PopupPlacement Положение всплывающего окна. Возможные значения:
  • TopRight - (по умолчанию) сверху, разворачивается вправо.
  • TopLeft - сверху, разворачивается влево.
  • RightBottom - справа, разворачивается вниз.
  • RightTop - справа, разворачивается вверх.
  • BottomRight - снизу, разворачивается вправо.
  • BottomLeft - снизу, разворачивается влево.
  • LeftBottom - слева, разворачивается вниз.
  • LeftTop - слева, разворачивается вверх.
Underline PopoverUnderlineMode Стиль подчеркивания ссылки при наведении мыши. Возможные значения:
  • Enable - (по умолчанию) подчеркивать.
  • Disable - не подчеркивать.
  • ShowOnHover - подчеркивать при наведении мыши.
Background PopoverBackgroundStyle Цвет фона всплывающего окна. Возможные значения (обратите внимание, что конкретные цвета зависят от текущей темы пользовательского интерфейса):
  • #fffce9 Default - по умолчанию.
  • #fffce9 Yellow - желтый.
  • #edfcfe Cyan - голубой.
  • #f4ffd8 Green - зеленый.
  • #fedbdb Red - красный.
  • #e5f1fe Blue - синий.
  • #ffffff White - белый.
OffsetX Length Горизонтальное смещение всплывающего окна. Может быть отрицательным.
Свойства базового элемента

Обратите внимание! Если установлено свойство ShowOnHover, то размещать в элементе другие органы управления не имеет смысла. Все равно к ним нельзя будет получить доступ. Ведь, как только вы уберете курсор мыши с текста, всплывающее окно исчезнет.

Кроме того, свойство ShowOnHover не совместимо с возможностью получать содержимое окна с сервера (внешний Url) Это сделано из соображений производительности, чтобы не обращаться к серверу при каждом движении мышью.
