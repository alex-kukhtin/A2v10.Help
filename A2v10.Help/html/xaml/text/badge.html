Элемент Badge

Все текстовые элементы

Представляет собой бейджик с вспомогательным текстом (например, количество элементов).

Наследует: Inline : UIElement : UIElementBase

Свойство содержимого: Content

Свойства

Имя Тип Описание
Content String Свойство содержимого. Текст бейджика.
Small Boolean Уменьшенный шрифт. Конкретный размер шрифта зависит от темы пользовательского интерфейса.
Свойства базового элемента

Обратите внимание! Некоторые элементы, например Header или Tab, имеют свойство Badge, которое фактически вставляет элемент Badge в нужное место.
