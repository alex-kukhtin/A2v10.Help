Елемент TagLabel

Всі текстові елементи

Являє собою текстову мітку-маркер. Найчастіше використовується для відображення стану чйого-либо.

Наслідує: Inline : UIElement : UIElementBase

Властивість вмісту: Content

Властивості

Ім'я Тип Опис
Content String Властивість вмісту. Текстовое вміст мітки
Style Enum Стиль відображення мітки. Можливі значення:
Default Green Success Warning Orange Info Cyan
Error Danger Red Purple Pink Gold Blue
Salmon Seagreen Tan Magenta LightGray
