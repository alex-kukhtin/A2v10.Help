Прив'язка Bind

Все привязки

Представляет собой специальный об'єкт, що описує связь властивості с елементом даних. Практически завжди задається як розширення розмітки.

Прив'язка завжди является двухсторонней. Это означает, что при изменении значення в моделі, связанное властивість меняется автоматически і наоборот.

Привязывать можна практически будь-які властивості любых елементів. Зверніть увагу, что властивості привязки в свою очередь тоже можуть быть привязками.

Властивості

Им'я Тип Опис
Path String За замовчанням. Выражение, описывающее шлях до властивості моделі.
DataType DataType Тип даних.
Format String Формат отображения.
Mask String Маска отображения значения.
HideZeros Boolean Прятать нулевые значения. Має сенс тільки для числовых форматов.
NegativeRed Boolean Показывать отрицательные значення красным цветом. Має сенс тільки для числовых форматов.
Filters FilterCollection Фильтры для обработки связанного значення після оновлення. Представляет собой список перечислений (через запятую). Можливі значення:
  • Trim - обрезать пробелы в начале і в конце текста.
  • Upper - преобразовать символы в верхний регистр.
  • Lower - преобразовать символы в нижний регистр.
  • Barcode - використовується для алфавитно-цифровых кодов. Заменяет кириллические буквы на латинские у відповідності до раскладкой клавиатуры.
