Привязка Bind

Все привязки

Представляет собой специальный объект, описывающий связь свойства с элементом данных. Практически всегда задается как расширение разметки.

Привязка всегда является двухсторонней. Это означает, что при изменении значения в модели, связанное свойство меняется автоматически и наоборот.

Привязывать можно практически любые свойства любых элементов. Обратите внимание, что свойства привязки в свою очередь тоже могут быть привязками.

Свойства

Имя Тип Описание
Path String По умолчанию. Выражение, описывающее путь к свойству модели.
DataType DataType Тип данных.
Format String Формат отображения.
Mask String Маска отображения значения.
HideZeros Boolean Прятать нулевые значения. Имеет смысл только для числовых форматов.
NegativeRed Boolean Показывать отрицательные значения красным цветом. Имеет смысл только для числовых форматов.
Filters FilterCollection Фильтры для обработки связанного значения после обновления. Представляет собой список перечислений (через запятую). Возможные значения:
  • Trim - обрезать пробелы в начале и в конце текста.
  • Upper - преобразовать символы в верхний регистр.
  • Lower - преобразовать символы в нижний регистр.
  • Barcode - используется для алфавитно-цифровых кодов. Заменяет кириллические буквы на латинские в соответствии с раскладкой клавиатуры.
