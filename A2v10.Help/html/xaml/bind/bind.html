Прив'язка Bind

Все прив`язки

Являє собою специальный об'єкт, що описує связь властивості з елементом даних. Практически завжди задається як розширення розмітки.

Прив'язка завжди є двухсторонней. Це означає, що при изменении значення в моделі, связанное властивість меняется автоматично і навпаки.

Привязывать можна практически будь-які властивості любых елементів. Зверніть увагу, що властивості прив`язки в свою очередь тоже можуть бути привязками.

Властивості

Ім'я Тип Опис
Path String За замовчанням. Выражение, описывающее шлях до властивості моделі.
DataType DataType Тип даних.
Format String Формат відображення.
Mask String Маска відображення значения.
HideZeros Boolean Прятать нулевые значения. Має сенс тільки для числовых форматов.
NegativeRed Boolean Показывать отрицательные значення красным цветом. Має сенс тільки для числовых форматов.
Filters FilterCollection Фильтры для обробки связанного значення після оновлення. Являє собою список перечислений (через запятую). Можливі значення:
  • Trim - обрезать пробелы в начале і в конце текста.
  • Upper - преобразовать символы в верхний регистр.
  • Lower - преобразовать символы в нижний регистр.
  • Barcode - використовується для алфавитно-цифровых кодов. Заменяет кириллические буквы на латинские у відповідності до раскладкой клавиатуры.
