Абстрактні елементи

Абстрактні елементи призначені для наслідування іншими елементами. Їх неможна створювати безпосередньо в розмітці.

Абстрактні елементи

  • CheckBoxBase - елемент керування для позначення.
  • Control - узагальнений елемент керування.
  • CommandControl - командний елемент керування.
  • Container - узагальнений контейнер.
  • ContentControl - елемент керування, який підтримує вміст.
  • Inline - рядковий елемент.
  • RootContainer - кореневий контейнер.
  • ValuedControl - елемент керування, який підтримує значення.
  • UIContentElement - базовий елемент UI з вмістом.
  • UIElementBase - базовий елемент UI.
  • UIElement - елемент UI.
