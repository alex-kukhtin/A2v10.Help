Абстрактные элементы

Абстрактные элементы предназначены для наследования другими элементами. Их нельзя создать непосредственно в разметке.

Абстрактные элементы

  • CheckBoxBase - отмечаемый элемент управления.
  • Control - обобщенный элемент управления.
  • CommandControl - командный элемент управления.
  • Container - обобщенный контейнер.
  • ContentControl - элемент управления, который поддерживает содержимое.
  • Inline - строчный элемент.
  • RootContainer - корневой контейнер.
  • ValuedControl - элемент управления, который поддерживает значение.
  • UIContentElement - базовый элемент UI с содержимым.
  • UIElementBase - базовый элемент UI.
  • UIElement - элемент UI.
