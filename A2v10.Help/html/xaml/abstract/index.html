Абстрактные елементи

Абстрактные елементи предназначены для наследования другими елементуми. Их нельзя создать безпосередньо в розмітці.

Абстрактные елементи

  • CheckBoxBase - отмечаемый елемент керування.
  • Control - обобщенный елемент керування.
  • CommandControl - командный елемент керування.
  • Container - обобщенный контейнер.
  • ContentControl - елемент керування, который підтримує вміст.
  • Inline - строчный елемент.
  • RootContainer - кореневий контейнер.
  • ValuedControl - елемент керування, который підтримує значення.
  • UIContentElement - базовий елемент UI з содержимым.
  • UIElementBase - базовий елемент UI.
  • UIElement - елемент UI.
