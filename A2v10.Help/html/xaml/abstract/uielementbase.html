Абстрактний елемент UIElementBase

Всі абстрактні елементи

Являє собою базовий елемент користувацького інтерфейсу.

Властивості

Ім'я Тип Опис
If Boolean? Включати елемент в потік рендеринга.
Show Boolean? Відображати елемент. Інверсія властивості Hide.
Hide Boolean? Приховати елемент. Інверсія властивості Show.
Margin Thickness Відступи для елементу.
Padding Thickness Поля для елементу.
Wrap WrapMode Перенесення тексту.
Absolute Thickness Абсолютне позиціонування елементу.
Tip String Спливаюча підказка для елементу.
Render RenderMode? Чи включати елемент в підсумкову розмітку і як саме це робити.
На відміну від властивості If, ця властивість працює на сервері в момент формуваня розмітки.
HtmlId String Ідентифікатор (html атрибут id) для елементу. Зазвичай використовується для тестування або для навігації по елементах за допомогою методу $focus контролера IController.
XamlStyle StyleDescriptor Стиль елементу. Завжди повинен бути прив'язкою StyleResource.
