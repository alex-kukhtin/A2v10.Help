Абстрактный элемент UIElementBase

Все абстрактные элементы

Представляет собой базовый элемент пользовательского интерфейса.

Свойства

Имя Тип Описание
If Boolean? Включать элемент в поток рендеринга.
Show Boolean? Показывать элемент. Инверсия свойства Hide.
Hide Boolean? Скрывать элемент. Инверсия свойства Show.
Margin Thickness Отступы для элемента.
Padding Thickness Поля для элемента.
Wrap WrapMode Перенос текста.
Absolute Thickness Абсолютное позиционирование элемента.
Tip String Всплывающая подсказка для элемента.
Render RenderMode? Включать ли элемент в итоговую разметку и как именно это делать.
В отличие от свойства If, это свойство работает на сервере в момент формирования разметки.
HtmlId String Идентификатор (html атрибут id) для элемента. Обычно используется для тестирования либо для навигации по элементам с помощью метода $focus контроллера IController.
XamlStyle StyleDescriptor Стиль элемента. Всегда должен быть привязкой StyleResource.
