Абстрактный елемент UIElementBase

Все абстрактные елементи

Являє собою базовий елемент користувацького інтерфейса.

Властивості

Ім'я Тип Опис
If Boolean? Включать елемент в потік рендеринга.
Show Boolean? Показывать елемент. Инверсия властивості Hide.
Hide Boolean? Скрывать елемент. Инверсия властивості Show.
Margin Thickness Отступы для елементу.
Padding Thickness Поля для елементу.
Wrap WrapMode Перенос текста.
Absolute Thickness Абсолютное позиционирование елементу.
Tip String Всплывающая подсказка для елементу.
Render RenderMode? Включать ли елемент в итоговую розмітку і як именно це делать.
В отличие від властивості If, це властивість работает на сервере в момент формуваня розмітки.
HtmlId String Ідентифікатор (html атрибут id) для елементу. Зазвичай використовується для тестування або для навигации по елементум з помощью метода $focus контролера IController.
XamlStyle StyleDescriptor Стиль елементу. Завжди повинен бути прив'язкою StyleResource.
