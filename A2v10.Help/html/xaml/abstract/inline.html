Абстрактный элемент Inline

Все абстрактные элементы

Представляет собой элемент текста (строчный элемент)

Наследует: UIElement : UIElementBase

Свойства

Имя Тип Описание
Block Boolean Отображать элемент как блочный.
Float FloatMode Сделать элемент "плавающим".
  • None - (по умолчанию) нет.
  • Left - разместить слева.
  • Right - разместить справа.
Color TextColor Цвет текста.
Свойства базового элемента

Особенности

Все элементы, которые в качестве содержимого используют коллекцию Inline ( например Text или Paragraph), могут включать как дочерние элементы, так и из простые текстовые строки. Текстовые строки просто переносятся в разметку (никакие дополнительные теги не добавляются).
