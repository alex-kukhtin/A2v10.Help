Абстрактный елемент Inline

Все абстрактные елементи

Являє собою елемент текста (строчный елемент)

Наследует: UIElement : UIElementBase

Властивості

Ім'я Тип Опис
Block Boolean Отображать елемент як блочный.
Float FloatMode Сделать елемент "плавающим".
  • None - (за замовчанням) нет.
  • Left - разместить слева.
  • Right - разместить справа.
Color TextColor Цвет текста.
Властивості базового елементу

Особливості

Все елементи, які в якості вмісту используют коллекцию Inline ( наприклад Text або Paragraph), можуть включать як дочірні елементи, так і з простые текстові строки. Текстовые строки просто переносятся в розмітку (никакие додаткові теги не додаються).
