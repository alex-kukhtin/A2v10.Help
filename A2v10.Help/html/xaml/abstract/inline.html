Абстрактний елемент Inline

Всі абстрактні елементи

Являє собою елемент тексту (рядковий елемент)

Наслідує: UIElement : UIElementBase

Властивості

Ім'я Тип Опис
Block Boolean Відображати елемент як блочний.
Float FloatMode Зробити елемент "плаваючим".
  • None - (за замовчанням) ні.
  • Left - розмістити ліворуч.
  • Right - розмістити праворуч.
Color TextColor Колір тексту.
Властивості базового елементу

Особливості

Всі елементи, які в якості вмісту використовують колекцію Inline ( наприклад Text або Paragraph), можуть містити як дочірні елементи, так і прості текстові рядки. Текстові рядки просто переносяться в розмітку (жодні додаткові теги не додаються).
