Абстрактний елемент Control

Всі абстрактні елементи

Являє собою абстрактний елемент керування.

Наслідує: UIElement : UIElementBase

Похідні елементи: ValuedControl, ContentControl, List, DataGrid, TreeView, Table

Властивості

Ім'я Тип Опис
Block Boolean Виводить елемент керування у вигляді блочного, а не рядкового елементу.
Label String Текстова мітка для елементу керування. Зазвичай виводиться над полем або ліворуч від поля введення.
Description String Опис для елементу керування. Зазвичай виводиться під полем введення.
Required Boolean Обов'язкове значення для введення. Зверніть увагу, що це тільки візуальне відображення обов'язкового вмісту поле введення (конкретний вигляд залежить від теми користувацького інтерфейсу). Для керування перевірками використовуйте валідатори шаблонів.
Disabled Boolean Елемент керування недоступний для введення. Для керування цим полем зазвичай використовується прив'язка Bind.
Width Length Ширина елементу керування. За замовчанням елемент займає 100% ширини батьківського елементу. Для деяких контейнерів (наприклад Toolbar) ширина вкладених елементів буде встановлена в певне значення за замовчанням.
TabIndex Int32 Порядковий номер елементу в ланцюзі переходів по клавіші Tab. Значення "-1" виключає елемент з ланцюга переходів.
TestId String Ідентифікатор елементу для інструментів тестування.
Popover Popover Спливаюча підказка для елементу. Виводиться у вигляді посилання праворуч від елементу.
Hint Popover Спливаюча підказка для елементу. Виводиться у вигляді після мітки. Іконку можна замінити, вказавши властивість Icon об'єкта Popover.
Validator Validator Дозволяє задати властивості валідатора елементу, такі як позицію або ширину.
AddOns UIElementCollection Додаткові елементи керування. Зазвичай це гіперпосилання, які виконують якісь додаткові команди.
Link UIElement Додаткове просилання, пов'язане з елементом. Відображається в правому верхньому куті елементу. Зазвичай це гіперпосилання.
