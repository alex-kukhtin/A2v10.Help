Абстрактный елемент Control

Все абстрактные елементи

Являє собою абстрактний елемент керування.

Наследует: UIElement : UIElementBase

Похідні елементи: ValuedControl, ContentControl, List, DataGrid, TreeView, Table

Властивості

Ім'я Тип Опис
Block Boolean Выводить елемент керування у вигляді блочного, а не строчного елементу.
Label String Текстовая метка для елементу керування. Зазвичай выводится сверху або слева від поля ввода.
Description String Опис для елементу керування. Зазвичай выводится под полем ввода.
Required Boolean Обов'язковое значення для ввода. Зверніть увагу, що це тільки визуальное вид обов'язкового ввода в поле ввода (конкретный вид зависит від темы користувацького інтерфейса). Для керування перевірками используйте валідатори шаблонов.
Disabled Boolean Елемент керування недоступен для ввода. Для керування этим полем обычно використовується прив'язка Bind.
Width Length Ширина елементу керування. За замовчанням елемент занимает 100% ширины батьківського елементу. Для некоторых контейнеров (наприклад Toolbar) ширина вложенных елементів буде установлена в некоторое значення за замовчанням.
TabIndex Int32 Порядковый номер елементу в цепочке переходов по клавише Tab. Значення "-1" исключает елемент з цепочки переходов.
TestId String Ідентифікатор елементу для інструментів тестування.
Popover Popover Всплывающая подсказка для елементу. Выводится у вигляді посилання справа від елементу.
Hint Popover Всплывающая подсказка для елементу. Выводится у вигляді після метки. Иконку можна заменить, указав властивість Icon об'єкта Popover.
Validator Validator Позволяет задать властивості валідатора елементу, такі як положение або ширину.
AddOns UIElementCollection Додаткові елементи керування. Зазвичай це гиперпосилання, які выполняют якісь додаткові команди.
Link UIElement Додаткова ссылка, пов'язана з елементом. Отображается в правом верхнем углу елементу. Зазвичай це гиперссылка.
