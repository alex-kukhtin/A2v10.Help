Абстрактний елемент ValuedControl

Всі елементи керування

Являє собою абстрактний елемент керування, що містить значення.

Наслідує: Control : UIElement : UIElementBase

Властивості

Ім'я Тип Опис
Value Object Тільки прив'язка (Bind). Прив'язка до значення моделі даних.
ValidateValue Object Тільки прив'язка (Bind). Прив'язка до значення для валідації. Зазначається тільки у випадку, якщо валідатор потрібно зв'язувати не зі заченнями в елементі, а з якоюсь іншою властивістю моделі.
