Абстрактный елемент Container

Все абстрактные елементи

Елемент Container являє собою абстрактный контейнер. Властивість Children задает список дочірніх елементів.

Наследует: UIElement : UIElementBase

Производные елементи: CardBody Toolbar, StackPanel, Splitter, DropDownMenu, DropDownMegaMenu, FieldSet, Flex, Case, TaskPad, Group, GridGroup, Tab, FullHeightPanel, CommandBar, FlexList, Panel, Grid, Card, SheetPage

Властивість содержимого: Children

Властивості

Им'я Тип Опис
Children UIElementCollection Властивість содержимого. Коллекция дочірніх елементів.
ItemsSource Array Всегда привязка (Bind). Джерело даних для отримання дочірніх елементів.
