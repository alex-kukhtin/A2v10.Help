Абстрактный элемент Container

Все абстрактные элементы

Элемент Container представляет собой абстрактный контейнер. Свойство Children задает список дочерних элементов.

Наследует: UIElement : UIElementBase

Производные элементы: CardBody Toolbar, StackPanel, Splitter, DropDownMenu, DropDownMegaMenu, FieldSet, Flex, Case, TaskPad, Group, GridGroup, Tab, FullHeightPanel, CommandBar, FlexList, Panel, Grid, Card, SheetPage

Свойство содержимого: Children

Свойства

Имя Тип Описание
Children UIElementCollection Свойство содержимого. Коллекция дочерних элементов.
ItemsSource Array Всегда привязка (Bind). Источник данных для получения дочерних элементов.
