Елемент Block

Все контейнеры

Являє собою контейнер, які містить інші елементи керування або інші контейнеры.

Наследует: UIElement : UIElementBase

Властивість вмісту: Children

Властивості

Ім'я Тип Опис
Children UIElementCollection Властивість вмісту. Колекція дочірніх елементів.
Align TextAlign Горизонтальное выравнивание вмісту контейнера.
Color TextColor Цвет текста.
Background BackgroundStyle Цвет фона.
Height Length Высота блока.
Width Length Ширина блока. За замовчанням блок занимает 100% ширины батьківського елементу.
MaxWidth Length Максимальная ширина блока. Якщо вміст шире, то оно переносится.
Border Boolean Отображать рамку для блока.
Scroll Boolean? Допускает ли блок прокрутку. Має сенс тільки, якщо установлена высота блока. Якщо установлено в True, то буде появляться полоса прокрутки. Якщо установлено в False, то лишнее вміст буде обрезано. За замовчанням высота блока буде такой, щоб включать всі вложенные елементи.
DropShadow ShadowStyle Вид тени.
Relative Boolean Относительное позиционирование для блока. Все дочірні елементи с установленным властивістю Absolute будуть позиционироваться відносно цього блока.
