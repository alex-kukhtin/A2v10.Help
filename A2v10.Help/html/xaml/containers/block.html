Елемент Block

Всі контейнери

Являє собою контейнер, який містить інші елементи керування, або інші контейнери.

Наслідує: UIElement : UIElementBase

Властивість вмісту: Children

Властивості

Ім'я Тип Опис
Children UIElementCollection Властивість вмісту. Колекція дочірніх елементів.
Align TextAlign Горизонтальне вирівнювання вмісту контейнера.
Color TextColor Колір тексту.
Background BackgroundStyle Колір фону.
Height Length Висота блоку.
Width Length Ширина блоку. За замовчуванням блок займає 100% ширини батьківського елементу.
MaxWidth Length Максимальна ширина блоку. Якщо вміст ширше, то він переноситься.
Border Boolean Відображати рамку для блоку.
Scroll Boolean? Чи дозволяє блок прокрутку. Має сенс тільки тоді, коли встановлено висоту блоку. Якщо встановлено в True, то буде з'являтися смуга прокрутки. Якщо встановлено в False, то зайвий вміст буде обрізано. За замовчуванням висота блоку буде такою, щоб вмістити всі вкладені елементи.
DropShadow ShadowStyle Вид тіні.
Relative Boolean Відносне позиціонування для блоку. Всі дочірні елементи з встановленою властивістю Absolute будуть позиціонуватися відносно цього блоку.
