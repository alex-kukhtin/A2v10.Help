Элемент Block

Все контейнеры

Представляет собой контейнер, которые содержит другие элементы управления или другие контейнеры.

Наследует: UIElement : UIElementBase

Свойство содержимого: Children

Свойства

Имя Тип Описание
Children UIElementCollection Свойство содержимого. Коллекция дочерних элементов.
Align TextAlign Горизонтальное выравнивание содержимого контейнера.
Color TextColor Цвет текста.
Background BackgroundStyle Цвет фона.
Height Length Высота блока.
Width Length Ширина блока. По умолчанию блок занимает 100% ширины родительского элемента.
MaxWidth Length Максимальная ширина блока. Если содержимое шире, то оно переносится.
Border Boolean Отображать рамку для блока.
Scroll Boolean? Допускает ли блок прокрутку. Имеет смысл только, если установлена высота блока. Если установлено в True, то будет появляться полоса прокрутки. Если установлено в False, то лишнее содержимое будет обрезано. По умолчанию высота блока будет такой, чтобы включать все вложенные элементы.
DropShadow ShadowStyle Вид тени.
Relative Boolean Относительное позиционирование для блока. Все дочерние элементы с установленным свойством Absolute будут позиционироваться относительно этого блока.
