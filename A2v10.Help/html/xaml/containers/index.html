Контейнери

Контейнерні елементи або просто контейнери, що використовуються для розміщення всередині себе інших елементів.

Контейнери

  • Block - блок.
  • Card - картка.
  • CardBody - вміст картки.
  • Case - один варіант елементу вибору.
  • CommandBar - панель команд.
  • Dialog - діалог.
  • DropDownMenu - спадне меню.
  • DropDownMegaMenu - спадне меню з розділами.
  • FieldSet - панель для елементів.
  • Flex - контейнер flexbox.
  • FlexList - перелік (flexbox) з гнучким розміщенням елементів.
  • FullHeightPanel - панель повної висоти.
  • Grid - сітка.
  • GridGroup - група елементів сітки.
  • IFrame - фрейм.
  • Include - вкладений вміст.
  • InlineDialog - вбудований діалог.
  • Page - сторінка.
  • Panel - панель.
  • Partial - частина сторінки.
  • Popup - спливне вікно.
  • Repeater - блок, що повторюється.
  • Splitter - роздільник.
  • StackPanel - панель.
  • Tab - одна закладка.
  • TabBar - панель вибору у вигляді закладок.
  • TabPanel - панель із закладками.
  • Taskpad - панель задач.
  • Toolbar - панель інструментів.
  • Wizard - діалог у вигляді "майстеру".
  • WizardPage - сторінка "майстеру".
