Контейнеры

Контейнерные элементы или просто контейнеры используются для размещения внутри себя других элементов.

Контейнеры

  • Block - блок.
  • Card - карточка.
  • CardBody - содержимое карточки.
  • Case - один вариант элемента выбора.
  • CommandBar - панель команд.
  • Dialog - диалог.
  • DropDownMenu - выпадающее меню.
  • DropDownMegaMenu - выпадающее меню с разделами.
  • FieldSet - панель для элементов.
  • Flex - контейнер flexbox.
  • FlexList - список (flexbox) с гибким размещением элементов.
  • FullHeightPanel - панель полной высоты.
  • Grid - сетка.
  • GridGroup - группа элементов сетки.
  • IFrame - фрейм.
  • Include - вложенное содержимое.
  • InlineDialog - встроенный диалог.
  • Page - страница.
  • Panel - панель.
  • Partial - часть страницы.
  • Popup - всплывающее окно.
  • Repeater - повторяющийся блок.
  • Splitter - разделитель.
  • StackPanel - панель.
  • Tab - одна закладка.
  • TabBar - панель выбора в виде закладок.
  • TabPanel - панель с закладками.
  • Taskpad - панель задач.
  • Toolbar - панель инструментов.
  • Wizard - диалог в виде "мастера".
  • WizardPage - страница мастера.
