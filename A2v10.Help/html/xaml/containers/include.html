Елемент Include

Все контейнеры

Являє собою контейнер, в якому отображается полная независимая сторінка. Частіше за всі використовується для динамического выбора страницы в залежності від властивостей какого-либо об'єкта. Корневым елементом такой страницы обычно буде елемент Partial.

Наследует: UIElementBase

Властивості

Им'я Тип Опис
Source String URL вмісту страницы. Може бути привязкой.
Argument Object Аргумент (обычно це прив'язка Bind) для формирования идентификатора об'єкта в Url.
Data Object Об'єкт, властивості якого будуть переданы в Url в якості параметров.
FullHeight Boolean Якщо властивість установлено в True, то контейнер буде занимать всю родительскую страницу по высоте.
