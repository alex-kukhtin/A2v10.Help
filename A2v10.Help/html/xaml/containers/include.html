Елемент Include

Всі контейнери

Являє собою контейнер, в якому відображається повна незалежна сторінка. Частіше за все використовується для динамічного вибору сторінки в залежності від властивостей якогось об'єкта. Кореневим елементом такої сторінки зазвичай буде елемент Partial.

Наслідує: UIElementBase

Властивості

Ім'я Тип Опис
Source String URL вмісту сторінки. Може бути прив'язкою.
Argument Object Аргумент (зазвичай це прив'язка Bind) для формуваня ідентифікатора об'єкту в Url.
Data Object Об'єкт, властивості якого будуть передані в Url в якості параметрів.
FullHeight Boolean Якщо властивість встановлено в True, то контейнер буде займати всю батьківську сторінку по висоті.
Властивості базового елементу
