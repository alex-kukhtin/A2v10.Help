Элемент Include

Все контейнеры

Представляет собой контейнер, в котором отображается полная независимая страница. Чаще всего используется для динамического выбора страницы в зависимости от свойств какого-либо объекта. Корневым элементом такой страницы обычно будет элемент Partial.

Наследует: UIElementBase

Свойства

Имя Тип Описание
Source String URL содержимого страницы. Может быть привязкой.
Argument Object Аргумент (обычно это привязка Bind) для формирования идентификатора объекта в Url.
Data Object Объект, свойства которого будут переданы в Url в качестве параметров.
FullHeight Boolean Если свойство установлено в True, то контейнер будет занимать всю родительскую страницу по высоте.
