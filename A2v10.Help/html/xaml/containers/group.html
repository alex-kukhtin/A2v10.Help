Элемент Group

Все контейнеры

Представляет собой группу элементов. Элемент не имеет внешнего представления и предназначен для того, чтобы разместить несколько элементов в том месте, где допустим только один элемент, а также для того, чтобы управлять видимостью/наличием всех элементов в группе одновременно (с помощью свойств If, Show).

Наследует: Container : UIElement : UIElementBase

Элемент не содержит собственных свойств. Свойства базового элемента.
