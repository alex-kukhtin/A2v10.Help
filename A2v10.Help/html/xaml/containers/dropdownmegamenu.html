Елемент DropDownMegaMenu

Всі контейнери

Являє собою спадне меню з розділами.

Зверніть увагу, що групування меню доступне тільки для зв'язаних джерел даних. Групування статичних елементів меню не підтримується.

Наслідує: Container : UIElement : UIElementBase

Властивості

Ім'я Тип Опис
GroupBy String Ім'я властивості в джерелі даних, по якому буде виконуватися групування. Кожна група меню буде показана в окремому розділі.
Columns Int32 Кількість стовпчиків в меню. За замовчанням - один стовпчик.
Width Length Ширина меню.
Direction DropDownDirection Напрямок спадання меню. Можливі значення:
  • DownRight - вниз, розгортається праворуч (за замовчанням). Прив'язка лівого краю.
  • DownLeft - вниз, розгортається ліворуч. Прив'язка правого краю.
  • UpRight - вверх, розгортається праворуч. Прив'язка лівого краю.
  • UpLeft - вверх, розгортається ліворуч. Прив'язка правого краю.
Separate Boolean Показувати меню окремо від батьківської кнопки. В цьому випадку фон елементу буде білим.
Background BackgroundStyle Колір фону.
