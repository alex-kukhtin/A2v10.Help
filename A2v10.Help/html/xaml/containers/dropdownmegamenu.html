Елемент DropDownMegaMenu

Все контейнеры

Являє собою выпадающее меню c розділуми.

Зверніть увагу, що группировка меню досупна тільки для связанных джерелоов даних. Группировка статических елементів меню не підтримується.

Наследует: Container : UIElement : UIElementBase

Властивості

Им'я Тип Опис
GroupBy String Им'я властивості в джерелі даних, по которому буде виконуватися группировка. Кожна группа меню буде показана в отдельном разделе.
Columns Int32 Количество колонок в меню. За замовчанням - одна колонка.
Width Length Ширина меню.
Direction DropDownDirection Направление "выпадения" меню. Можливі значення:
  • DownRight - (за замовчанням) вниз, розгортаєся вправо. Прив'язка левого края.
  • DownLeft - вниз, розгортаєся влево. Прив'язка правого края.
  • UpRight - вверх, розгортаєся вправо. Прив'язка левого края.
  • UpLeft - вверх, розгортаєся влево. Прив'язка правого края.
Separate Boolean Показывать меню раздельно від родительской кнопки. В цьому випадку фон елементу буде белым.
Background BackgroundStyle Цвет фона.
