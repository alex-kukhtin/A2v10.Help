Елемент StackPanel

Всі контейнери

Являє собою контейнер, в якому дочірні елементи вирівнюються в одну лінію, орієнтовану вертикально або горизонтально.

Наслідує: Container : UIElement : UIElementBase

Властивості

Ім'я Тип Опис
Orientation Orientation Розташування елементів в панелі.
AlignItems AlignItems Вирівнювання дочірніх елементів.
JustifyItems JustifyItems Розподіл дочірніх елементів.
Inline Boolean Рядковий елемент (не блочний).
Gap GapSize Проміжки між елементами по основній та перпендикулярній осі.
Властивості базового елементу

Роздільники

Дочірні елементи панелі можуть містити роздільник Separator.

Вставка роздільника призведе до того, що вміст розделіться на два блоки. Все, що буде до роздільника, вирівняється по лівій (або верхній) стороні панелі, а те, що після роздільника - по правій (або нижній).
