Элемент DropDownMenu

Все контейнеры

Представляет собой выпадающее меню. Содержимое обычно представляет собой коллекцию элементов MenuItem.

Наследует: Container : UIElement : UIElementBase

Свойства

Имя Тип Описание
Direction DropDownDirection Направление "выпадения" меню. Возможные значения:
  • DownRight - (по умолчанию) вниз, разворачивается вправо. Привязка левого края.
  • DownLeft - вниз, разворачивается влево. Привязка правого края.
  • UpRight - вверх, разворачивается вправо. Привязка левого края.
  • UpLeft - вверх, разворачивается влево. Привязка правого края.
Separate Boolean Показывать меню раздельно от родительской кнопки. В этом случае фон элемента будет белым.
Background BackgroundStyle Цвет фона.
