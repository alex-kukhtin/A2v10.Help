Елемент IFrame

Всі контейнери

Являє собою фрейм для завантаження і відображення зовнішнього незалежного документа. Відображується в html тегу <irame>.

Наслідує: UIElementBase

Властивості

Ім'я Тип Опис
Source String Url сторінки, яка буде завантажена у фрейм
Size Size Розмір фрейма. За замовчанням - "100% 100%"
