Елемент InlineDialog

Все контейнеры

Являє собою діалог, который працює з поточної моделью і не требует додатковой завантаження з сервера.

Кожен такой діалог повинен мати уникальный (в пределах поточної страницы) текстовый ідентифікатор.

Діалог працює в пределах поточної страницы. Фактично він являє собою простий блок, который отображается по команде у вигляді діалога. Такие возможнасти как, наприклад, отмену дії потрібно реалізовувати самостоятельно.

Открыть або закрыть діалог можна командами контролера $inlineOpen, $inlineClose передав им ідентифікатор діалога.

Наследует: Container : UIElement : UIElementBase

Властивості

Ім'я Тип Опис
Id String Уникальный (в пределах страницы) ідентифікатор діалога.
Title String Заголовок вікна діалога.
Width Length Ширина вікна діалога. За замовчанням визначається содержимым.
Buttons UIElementCollection Вміст панели кнопок. Зазвичай набір управляющих кнопок.
NoClose Boolean Запретить закрытие діалога клавишей Escape і кнопкой в заголовке вікна. Може бути полезным для діалогів відображення длительных операций, які управляются не користувачем, а системой. Наприклад діалог оплаты по карте через еквайринговый терминал.
ShowWaitCursor Boolean Якщо властивість установлено в True, то при виконанні запитів к серверу окно діалога буде затеняться і буде показываться индикатор виконання запиту.
