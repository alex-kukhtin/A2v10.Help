Элемент InlineDialog

Все контейнеры

Представляет собой диалог, который работает с текущей моделью и не требует дополнительной загрузки с сервера.

Каждый такой диалог должен иметь уникальный (в пределах текущей страницы) текстовый идентификатор.

Диалог работает в пределах текущей страницы. Фактически он представляет собой простой блок, который отображается по команде в виде диалога. Такие возможности как, например, отмену действия нужно реализовывать самостоятельно.

Открыть или закрыть диалог можно командами контроллера $inlineOpen, $inlineClose передав им идентификтор диалога.

Наследует: Container : UIElement : UIElementBase

Свойства

Имя Тип Описание
Id String Уникальный (в пределах страницы) идентификатор диалога.
Title String Заголовок окна диалога.
Width Length Ширина окна диалога. По умолчанию определяется содержимым.
Buttons UIElementCollection Содержимое панели кнопок. Обычно набор управляющих кнопок.
NoClose Boolean Запретить закрытие диалога клавишей Escape и кнопкой в заголовке окна. Может быть полезным для диалогов отображения длительных операций, которые управляются не пользователем, а системой. Например диалог оплаты по карте через еквайринговый терминал.
ShowWaitCursor Boolean Если свойство установлено в True, то при выполнении запросов к серверу окно диалога будет затеняться и будет показываться индикатор выполнения запроса.
