Елемент InlineDialog

Всі контейнери

Являє собою діалог, який працює з поточної моделлю і не потребує додаткового завантаження з сервера.

Кожен такий діалог повинен мати унікальний (в межах поточної сторінки) текстовий ідентифікатор.

Діалог працює в межах поточної сторінки. Фактично він являє собою простий блок, який відображається по команді у вигляді діалога. Такі можливості як, наприклад, скасування дії, потрібно реалізовувати самостійно.

Відкрити або закрити діалог можна командами контролера $inlineOpen, $inlineClose передавшит їм ідентифікатор діалогу.

Наслідує: Container : UIElement : UIElementBase

Властивості

Ім'я Тип Опис
Id String Унікальний (в межах сторінки) ідентифікатор діалогу.
Title String Заголовок вікна діалогу.
Width Length Ширина вікна діалогу. За замовчанням визначається вмістом.
Buttons UIElementCollection Вміст панелі кнопок. Зазвичай набір кнопок керування.
NoClose Boolean Заборонити закриття діалога клавішею Escape і кнопкою в заголовку вікна. Може бути корисним для діалогів відображення довготривалих операцій, які управляються не користувачем, а системою. Наприклад, діалог оплати по карті через еквайринговий термінал.
ShowWaitCursor Boolean Якщо властивість встановлено в True, то при виконанні запитів до серверу вікно діалогу буде затінятися і буде відображатися індикатор виконання запиту.
