Элемент FieldSet

Все контейнеры

Представляет собой контейнер (рамку) для элементов.

Наследует: Container : UIElement : UIElementBase

Свойства

Имя Тип Описание
Orientation Orientation Ориентация элементов в панели.
Title String Заголовок панели.
Hint Popover Всплывающая подсказка для рамки. Выводится в виде после текста заголовка.
Disabled Boolean Все элементы управления внутри панели будут недоступны для ввода. Для управления этим полем обычно используется привязка Bind.
