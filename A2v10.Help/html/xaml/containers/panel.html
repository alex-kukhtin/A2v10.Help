Элемент Panel

Все контейнеры

Представляет собой панель.

Наследует: Container : UIElement : UIElementBase

Свойства

Имя Тип Описание
Style PaneStyle Стиль отображения панели. Возможные значения:
  • Default - Обычная панель.
  • Danger, Error, Red - ошибка (красный).
  • Warning, Yellow - предупреждение (желтая).
  • Success, Green - успех (зеленая).
  • Info, Cyan - информация (голубая).
  • Cut - Сворачиваемая панель с заголовком в виде ссылки.
Compact Boolean Компактный стиль отображения панели.
Collapsible Boolean Можно ли сворачивать панель.
Collapsed Boolean? Свернута ли панель в исходном состоянии.
Header Object Заголовок. Может быть строкой, привязкой (Bind) или элементом UIElementBase.
Icon Icon Иконка. Отображается в левой части заголовка.
Hint Popover Всплывающая подсказка для панели. Выводится в виде после текста заголовка.
Height Length Высота панели.
DropShadow ShadowStyle Вид тени.
Background BackgroundStyle Цвет фона.
TestId String Идентификатор элемента для инструментов тестирования.
