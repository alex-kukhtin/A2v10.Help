Елемент Panel

Всі контейнери

Являє собою панель.

Наслідує: Container : UIElement : UIElementBase

Властивості

Ім'я Тип Опис
Style PaneStyle Стиль відображення панелі. Можливі значення:
  • Default - Звичайна панель.
  • Danger, Error, Red - помилка (червона).
  • Warning, Yellow - попередження (жовта).
  • Success, Green - успіх (зелена).
  • Info, Cyan - інформація (блакитна).
  • Cut - Панель, що згортається, з заголовком у вигляді посилання.
Compact Boolean Компактний стиль відображення панелі.
Collapsible Boolean Чи можна згортати панель.
Collapsed Boolean? Чи згорнута панель у вихідному стані.
Header Object Заголовок. Може бути рядком, прив'язкою (Bind) або елементом UIElementBase.
Icon Icon Іконка. Відображається в левій частині заголовку.
Hint Popover Спливаюча підказка для панелі. Виводиться у вигляді після текста заголовку.
Height Length Висота панелі.
DropShadow ShadowStyle Вид тіні.
Background BackgroundStyle Колір фону.
TestId String Ідентифікатор елементу для інструментів тестування.
