Элемент Taskpad

Все контейнеры

Представляет собой контейнер - панель задач. Отображается в правой части страницы или диалога. Задается свойством Taskpad страницы (Page) или диалога (Dialog).

Наследует: Container : UIElement : UIElementBase

Свойство содержимого: Children

Свойства

Имя Тип Описание
Title String Заголовок панели задач.
Overflow Boolean Может ли появляться скролл-бар в панели задач.
Collapsible Boolean Можно ли сворачивать панель задач.
Collapsed Boolean? Свернута ли панель задач в исходном состоянии.
Width Length Ширина панели задач.
Background BackgroundStyle Цвет фона.
