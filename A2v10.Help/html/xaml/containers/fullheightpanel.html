Элемент FullHeightPanel

Все контейнеры

Представляет панель, высота которой всегда равна 100%.

Наследует: Container : UIElement : UIElementBase

Свойства

Имя Тип Описание
MinWidth Length Минимальная ширина панели
Свойства базового элемента

Присоединенные свойства

Имя Тип Описание
FullHeightPanel.Fill Boolean Определяет дочерний элемент, высота которого будет изменяться при изменении высоты родительского контейнера.
Только один дочерний элемент может иметь свойство Fill = True.
FullHeightPanel.Skip Boolean Пропускать этот элемент при создании контейнера. Это свойство используется в том случае, если нужно, чтобы в одном блоке контейнера выводился один из нескольких элементов (например с помощью привязки If).
Установите Skip = True для всех элементов в блоке, кроме одного (любого).

Обратите внимание! Для реализации панели используется сетка (css grid). Из-за этого, при скрытии элементов с помощью привязки If, выбор элемента для заполнения будет неправильным.
Это происходит потому, что адресация панелей осуществляется по порядковому номеру.
Если нужно использовать привязку If, то поместите скрываемый элемент в дополнительный блок Block либо используйте присоединенное свойство FullHeightPanel.Skip.
