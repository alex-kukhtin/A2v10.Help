Елемент FullHeightPanel

Всі контейнери

Являє собою панель, висота якої завжди дорівнює 100%.

Наслідує: Container : UIElement : UIElementBase

Властивості

Ім'я Тип Опис
MinWidth Length Мінімальна ширина панелі
Властивості базового елементу

Приєднані властивості

Ім'я Тип Опис
FullHeightPanel.Fill Boolean Визначає дочірній елемент, висота якого буде змінюватися при зміні висоти батьківського контейнера.
Тільки один дочірній елемент може мати властивість Fill = True.
FullHeightPanel.Skip Boolean Пропускати цей елемент при створенні контейнера. Ця властивість використовується в тому випадку, коли потрібно, щоб в одному блоці контейнера виводився один з кількох елементів (наприклад за допомогою прив`язки If).
Встановіть Skip = True для всех елементів в блоці, крім одного (будь-якого).

Зверніть увагу! Для реалізації панелі використовується сітка (css grid). Из-за этого, при скрытии елементів за допомогою прив`язки If, вибір елементу для заполнения буде неправильным.
Це відбувається потому, что адресация панелей осуществляется по порядковому номеру.
Якщо потрібно використовувати прив'язку If, то поместите скрываемый елемент в додатковий блок Block або використовуйте присоединенное властивість FullHeightPanel.Skip.
