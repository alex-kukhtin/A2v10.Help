Елемент FullHeightPanel

Все контейнеры

Представляє панель, высота которой завжди равна 100%.

Наследует: Container : UIElement : UIElementBase

Властивості

Им'я Тип Опис
MinWidth Length Минимальная ширина панели
Властивості базового елементу

Присоединенные властивості

Им'я Тип Опис
FullHeightPanel.Fill Boolean Определяет дочірній елемент, высота якого буде изменяться при изменении высоты батьківського контейнера.
Тільки один дочірній елемент може мати властивість Fill = True.
FullHeightPanel.Skip Boolean Пропускать цей елемент при создании контейнера. Це властивість використовується в том случае, якщо потрібно, щоб в одном блоке контейнера выводился один з нескольких елементів (наприклад за допомогою прив`язки If).
Установите Skip = True для всех елементів в блоке, крім одного (любого).

Зверніть увагу! Для реализации панели використовується сетка (css grid). Из-за этого, при скрытии елементів за допомогою прив`язки If, выбор елементу для заполнения буде неправильным.
Це происходит потому, что адресация панелей осуществляется по порядковому номеру.
Якщо потрібно використовувати привязку If, то поместите скрываемый елемент в дополнительный блок Block або використовуйте присоединенное властивість FullHeightPanel.Skip.
