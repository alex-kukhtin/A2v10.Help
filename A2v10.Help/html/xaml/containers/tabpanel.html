Елемент TabPanel

Все контейнеры

Представляет собой панель с закладками. Содержимое представлено елементуми Tab.

Наследует: UIElement : UIElementBase

Властивість содержимого: Tabs

Властивості

Им'я Тип Опис
Tabs TabCollection Набор закладок (елементи Tab).
ItemsSource Array Всегда привязка - Bind. Джерело даних для закладок. Содержимое елементу буде повторяться у відповідності до содержимым ItemsSource.
Header Object Заголовок панели. Выводится в правой части панели с закладками. Може бути рядком, привязкой или об'єктом UIElementBase.
Border Boolean Выводить ли рамку вокруг панели.
FullPage Boolean Панель буде занимать всю страницу (является корневым елементом).
MinHeight Length Минимальная высота панели.
Overflow Boolean Может ли появляться скролл-бар в панели. Установите це властивість, якщо потрібно щоб спливаючі вікна (наприклад DatePicker) могли выходить за границы елементу. В цьому випадку потрібно внимательно следить за тем, щоб основное содержимое не вышло за границы. Скролл-бар отображаться не буде.
TabPosition TabPosition Положение панели закладок. Можливі значення:
  • Top - (за замовчанням) Сверху.
  • Bottom - Снизу.
