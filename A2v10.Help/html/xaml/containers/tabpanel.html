Элемент TabPanel

Все контейнеры

Представляет собой панель с закладками. Содержимое представлено элементами Tab.

Наследует: UIElement : UIElementBase

Свойство содержимого: Tabs

Свойства

Имя Тип Описание
Tabs TabCollection Набор закладок (элементы Tab).
ItemsSource Array Всегда привязка - Bind. Источник данных для закладок. Содержимое элемента будет повторяться в соответствии с содержимым ItemsSource.
Header Object Заголовок панели. Выводится в правой части панели с закладками. Может быть строкой, привязкой или объектом UIElementBase.
Border Boolean Выводить ли рамку вокруг панели.
FullPage Boolean Панель будет занимать всю страницу (является корневым элементом).
MinHeight Length Минимальная высота панели.
Overflow Boolean Может ли появляться скролл-бар в панели. Установите это свойство, если нужно чтобы всплывающие окна (например DatePicker) могли выходить за границы элемента. В этом случае нужно внимательно следить за тем, чтобы основное содержимое не вышло за границы. Скролл-бар отображаться не будет.
TabPosition TabPosition Положение панели закладок. Возможные значения:
  • Top - (по умолчанию) Сверху.
  • Bottom - Снизу.
