Елемент TabPanel

Всі контейнери

Являє собою панель з закладками. Вміст представлено елементами Tab.

Наслідує: UIElement : UIElementBase

Властивість вмісту: Tabs

Властивості

Ім'я Тип Опис
Tabs TabCollection Набір закладок (елементи Tab).
ItemsSource Array Завжди прив'язка - Bind. Джерело даних для закладок. Вміст елементу буде повторюватися у відповідності до вмісту ItemsSource.
Header Object Заголовок панелі. Виводиться в правій частині панелі з закладками. Може бути рядком, прив'язкою або об'єктом UIElementBase.
Border Boolean Чи виводити рамку навколо панелі.
FullPage Boolean Панель буде займати всю сторінку (є кореневим елементом).
MinHeight Length Мінімальна висота панелі.
Height Length Висота панелі.
Overflow Boolean Чи може з'являтися скролл-бар в панелі. Встановіть цю властивість, якщо потрібно, щоб спливаючі вікна (наприклад DatePicker) могли виходити за межі елементу. У цьому випадку потрібно уважно слідкувати за тим, щоб основний вміст не вийшов за межі. Скролл-бар відображатися не буде.
TabPosition TabPosition Положення панелі закладок. Можливі значення:
  • Top - Зверху (за замовчуванням).
  • Bottom - Внизу.
