Елемент Splitter

Всі контейнери

Являє собою контейнер, що містить две панелі зі змінними розмірами.

Зверніть увагу, що елемент завжди повинен містити рівно ДВА дочірніх елементи.

Наслідує: Container : UIElement : UIElementBase

Властивості

Ім'я Тип Опис
Orientation Orientation Орієнтація контейнера.
Height Length Висота елементу.
MinWidth Length Мінімальна ширина елементу.
Властивості базового елементу

Приєднані властивості

Ім'я Тип Опис
Splitter.Width GridLength Визначає ширину панелі.
Splitter.MinWidth Length Визначає мінімальну ширину панелі.
