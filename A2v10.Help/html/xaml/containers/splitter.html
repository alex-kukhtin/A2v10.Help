Элемент Splitter

Все контейнеры

Представляет собой контейнер, содержащий две панели с изменяемыми размерами.

Обратите внимание, что элемент всегда должен содержать ровно ДВА дочерних элемента.

Наследует: Container : UIElement : UIElementBase

Свойства

Имя Тип Описание
Orientation Orientation Ориентация контейнера.
Height Length Высота элемента.
MinWidth Length Минимальная ширина элемента.
Свойства базового элемента

Имя Тип Описание
