Елемент Toolbar

Всі контейнери

Являє собою панель інструментів.

В панель інструментів можна розміщувати будь-які елементи, але найчастіше там розміщаються прості кнопки Button, текстові поля TextBox тощо Для візуального розділення кнопок використовується елемент Separator.

Для того, щоб розділити блоки кнопок на дві частині - ліву і праву, використовується додана властивість Toolbar.Align.

Наслідує: Container : UIElement : UIElementBase

Властивості

Ім'я Тип Опис
Style ToolbarStyle Стиль відображення панелі інструментів. Можливі значення:
  • Default - Стандартне відображення (за замовчуванням).
  • Transparent - Прозора панель інструментів.
  • Light - Світлий стиль.
Конкретний вид відображення залежить від теми користувацького інтерфейса.
Border ToolbarBorderStyle Рамка. Можливі значення:
  • None - Нема рамки. (за замовчуванням)
  • Bottom - Рамка тільки знизу.
  • BottomShadow - Рамка знизу з тінню.
Конкретний вид відображення залежить від теми користувацького інтерфейса.
AlignItems AlignItems Режим вирівнювання елементів.
Властивості базового елементу

Приєднані властивості

Ім'я Тип Опис
Toolbar.Align ToolbarAlign Куди розміщувати елемент. Можливі значення:
  • Left - В лівий блок кнопок. (за замовчуванням)
  • Right - В правий блок кнопок.
