Элемент Toolbar

Все контейнеры

Представляет собой панель инструментов.

В панель инструментов можно помещать любые элементы, но чаще всего там размещаются простые кнопки Button, текстовые поля TextBox и т.д. Для визуального разделения кнопок используется элемент Separator.

Для того, чтобы разделить блоки кнопок на две части - левую и правую, используется присоединенное свойство Toolbar.Align.

Наследует: Container : UIElement : UIElementBase

Свойства

Имя Тип Описание
Style ToolbarStyle Стиль отображения панели инструментов. Возможные значения:
  • Default - (по умолчанию) Стандартное отображение.
  • Transparent - Прозрачная панель инструментов.
  • Light - Светлый стиль.
Конретный вид отображения зависит от темы пользовательского интерфейса.
Border ToolbarBorderStyle Рамка. Возможные значения:
  • None - (по умолчанию) Нет рамки.
  • Bottom - Рамка только снизу.
  • BottomShadow - Рамка снизу с тенью.
Конретный вид отображения зависит от темы пользовательского интерфейса.
AlignItems AlignItems Режим выравнивания элементов.
Свойства базового элемента

Присоединенные свойства

Имя Тип Описание
Toolbar.Align ToolbarAlign Куда помещать элемент. Возможные значения:
  • Left - (по умолчанию) В левый блок кнопок.
  • Right - В правый блок кнопок.
