Елемент Toolbar

Все контейнеры

Представляет собой панель инструментов.

В панель инструментов можна помещать будь-які елементи, но чаще всього там размещаются простые кнопки Button, текстовые поля TextBox і т.д. Для визуального разделения кнопок використовується елемент Separator.

Для того, щоб разделить блоки кнопок на две части - левую і правую, використовується присоединенное властивість Toolbar.Align.

Наследует: Container : UIElement : UIElementBase

Властивості

Им'я Тип Опис
Style ToolbarStyle Стиль отображения панели инструментов. Можливі значення:
  • Default - (за замовчанням) Стандартное отображение.
  • Transparent - Прозрачная панель инструментов.
  • Light - Светлый стиль.
Конретный вид отображения зависит від темы пользовательского інтерфейса.
Border ToolbarBorderStyle Рамка. Можливі значення:
  • None - (за замовчанням) Нет рамки.
  • Bottom - Рамка тільки снизу.
  • BottomShadow - Рамка снизу с тенью.
Конретный вид отображения зависит від темы пользовательского інтерфейса.
AlignItems AlignItems Режим выравнивания елементів.
Властивості базового елементу

Присоединенные властивості

Им'я Тип Опис
Toolbar.Align ToolbarAlign Куда помещать елемент. Можливі значення:
  • Left - (за замовчанням) В левый блок кнопок.
  • Right - В правый блок кнопок.
