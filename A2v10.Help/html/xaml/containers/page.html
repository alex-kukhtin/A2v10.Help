Элемент Page

Все контейнеры

Представляет собой страницу.

Страница имеет свой Url адрес и попадает в контекст переходов бразуера.

Наследует: RootContainer : Container : UIElement : UIElementBase

Свойство содержимого: Children

Свойства

Имя Тип Описание
Title String Заголовок окна страницы.
Taskpad UIElementBase Панель задач страницы. Чаще всего это элемент Taskpad.
Toolbar UIElementBase Панель инструментов страницы. Чаще всего это элемент Toolbar.
CollectionView CollectionView Представление коллекции, которая будет отображаться на странице. Обычно это серверная коллекция. Свойство необходимо чтобы вынести управление коллекцией "за скобки" содержимого страницы.
Pager Pager Пейджер для коллекции, которая будет отображаться на странице. Свойство необходимо чтобы вынести управление коллекцией "за скобки" содержимого страницы.
Background BackgroundStyle Цвет фона страницы.
