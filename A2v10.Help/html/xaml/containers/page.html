Елемент Page

Всі контейнери

Являє собою сторінку.

Сторінка має свою Url адресу і потрапляє в контекст переходів бразуера.

Наслідує: RootContainer : Container : UIElement : UIElementBase

Властивість вмісту: Children

Властивості

Ім'я Тип Опис
Title String Заголовок вікна сторінки.
Taskpad UIElementBase Панель задач сторінки. Частіше за все це елемент Taskpad.
Toolbar UIElementBase Панель інструментів сторінки. Частіше за все це елемент Toolbar.
CollectionView CollectionView Вид колекції, яка буде відображатися на сторінці. Зазвичай це серверна колекція. Властивість необхідна, щоб винести управління колекцією "за дужки" вмісту сторінки.
Pager Pager Пейджер для колекції, яка буде відображатися на сторінці. Властивість необхідна, щоб винести управління колекцією "за дужки" вмісту сторінки.
Background BackgroundStyle Колір фону сторінки.
