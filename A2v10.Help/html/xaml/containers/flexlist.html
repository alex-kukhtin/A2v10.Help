Елемент FlexList

Всі елементи керування

Являє собою контейнер з послідовним розташуванням елементів.

Наслідує: Container : UIElement : UIElementBase

Властивість вмісту: Children

Властивості

Ім'я Тип Опис
Orientation Orientation Орієнтація елементів в панелі.
JustifyItems JustifyItems Режим вирівнювання елементів по основній осі.
AlignItems AlignItems Режим вирівнювання елементів по перпендикулярній осі.
BorderStyle BorderStyle Стиль рамки навколо елементу.
MinWidth Length Мінімальна ширина елементу.
Gap GapSize Проміжки між елементами по основній і перпендикулярній осі.
Властивості базового елементу
