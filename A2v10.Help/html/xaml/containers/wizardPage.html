Элемент WizardPage

Все контейнеры

Представляет собой одну страницу мастера Wizard.

Наследует: Container : UIElement : UIElementBase

Свойство содержимого: Children

Свойства

Имя Тип Описание
Header Object Заголовок. Может быть строкой или привязкой (Bind).
Description Object Текст который будет выводиться в качестве подсказки в нижней части заголовка страницы. Может быть привязкой.
Height Length Высота страницы.
