Елемент WizardPage

Все контейнеры

Являє собою одну страницу мастера Wizard.

Наследует: Container : UIElement : UIElementBase

Властивість вмісту: Children

Властивості

Им'я Тип Опис
Header Object Заголовок. Може бути рядком або привязкой (Bind).
Description Object Текст который буде выводиться в якості подсказки в нижней части заголовка страницы. Може бути привязкой.
Height Length Высота страницы.
