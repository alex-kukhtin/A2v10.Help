Елемент Wizard

Все контейнеры

Являє собою мастер, состоящий з кількох отдельных страниц WiazrdPage

Наследует: Dialog : RootContainer : Container : UIElement : UIElementBase

Властивість вмісту: Children. Завжди буде коллекцией елементів WiazrdPage.

Властивості

Ім'я Тип Опис
FinishCommand BindCmd Команда, яка буде выполнена при нажатии кнопки Finish.
Властивості базового елементу

Особливості

Кнопки навигации (кнопки Назад, Далее, Готово і Отмена) отображаются і управляются автоматически. При нажатии кнопки Готово буде выполнена команда, заданная властивістю FinishCommand.
