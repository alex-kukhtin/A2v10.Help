Элемент Wizard

Все контейнеры

Представляет собой мастер, состоящий из нескольких отдельных страниц WiazrdPage

Наследует: Dialog : RootContainer : Container : UIElement : UIElementBase

Свойство содержимого: Children. Всегда будет коллекцией элементов WiazrdPage.

Свойства

Имя Тип Описание
FinishCommand BindCmd Команда, которая будет выполнена при нажатии кнопки Finish.
Свойства базового элемента

Особенности

Кнопки навигации (кнопки Назад, Далее, Готово и Отмена) отображаются и управляются автоматически. При нажатии кнопки Готово будет выполнена команда, заданная свойством FinishCommand.
