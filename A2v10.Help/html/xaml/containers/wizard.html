Елемент Wizard

Всі контейнери

Являє собою майстер, що складається з кількох окремих сторінок WiazrdPage

Наслідує: Dialog : RootContainer : Container : UIElement : UIElementBase

Властивість вмісту: Children. Завжди буде колекцією елементів WiazrdPage.

Властивості

Ім'я Тип Опис
FinishCommand BindCmd Команда, яка буде виконана при натисканні кнопки Finish.
Властивості базового елементу

Особливості

Кнопки навигації (кнопки Назад, Далі, Готово і Скасувати) відображаються і управляються автоматично. При натисканні кнопки Готово буде виконана команда, задана властивістю FinishCommand.
