Элемент Popup

Все контейнеры

Представляет собой всплывающее окно, которое должно загружаться с сервера. Этот элемент всегда будет корневым элементом разметки.

Наследует: RootContainer : Container : UIElement : UIElementBase

Свойство содержимого: Children

Свойства

Имя Тип Описание
Width Length Ширина окна. По умолчанию определяется содержимым.
MinWidth Length Минимальная ширина окна.
