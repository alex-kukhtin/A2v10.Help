Елемент Popup

Все контейнеры

Являє собою всплывающее окно, которое повинно завантажуватися с сервера. Этот елемент завжди буде корневым елементом розмітки.

Наследует: RootContainer : Container : UIElement : UIElementBase

Властивість вмісту: Children

Властивості

Ім'я Тип Опис
Width Length Ширина вікна. За замовчанням визначається содержимым.
MinWidth Length Минимальная ширина вікна.
