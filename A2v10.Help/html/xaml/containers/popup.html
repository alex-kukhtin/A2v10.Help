Елемент Popup

Всі контейнери

Являє собою спливаюче вікно, яке повинно завантажуватися з сервера. Цей елемент завжди буде кореневим елементом розмітки.

Наслідує: RootContainer : Container : UIElement : UIElementBase

Властивість вмісту: Children

Властивості

Ім'я Тип Опис
Width Length Ширина вікна. За замовчанням визначається вмістом.
MinWidth Length Мінімальна ширина вікна.
