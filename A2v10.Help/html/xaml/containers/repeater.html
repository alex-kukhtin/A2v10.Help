Елемент Repeater

Всі контейнери

Не має зовнішнього вигляду і обрамляючого тега. Використовується для виводу повторюваного вмісту. Іншими словами, в розмітку просто включаеться декілька елементів у відповідності до джерел даних.

Наслідує: UIElement : UIElementBase

Властивість вмісту: Content

Властивості

Ім'я Тип Опис
Content UIElementBase Елемент, який буде повторено для кожного значення з ItemsSource. Зверніть увагу, що прив`язки всередині вмісту будуть відносними. Корінь прив`язки буде являти собою один елемент з масиву ItemsSource.
ItemsSource Array Завжди прив'язка - Bind. Джерело даних для відображення елементів.
