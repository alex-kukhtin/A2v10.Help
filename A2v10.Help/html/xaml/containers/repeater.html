Елемент Repeater

Все контейнеры

Не має зовнішнього виду і обрамляющйого тега. Використовується для вывода повторяющйогося вмісту. Другими словами, в розмітку просто включается декілька елементів у відповідності до джерелоом даних.

Наследует: UIElement : UIElementBase

Властивість вмісту: Content

Властивості

Им'я Тип Опис
Content UIElementBase Елемент, который буде повторен для каждого значення з ItemsSource. Зверніть увагу, що прив`язки внутри вмісту будуть относительными. Корень прив`язки буде представлять собой один елемент з масива ItemsSource.
ItemsSource Array Завжди прив'язка - Bind. Джерело даних для отображения елементів.
