Елемент Repeater

Все контейнеры

Не має зовнішнього виду и обрамляющйого тега. Використовується для вывода повторяющйогося содержимого. Другими словами, в разметку просто включается несколько елементів у відповідності до джерелоом даних.

Наследует: UIElement : UIElementBase

Властивість содержимого: Content

Властивості

Им'я Тип Опис
Content UIElementBase Елемент, который буде повторен для каждого значення з ItemsSource. Зверніть увагу, что привязки внутри содержимого будуть относительными. Корень привязки буде представлять собой один елемент з масива ItemsSource.
ItemsSource Array Всегда привязка - Bind. Джерело даних для отображения елементів.
