Елемент TabBar

Все контейнеры

Являє собою панель для выбора елементу. Фактично це просто контейнер с кнопками TabButton, в якому текущая выделенная кнопка связывается зі заченнями. При нажатии кнопок, связанное значення изменяется і наоборот, при изменении значения, выбирается нужная кнопка.

Для того, щоб управлять, наприклад, видимостью панелей можна використовувати елемент Switch.

Наследует: UIElement : UIElementBase

Властивість вмісту: Buttons

Властивості

Им'я Тип Опис
Buttons TabButtonCollection Колекція кнопок (елементи TabButton).
ItemsSource Array Завжди прив'язка - Bind. Джерело даних для кнопок.
Value Object Значення, определяющее активную кнопку.
Style TabBarStyle Стиль отображения панели. Можливі значення:
  • Default - (за замовчанням) Стандартное отображение.
  • MainMenu - В виде основного меню системи.
  • Tab - В виде панели закладок.
  • Wizard - В виде мастера.
  • ButtonGroup - В виде группы кнопок.
DropShadow ShadowStyle Вид тени.
Description String | UIElementBase Текст (або елемент) который буде выводиться справа від кнопок. Для некоторых стилей не має смысла, поэтому отображаться не буде.
