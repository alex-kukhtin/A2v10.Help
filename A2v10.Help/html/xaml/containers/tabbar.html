Елемент TabBar

Всі контейнери

Являє собою панель для вибору елементу. Фактично це просто контейнер з кнопками TabButton, в якому поточна виділена кнопка пов'язується зі заченнями. При натисканні кнопок, пов'язане значення змінюється і навпаки, при зміні значення, обирається потрібна кнопка.

Для того, щоб управляти, наприклад, видимістю панелей, можна використовувати елемент Switch.

Наслідує: UIElement : UIElementBase

Властивість вмісту: Buttons

Властивості

Ім'я Тип Опис
Buttons TabButtonCollection Колекція кнопок (елементи TabButton).
ItemsSource Array Завжди прив'язка - Bind. Джерело даних для кнопок.
Value Object Значення, що визначає активну кнопку.
Style TabBarStyle Стиль відображення панелі. Можливі значення:
  • Default - Стандартне відображення (за замовчанням).
  • MainMenu - У вигляді основного меню системи.
  • Tab - У вигляді панелі закладок.
  • Wizard - У вигляді майстеру.
  • ButtonGroup - У вигляді групи кнопок.
DropShadow ShadowStyle Вид тіні.
Description String | UIElementBase Текст (або елемент) який буде виводитися праворуч від кнопок. Для деяких стилів не має сенсу, тому відображатися не буде.
