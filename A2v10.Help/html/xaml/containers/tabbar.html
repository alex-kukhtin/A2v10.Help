Элемент TabBar

Все контейнеры

Представляет собой панель для выбора элемента. Фактически это просто контейнер с кнопками TabButton, в котором текущая выделенная кнопка связывается со значением. При нажатии кнопок, связанное значение изменяется и наоборот, при изменении значения, выбирается нужная кнопка.

Для того, чтобы управлять, например, видимостью панелей можно использовать элемент Switch.

Наследует: UIElement : UIElementBase

Свойство содержимого: Buttons

Свойства

Имя Тип Описание
Buttons TabButtonCollection Коллекция кнопок (элементы TabButton).
ItemsSource Array Всегда привязка - Bind. Источник данных для кнопок.
Value Object Значение, определяющее активную кнопку.
Style TabBarStyle Стиль отображения панели. Возможные значения:
  • Default - (по умолчанию) Стандартное отображение.
  • MainMenu - В виде основного меню системы.
  • Tab - В виде панели закладок.
  • Wizard - В виде мастера.
  • ButtonGroup - В виде группы кнопок.
DropShadow ShadowStyle Вид тени.
Description String | UIElementBase Текст (или элемент) который будет выводиться справа от кнопок. Для некоторых стилей не имеет смысла, поэтому отображаться не будет.
