Елемент Card

Все контейнеры

Являє собою карточку.

Наследует: Container : UIElement : UIElementBase

Властивості

Им'я Тип Опис
Body Object Вміст карточки. Зазвичай являє собою елемент CardBody.
Header Object Заголовок. Може бути рядком, привязкой (Bind) або елементом UIElementBase.
Footer Object Футер. Може бути рядком, привязкой (Bind) або елементом UIElementBase.
Width Length Ширина карточки.
Height Length Высота карточки.
Compact Boolean Компактный стиль отображения карточки.
MaxHeight Length Максимальная высота карточки. Якщо размер больше, появляется вертикальный скроллер.
Align TextAlign Горизонтальное выравнивание вмісту.
Hint Popover Всплывающая подсказка для елементу. Выводится у вигляді після текста заголовка.
DropShadow ShadowStyle Вид тени.
Background BackgroundStyle Цвет фона.
