Элемент Card

Все контейнеры

Представляет собой карточку.

Наследует: Container : UIElement : UIElementBase

Свойства

Имя Тип Описание
Body Object Содержимое карточки. Обычно представляет собой элемент CardBody.
Header Object Заголовок. Может быть строкой, привязкой (Bind) или элементом UIElementBase.
Footer Object Футер. Может быть строкой, привязкой (Bind) или элементом UIElementBase.
Width Length Ширина карточки.
Height Length Высота карточки.
Compact Boolean Компактный стиль отображения карточки.
MaxHeight Length Максимальная высота карточки. Если размер больше, появляется вертикальный скроллер.
Align TextAlign Горизонтальное выравнивание содержимого.
Hint Popover Всплывающая подсказка для элемента. Выводится в виде после текста заголовка.
DropShadow ShadowStyle Вид тени.
Background BackgroundStyle Цвет фона.
