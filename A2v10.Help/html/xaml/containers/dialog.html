Елемент Dialog

Все контейнеры

Являє собою контейнер, які містить інші елементи керування або інші контейнеры. Завжди використовується як кореневий елемент розмітки.

Наследует: RootContainer : Container : UIElement : UIElementBase

Властивість вмісту: Children

Властивості

Им'я Тип Опис
Title String Заголовок вікна діалога.
HelpUrl String Путь к странице справки. Якщо указан, то в заголовке вікна діалога появится гиперссылка на страницу справки.
TitleInfo UIElementBase Дополнительный елемент в заголовке. Выводится після текста заголовка.
Buttons UIElementCollection Вміст панели кнопок. Зазвичай набір управляющих кнопок.
Taskpad UIElementBase Панель задач діалога. Частіше за всі це елемент Taskpad.
Width Length Ширина вікна діалога. За замовчанням визначається содержимым.
Height Length Высота вікна діалога. За замовчанням визначається содержимым.
MinWidth Length Минимальная ширина вікна.
Size DialogSize Размер вікна діалога. Можливі значення:
  • Default - за замовчанням.
  • Small - маленький.
  • Medium - нормальный.
  • Large - большой.
  • Max - Развернутый на весь экран. Эквивалентно Maximize="True"
Конкретные значення розміру зависят від темы пользовательского інтерфейса. Якщо указано властивість Width, то це властивість ігнорується.
Maximize Boolean Максимально возможный размер діалога. Будет меняться в залежності від размеров вікна браузера.
ButtonOnTop Boolean Размешать панель кнопок в верхней части вікна діалога.
ShowWaitCursor Boolean Якщо властивість установлено в True, то при виконанні запитов к серверу окно діалога буде затеняться і буде показываться индикатор виконання запиту.
TestId String Ідентифікатор вікна. Використовується при тестировании.
AlwaysOk Boolean Якщо властивість установлено в True, то при закрытии діалога любым способом буде считаться что возвращаемый результат равен true.
CanCloseDelegate String Им'я делегата, который буде викликатися при спробі закриття вікна (любым способом).
Background BackgroundStyle Цвет фона діалога.
Overflow Boolean Может ли появляться скролл-бар в діалоге. Установите це властивість, якщо потрібно щоб спливаючі вікна (наприклад DatePicker) могли выходить за границы елементу. В цьому випадку потрібно внимательно следить за тем, щоб основное вміст не вышло за границы. Скролл-бар отображаться не буде.
Властивості базового елементу

Формат делегата CanCloseDelegate

canClose(this:IRoot): any | Promise

Аргументи:

  • this - корень моделі даних.

Возвращаемое значення:

  • Promise - Якщо Promise вернет truthy значення, то діалог закриєся с этим результатом.
  • any - Якщо значення truthy, то діалог закриєся с этим результатом.

Примечание: truthy - любое значення, которое вважається истинным.
