Элемент Dialog

Все контейнеры

Представляет собой контейнер, которые содержит другие элементы управления или другие контейнеры. Всегда используется как корневой элемент разметки.

Наследует: RootContainer : Container : UIElement : UIElementBase

Свойство содержимого: Children

Свойства

Имя Тип Описание
Title String Заголовок окна диалога.
HelpUrl String Путь к странице справки. Если указан, то в заголовке окна диалога появится гиперссылка на страницу справки.
TitleInfo UIElementBase Дополнительный элемент в заголовке. Выводится после текста заголовка.
Buttons UIElementCollection Содержимое панели кнопок. Обычно набор управляющих кнопок.
Taskpad UIElementBase Панель задач диалога. Чаще всего это элемент Taskpad.
Width Length Ширина окна диалога. По умолчанию определяется содержимым.
Height Length Высота окна диалога. По умолчанию определяется содержимым.
MinWidth Length Минимальная ширина окна.
Size DialogSize Размер окна диалога. Возможные значения:
  • Default - по умолчанию.
  • Small - маленький.
  • Medium - нормальный.
  • Large - большой.
  • Max - Развернутый на весь экран. Эквивалентно Maximize="True"
Конкретные значения размера зависят от темы пользовательского интерфейса. Если указано свойство Width, то это свойство игнорируется.
Maximize Boolean Максимально возможный размер диалога. Будет меняться в зависимости от размеров окна браузера.
ButtonOnTop Boolean Размешать панель кнопок в верхней части окна диалога.
ShowWaitCursor Boolean Если свойство установлено в True, то при выполнении запросов к серверу окно диалога будет затеняться и будет показываться индикатор выполнения запроса.
TestId String Идентификатор окна. Используется при тестировании.
AlwaysOk Boolean Если свойство установлено в True, то при закрытии диалога любым способом будет считаться что возвращаемый результат равен true.
CanCloseDelegate String Имя делегата, который будет вызываться при попытке закрытия окна (любым способом).
Background BackgroundStyle Цвет фона диалога.
Overflow Boolean Может ли появляться скролл-бар в диалоге. Установите это свойство, если нужно чтобы всплывающие окна (например DatePicker) могли выходить за границы элемента. В этом случае нужно внимательно следить за тем, чтобы основное содержимое не вышло за границы. Скролл-бар отображаться не будет.
Свойства базового элемента

Формат делегата CanCloseDelegate

canClose(this:IRoot): any | Promise

Аргументы:

  • this - корень модели данных.

Возвращаемое значение:

  • Promise - Если Promise вернет truthy значение, то диалог закроется с этим результатом.
  • any - Если значение truthy, то диалог закроется с этим результатом.

Примечание: truthy - любое значение, которое считается истинным.
