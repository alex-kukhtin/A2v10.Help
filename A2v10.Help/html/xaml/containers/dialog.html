Елемент Dialog

Все контейнеры

Представляет собой контейнер, які містить другие елементи керування або другие контейнеры. Всегда використовується как корневой елемент розмітки.

Наследует: RootContainer : Container : UIElement : UIElementBase

Властивість содержимого: Children

Властивості

Им'я Тип Опис
Title String Заголовок окна діалога.
HelpUrl String Путь к странице справки. Якщо указан, то в заголовке окна діалога появится гиперссылка на страницу справки.
TitleInfo UIElementBase Дополнительный елемент в заголовке. Выводится после текста заголовка.
Buttons UIElementCollection Содержимое панели кнопок. Зазвичай набор управляющих кнопок.
Taskpad UIElementBase Панель задач діалога. Частіше за все це елемент Taskpad.
Width Length Ширина окна діалога. За замовчанням визначається содержимым.
Height Length Высота окна діалога. За замовчанням визначається содержимым.
MinWidth Length Минимальная ширина окна.
Size DialogSize Размер окна діалога. Можливі значения:
  • Default - за замовчанням.
  • Small - маленький.
  • Medium - нормальный.
  • Large - большой.
  • Max - Развернутый на весь экран. Эквивалентно Maximize="True"
Конкретные значення розміру зависят від темы пользовательского интерфейса. Якщо указано властивість Width, то це властивість игнорируется.
Maximize Boolean Максимально возможный размер діалога. Будет меняться в залежності від размеров окна браузера.
ButtonOnTop Boolean Размешать панель кнопок в верхней части окна діалога.
ShowWaitCursor Boolean Якщо властивість установлено в True, то при выполнении запитов к серверу окно діалога буде затеняться и буде показываться индикатор виконання запиту.
TestId String Идентификатор окна. Використовується при тестировании.
AlwaysOk Boolean Якщо властивість установлено в True, то при закрытии діалога любым способом буде считаться что возвращаемый результат равен true.
CanCloseDelegate String Им'я делегата, который буде викликатися при попытке закрытия окна (любым способом).
Background BackgroundStyle Цвет фона діалога.
Overflow Boolean Может ли появляться скролл-бар в діалоге. Установите це властивість, якщо потрібно щоб спливаючі вікна (наприклад DatePicker) могли выходить за границы елементу. В цьому випадку потрібно внимательно следить за тем, щоб основное содержимое не вышло за границы. Скролл-бар отображаться не буде.
Властивості базового елементу

Формат делегата CanCloseDelegate

canClose(this:IRoot): any | Promise

Аргументы:

  • this - корень моделі даних.

Возвращаемое значення:

  • Promise - Якщо Promise вернет truthy значення, то діалог закроется с этим результатом.
  • any - Якщо значення truthy, то діалог закроется с этим результатом.

Примечание: truthy - любое значення, которое вважається истинным.
