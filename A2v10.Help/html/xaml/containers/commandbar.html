Элемент CommandBar

Все контейнеры

Представляет собой панель команд. Обычно используется внутри элементов DataGrid и ListItem.

Особенность этого элемента в том, что его видимость будет определяться состояние родительского контейнера. Например, если разместить его в колонке таблицы и установить свойство Visibility = "Active", то панель команд будет отображаться только для активной строки таблицы.

Наследует: Container : UIElement : UIElementBase

Свойства

Имя Тип Описание
Visibility CommandBarVisibility Видимость панели. Возможные значения:
  • Default - (по умолчанию) Видима всегда.
  • Active - Видима только для активного элемента.
  • Hover - Видима как для активного элемента, так и для элемента, на котором находится курсор мыши.
Float FloatMode Сделать панель "плавающей".
  • None - (по умолчанию) нет.
  • Left - разместить слева.
  • Right - разместить справа.
