Елемент CommandBar

Все контейнеры

Являє собою панель команд. Зазвичай використовується внутри елементів DataGrid и ListItem.

Особенность цього елементу в том, що його видимость буде визначатися состояние батьківського контейнера. Наприклад, якщо разместить його в колонке таблиці і встановити властивість Visibility = "Active", то панель команд буде отображаться тільки для активной строки таблиці.

Наследует: Container : UIElement : UIElementBase

Властивості

Ім'я Тип Опис
Visibility CommandBarVisibility Видимость панели. Можливі значення:
  • Default - (за замовчанням) Видима завжди.
  • Active - Видима тільки для активного елементу.
  • Hover - Видима як для активного елементу, так і для елементу, на котором знаходиться курсор мыши.
Float FloatMode Сделать панель "плавающей".
  • None - (за замовчанням) нет.
  • Left - разместить слева.
  • Right - разместить справа.
