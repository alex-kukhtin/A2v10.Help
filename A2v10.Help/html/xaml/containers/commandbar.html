Елемент CommandBar

Всі контейнери

Являє собою панель команд. Зазвичай використовується всередині елементів DataGrid та ListItem.

Особливість цього елементу в тому, що його видимість буде визначатися станом батьківського контейнеру. Наприклад, якщо розмістити його в стовпчику таблиці і встановити властивість Visibility = "Active", то панель команд буде відображатися тільки для активного рядку таблиці.

Наслідує: Container : UIElement : UIElementBase

Властивості

Ім'я Тип Опис
Visibility CommandBarVisibility Видимість панелі. Можливі значення:
  • Default - (за замовчуванням) Видима завжди.
  • Active - Видима тільки для активного елементу.
  • Hover - Видима як для активного елементу, так і для елементу, на якому знаходиться курсор миші.
Float FloatMode Зробити панель "плаваючою".
  • None - ні (за замовчуванням).
  • Left - розмістити ліворуч.
  • Right - розмістити праворуч.
