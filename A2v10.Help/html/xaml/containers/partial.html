Елемент Partial

Все контейнеры

Представляет собой часть страницы. Використовується у випадку, якщо потрібно отобразить фрейм с собственной моделью і шаблоном як часть другой страницы.

Наследует: RootContainer : Container : UIElement : UIElementBase

Властивість содержимого: Children

Им'я Тип Опис
Background BackgroundStyle Цвет фона страницы.
Властивості базового елементу

Приклад

Например есть сторінка з двух колонок. В левой колонке - список сущностей, а в правой властивості выбранной сущности.

В левой колонке выводим список сущностей, в правой - елемент Include. Властивість Source цього елементу связываем со властивістю (возможна вычисляемым) елементу списка, которое должно представлять собой url к правой части страницы.

По сформированным url размещаем виду с елементуми Partial в качестве кореня с собственной моделью і собственным шаблоном.

Зверніть увагу, что сами моделі і их виду в правой части можуть быть совершенно произвольными і никак не связанными друг с другом.
