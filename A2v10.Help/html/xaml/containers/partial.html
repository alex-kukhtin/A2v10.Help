Элемент Partial

Все контейнеры

Представляет собой часть страницы. Используется в случае, если нужно отобразить фрейм с собственной моделью и шаблоном как часть другой страницы.

Наследует: RootContainer : Container : UIElement : UIElementBase

Свойство содержимого: Children

Свойства

Имя Тип Описание
Background BackgroundStyle Цвет фона страницы.
Свойства базового элемента

Пример

Например есть страница из двух колонок. В левой колонке - список сущностей, а в правой свойства выбранной сущности.

В левой колонке выводим список сущностей, в правой - элемент Include. Свойство Source этого элемента связываем со свойством (возможно вычисляемым) элемента списка, которое должно представлять собой url к правой части страницы.

По сформированным url размещаем представления с элементами Partial в качестве корня с собственной моделью и собственным шаблоном.

Обратите внимание, что сами модели и их представления в правой части могут быть совершенно произвольными и никак не связанными друг с другом.
