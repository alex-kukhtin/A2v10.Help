Елемент Grid

Все контейнеры

Являє собою сетку з рядків і колонок.

Наследует: Container : UIElement : UIElementBase

Примечание. Контейнер Grid використовується в макетах пользовательского інтерфейса. Він не предназначен для відображення даних. Для відображення і прив`язки к данным використовуйте соответствующие елементи упрвления, такі как Table або DataGrid.

Властивості

Ім'я Тип Опис
Height Length Высота елементу керування.
Gap GapSize Промежутки между строками і колонками сетки.
DropShadow ShadowStyle Вид тени.
Background BackgroundStyle Цвет фона.
AutoFlow AutoFlowMode Режим автоматического размещения елементів.
AlignItems AlignItem Режим выравнивания елементів.
Rows RowDefinitions Определения рядків сетки.
Columns ColumnDefinitions Определения колонок сетки.
Властивості базового елементу

Присоединенные властивості

Ім'я Тип Опис
Grid.Row Int32 Определяет номер строки, в которую повинен бути помещен дочірній елемент.
Grid.Col Int32 Определяет номер колонки, в которую повинен бути помещен дочірній елемент.
Grid.RowSpan Int32 Определяет кількість рядків, які буде занимать дочірній елемент. За замовчанням - 1.
Grid.ColSpan Int32 Определяет кількість колонок, які буде занимать дочірній елемент. За замовчанням - 1.
Grid.VAlign AlignItem Определяет вертикальное выравнивание дочірнього елементу таблиці. За замовчанням - Top.
