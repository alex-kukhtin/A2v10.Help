Елемент Grid

Всі контейнери

Являє собою сітку з рядків і стовпчиків.

Наслідує: Container : UIElement : UIElementBase

Примітка. Контейнер Grid використовується в макетах користувацького інтерфейсу. Він не призначений для відображення даних. Для відображення і прив`язки до даних використовуйте відповідні елементи управління, такі як Table або DataGrid.

Властивості

Ім'я Тип Опис
Height Length Висота елементу керування.
Gap GapSize Проміжки між рядками і колонками сітки.
DropShadow ShadowStyle Вид тіні.
Background BackgroundStyle Колір фону.
AutoFlow AutoFlowMode Режим автоматичного розміщення елементів.
AlignItems AlignItem Режим вирівнювання елементів.
Rows RowDefinitions Визначення рядків сітки.
Columns ColumnDefinitions Визначення стовпчиків сітки.
Властивості базового елементу

Приєднані властивості

Ім'я Тип Опис
Grid.Row Int32 Визначає номер рядку, в який повинен бути поміщений дочірній елемент.
Grid.Col Int32 Визначає номер стовпчика, в який повинен бути поміщений дочірній елемент.
Grid.RowSpan Int32 Визначає кількість рядків, які буде займати дочірній елемент. За замовчанням - 1.
Grid.ColSpan Int32 Визначає кількість стовпчиків, які буде займати дочірній елемент. За замовчанням - 1.
Grid.VAlign AlignItem Визначає вертикальне вирівнювання дочірнього елементу таблиці. За замовчанням - Top.
