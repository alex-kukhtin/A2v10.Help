Элемент Grid

Все контейнеры

Представляет собой сетку из строк и колонок.

Наследует: Container : UIElement : UIElementBase

Примечание. Контейнер Grid используется в макетах пользовательского интерфейса. Он не предназначен для отображения данных. Для отображения и привязки к данным используйте соответствующие элементы упрвления, такие как Table или DataGrid.

Свойства

Имя Тип Описание
Height Length Высота элемента управления.
Gap GapSize Промежутки между строками и колонками сетки.
DropShadow ShadowStyle Вид тени.
Background BackgroundStyle Цвет фона.
AutoFlow AutoFlowMode Режим автоматического размещения элементов.
AlignItems AlignItem Режим выравнивания элементов.
Rows RowDefinitions Определения строк сетки.
Columns ColumnDefinitions Определения колонок сетки.
Свойства базового элемента

Присоединенные свойства

Имя Тип Описание
Grid.Row Int32 Определяет номер строки, в которую должен быть помещен дочерний элемент.
Grid.Col Int32 Определяет номер колонки, в которую должен быть помещен дочерний элемент.
Grid.RowSpan Int32 Определяет количество строк, которые будет занимать дочерний элемент. По умолчанию - 1.
Grid.ColSpan Int32 Определяет количество колонок, которые будет занимать дочерний элемент. По умолчанию - 1.
Grid.VAlign AlignItem Определяет вертикальное выравнивание дочернего элемента таблицы. По умолчанию - Top.
