Елемент Pager

Всі елементи керування

Являє собою елемент для постраничной навигации. Зазвичай використовується для всей сторінки Page і задається як властивість Pager.

Сам компонент не использует жодних колекцій. Для цього завжди використовується компонент роботи з коллекцією CollectionView.

Наслідує: UIElementBase

Властивості

Ім'я Тип Опис
Source Object Исходная колекція. Завжди привязка. Якщо використовується спільно с CollectionView, то привязка буде Source="{Bind Parent.Pager}"
Style PagerStyle Стиль відображення. Можливі значення:
  • Default - (за замовчанням) обычные кнопки.
  • Rounded - круглі кнопки.
EmptyText String Текст, який буде відображатися в правой частині елементу у випадку, якщо колекція пуста. Якщо не вказано, показывается текст "нет елементів" (с учетом поточної локалізації).
TemplateText String Текст, який буде відображатися в правой частині елементу. В тексте допускаются макроси:
#[Start] Номер першого елементу на сторінці.
#[End] Номер последнйого елементу на сторінці.
#[Count] Кількість елементів в колекції.
Якщо властивість не вказано, виводиться текст "елементи #[Start]-#[End] з #[Count]" (с учетом поточної локалізації).
