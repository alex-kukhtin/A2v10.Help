Елемент Pager

Всі елементи керування

Являє собою елемент для навігації по сторінках. Зазвичай використовується для всієї сторінки Page і задається як властивість Pager.

Сам компонент не використовує жодних колекцій. Для цього завжди використовується компонент роботи з коллекцією CollectionView.

Наслідує: UIElementBase

Властивості

Ім'я Тип Опис
Source Object Вихідна колекція. Завжди прив'язка. Якщо використовується спільно с CollectionView, то прив'язка буде Source="{Bind Parent.Pager}"
Style PagerStyle Стиль відображення. Можливі значення:
  • Default - (за замовчуванням) звичайні кнопки.
  • Rounded - круглі кнопки.
EmptyText String Текст, який буде відображатися в правій частині елементу у випадку, коли колекція прожня. Якщо не вказано, відображається текст "нема елементів" (з врахуванням поточної локалізації).
TemplateText String Текст, який буде відображатися в правій частині елементу. В тексті допускаються макроси:
#[Start] Номер першого елементу на сторінці.
#[End] Номер останнього елементу на сторінці.
#[Count] Кількість елементів в колекції.
Якщо властивість не вказано, виводиться текст "елементи #[Start]-#[End] з #[Count]" (з врахуванням поточної локалізації).
