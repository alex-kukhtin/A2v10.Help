Элемент Pager

Все элементы управления

Представляет собой элемент для постраничной навигации. Обычно используется для всей страницы Page и задается как свойство Pager.

Сам компонент не использует никаких коллекций. Для этого всегда используется компонент работы с коллекцией CollectionView.

Наследует: UIElementBase

Свойства

Имя Тип Описание
Source Object Исходная коллекция. Всегда привязка. Если используется совместно с CollectionView, то привязка будет Source="{Bind Parent.Pager}"
Style PagerStyle Стиль отображения. Возможные значения:
  • Default - (по умолчанию) обычные кнопки.
  • Rounded - круглые кнопки.
EmptyText String Текст, который будет отображаться в правой части элемента в случае, если коллекция пуста. Если не указано, показывается текст "нет элементов" (с учетом текущей локализации).
TemplateText String Текст, который будет отображаться в правой части элемента. В тексте допускаются макросы:
#[Start] Номер первого элемента на странице.
#[End] Номер последнего элемента на странице.
#[Count] Количество элементов в коллекции.
Если свойство не указано, выводится текст "элементы #[Start]-#[End] из #[Count]" (с учетом текущей локализации).
