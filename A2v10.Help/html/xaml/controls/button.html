Елемент Button

Все елементи керування

Являє собою командную кнопку.

Наследует: CommandControl : ContentControl : Control : UIElement : UIElementBase

Властивість вмісту: Content

Властивості

Им'я Тип Опис
Style ButtonStyle Стиль кнопки. Якщо властивість являє собою привязку (Bind), то значення прив`язки повинно бути рядком в нижнем регистре (наприклад 'warning', а не 'Warning');
Size ControlSize Размер кнопки.
Icon Icon Иконка на кнопке.
IconAlign IconAlign Положение иконки на кнопки. Можливі значення:
  • Default - (за замовчанням) слева.
  • Left - слева.
  • Top - сверху.
DropDown UIElementBase Выпадающий елемент, связанный с кнопкой. Зазвичай це елемент DropDownMenu.
Rounded Boolean Кнопка с закругленными уголками. Якщо на кнопке нет текста, а є тільки иконка, то кнопка буде круглой.
