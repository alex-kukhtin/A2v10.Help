Елемент Button

Всі елементи керування

Являє собою командну кнопку.

Наслідує: CommandControl : ContentControl : Control : UIElement : UIElementBase

Властивість вмісту: Content

Властивості

Ім'я Тип Опис
Style ButtonStyle Стиль кнопки. Якщо властивість являє собою прив'язку (Bind), то значення прив`язки повинно бути рядком в нижньому регістрі (наприклад, 'warning', а не 'Warning');
Size ControlSize Розмір кнопки.
Icon Icon Іконка на кнопці.
IconAlign IconAlign Розташування іконки на кнопці. Можливі значення:
  • Default - (за замовчуванням) ліворуч.
  • Left - ліворуч.
  • Top - зверху.
DropDown UIElementBase Спадний елемент, пов'язаний з кнопкою. Зазвичай це елемент DropDownMenu.
Rounded Boolean Кнопка з закругленими кутами. Якщо на кнопці нема тексту, а є тільки іконка, то кнопка буде круглою.
