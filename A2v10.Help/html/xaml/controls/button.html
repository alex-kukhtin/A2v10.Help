Элемент Button

Все элементы управления

Представляет собой командную кнопку.

Наследует: CommandControl : ContentControl : Control : UIElement : UIElementBase

Свойство содержимого: Content

Свойства

Имя Тип Описание
Style ButtonStyle Стиль кнопки. Если свойство представляет собой привязку (Bind), то значение привязки должно быть строкой в нижнем регистре (например 'warning', а не 'Warning');
Size ControlSize Размер кнопки.
Icon Icon Иконка на кнопке.
IconAlign IconAlign Положение иконки на кнопки. Возможные значения:
  • Default - (по умолчанию) слева.
  • Left - слева.
  • Top - сверху.
DropDown UIElementBase Выпадающий элемент, связанный с кнопкой. Обычно это элемент DropDownMenu.
Rounded Boolean Кнопка с закругленными уголками. Если на кнопке нет текста, а есть только иконка, то кнопка будет круглой.
