Элемент TabButton

Все элементы управления

Представляет одну кнопку в панели TabBar.

Наследует: UIElementBase

Свойство содержимого: Content

Свойства

Имя Тип Описание
Content String | UIElementBase Содержимое (текст) кнопки. Может быть строкой или другим элементом.
Description String | UIElementBase Описание кнопки. Может быть строкой или другим элементом.
ActiveValue String Активное значение. Если значение Value, родительской панели будет равно этому значению, то кнопка будет активной (выделенной).
Свойства базового элемента
