Элемент DataGrid

Все элементы управления

Представляет собой таблицу с данными. Работает только с источником данных. Отображает одну строку для каждого элемента массива - источника данных. Колонки таблицы определяются коллекцией вложенных объектов DataGridColumn.

Совместно с компонентом CollectionView, таблица поддерживает сортировку, фильтрацию, разбиение на страницы и группировку строк.

Наследует: Control : UIElement : UIElementBase

Свойства

Имя Тип Описание
ItemsSource Array Всегда привязка - Bind. Источник данных для отображения.
Compact Boolean Компактный стиль отображения таблицы.
Hover Boolean Выделение строки при наведении курсора мыши.
Striped Boolean Раскраска таблицы "в полоску" с выделением четных и нечетных строк.
FixedHeader Boolean Зафиксировать шапку таблицы при прокрутке.
Style DataGridStyle Стиль отображения таблицы. Возможные значения:
  • Default - (по умолчанию) обычный стиль.
  • Light - светлый стиль.
Background BackgroundStyle Цвет фона.
Mark Object Только привязка (Bind). Маркировка строки таблицы. Маркер представляет собой вертикальную линию в левой колонке и/или выделение цветом. Подробности ниже.
MarkerStyle RowMarkerStyle Стиль маркировки строки в таблице. Возможные значения:
  • None - (по умолчанию) нет маркировки.
  • Row - фоновый цвет.
  • Marker - маркер в виде линии в первой колонке.
  • Both - оба вида маркировки.
EmptyPanel UIElement Панель, которая отображается, если содержимое пустое. Обычно представляет собой объект EmptyPanel.
DoubleClick BindCmd Команда, которая будет выполнена при двойном щелчке на строке таблицы. Не забудьте передать в команду аргумент.
ContextMenu DropDownMenu Контекстное меню, которое будет показыватьcя по правой кнопке мыши.
RowDetails DataGridRowDetails Детализация для строки.
AutoSelect AutoSelectMode Автоматически выделять элемент после загрузки или обновления списка. Поддерживаются только значения FirstItem и LastItem.

Маркировка строк таблицы
