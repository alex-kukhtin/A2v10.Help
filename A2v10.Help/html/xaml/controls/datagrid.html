Елемент DataGrid

Всі елементи керування

Являє собою таблицю з даними. Працює тільки з джерелом даних. Отображает одну строку для кожного елементу масива - джерела даних. Колонки таблиці визначаються колекцією вкладених об'єктів DataGridColumn.

Совместно з компонентом CollectionView, таблица підтримує сортировку, фільтрацію, розбиття на сторінки і группировку строк.

Наслідує: Control : UIElement : UIElementBase

Властивості

Ім'я Тип Опис
ItemsSource Array Завжди прив'язка - Bind. Джерело даних для відображення.
Compact Boolean Компактный стиль відображення таблиці.
Hover Boolean Выделение рядки при наведении курсора мыши.
Striped Boolean Раскраска таблиці "в полоску" з выделением четных і нечетных строк.
FixedHeader Boolean Зафиксировать шапку таблиці при прокрутке.
Style DataGridStyle Стиль відображення таблиці. Можливі значення:
  • Default - (за замовчанням) звичайний стиль.
  • Light - светлый стиль.
Background BackgroundStyle Колір фона.
Mark Object Тільки прив'язка (Bind). Маркировка рядки таблиці. Маркер являє собою вертикальную линию в левой колонке и/або выделение цветом. Подробности ниже.
MarkerStyle RowMarkerStyle Стиль маркировки рядки в таблице. Можливі значення:
  • None - (за замовчанням) нет маркировки.
  • Row - фоновий цвет.
  • Marker - маркер у вигляді линии в першої колонке.
  • Both - оба вида маркировки.
EmptyPanel UIElement Панель, яка відображається, якщо вміст пустое. Зазвичай являє собою об'єкт EmptyPanel.
DoubleClick BindCmd Команда, яка буде виконана при двойном щелчке на строке таблиці. Не забудьте передать в команду аргумент.
ContextMenu DropDownMenu Контекстное меню, которое буде показыватьcя по правой кнопке мыши.
RowDetails DataGridRowDetails Детализация для рядки.
AutoSelect AutoSelectMode Автоматически выделять елемент після завантаження або оновлення списка. Підтримуються тільки значення FirstItem і LastItem.
