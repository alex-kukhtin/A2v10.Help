Елемент HtmlElement

Всі елементи керування

Являє собою елемент для включення в розмітку довільних html-елементів.

Наслідує: UIElementBase

Властивість вмісту: Children

Властивості

Ім'я Тип Опис
TagName String Тег елементу.
Children UIElementCollection Властивість вмісту. Колекція дочірніх елементів.
Attributes HtmlAttribute Колекція атрибутів.
Властивості базового елементу

Опис

Найчастіше використовюється для додавання в розмітку довільних html-елементів. Такий елемент зазвичай буде окремим елементом керування vue.js.
