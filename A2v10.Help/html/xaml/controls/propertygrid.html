Елемент PropertyGrid

Все елементи керування

Являє собою таблицю властивостей.

Наследует: UIElement : UIElementBase

Властивість вмісту: Children

Властивості

Ім'я Тип Опис
Children PropertyGridItems Властивість вмісту. Колекція дочірніх елементів PropertyGridItem.
ItemsSource Array Завжди прив'язка (Bind). Джерело даних для отримання дочірніх елементів.
Style PropertyGridStyle Стиль відображення таблиці. Можливі значення:
  • Default - Обычная таблиця.
  • Compact - Компактная таблиця (відповідає Compact = "True").
  • Small - Миниатюрная таблиця. Зазвичай використовується внутри других таблиц.
Compact Boolean Компактный стиль відображення таблиці.
Striped Boolean Раскраска таблиці "в полоску" с выделением четных і нечетных строк.
NoWrapName Boolean Запретить перенос имен властивостей.
GridLines GridLinesVisibility Отображение сетки. Можливі значення:
  • None - за замовчанням.
  • Horizontal - горизонтальные линии.
  • Vertical - вертикальные линии.
  • Both - сетка.
TestId String Ідентифікатор елементу для інструментів тестування.
