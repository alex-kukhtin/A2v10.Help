Елемент PropertyGrid

Всі елементи керування

Являє собою таблицю властивостей.

Наслідує: UIElement : UIElementBase

Властивість вмісту: Children

Властивості

Ім'я Тип Опис
Children PropertyGridItems Властивість вмісту. Колекція дочірніх елементів PropertyGridItem.
ItemsSource Array Завжди прив'язка (Bind). Джерело даних для отримання дочірніх елементів.
Style PropertyGridStyle Стиль відображення таблиці. Можливі значення:
  • Default - Звичайна таблиця.
  • Compact - Компактна таблиця (відповідає Compact = "True").
  • Small - Мініатюрна таблиця. Зазвичай використовується всередині інших таблиць.
Compact Boolean Компактный стиль відображення таблиці.
Striped Boolean Розфарбування таблиці "у смужку" з виділенням парних і непарних рядків.
NoWrapName Boolean Заборонити перенесення імен властивостей.
GridLines GridLinesVisibility Відображення сітки. Можливі значення:
  • None - за замовчанням.
  • Horizontal - горизонтальні лінії.
  • Vertical - вертикальні лінії.
  • Both - сітка.
TestId String Ідентифікатор елементу для інструментів тестування.
