Элемент PropertyGrid

Все элементы управления

Представляет собой таблицу свойств.

Наследует: UIElement : UIElementBase

Свойство содержимого: Children

Свойства

Имя Тип Описание
Children PropertyGridItems Свойство содержимого. Коллекция дочерних элементов PropertyGridItem.
ItemsSource Array Всегда привязка (Bind) Источник данных для получения дочерних элементов.
Style PropertyGridStyle Стиль отображения таблицы. Возможные значения:
  • Default - Обычная таблица.
  • Compact - Компактная таблица (соответствует Compact = "True".
  • Small - Миниатюрная таблица. Обычно используется внутри других таблиц.
Compact Boolean Компактный стиль отображения таблицы.
Striped Boolean Раскраска таблицы "в полоску" с выделением четных и нечетных строк.
NoWrapName Boolean Запретить перенос имен свойств.
GridLines GridLinesVisibility Отображение сетки. Возможные значения:
  • None - по умолчанию.
  • Horizontal - горизонтальные линии.
  • Vertical - вертикальные линии.
  • Both - сетка.
TestId String Идентификатор элемента для инструментов тестирования.
