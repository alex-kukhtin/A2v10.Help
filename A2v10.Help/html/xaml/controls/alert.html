Елемент Alert

Всі елементи керування

Являє собою панель повідомлення.

Наслідує: UIElement : UIElementBase

Властивість вмісту: Content

Властивості

Ім'я Тип Опис
Content Object Властивість вмісту. Вміст панелі. Може бути як звичайним текстом, так і елементом керування (UIElementBase).
Style AlertStyle Стиль відображення. Можливі значення (конкретний вигляд залежить від теми користувацького інтерфейса):
  • Default - Звичайна панель. (за замовчуванням)
  • Success - Успіх (зелена).
  • Warning - Попередження (помаранчева).
  • Info - Інформація (блакитна).
  • Danger - Помилка (червона).
  • Dark - Темна.
  • Light - Світла.
Icon Icon Іконка, яка буде відображатися перед вмістом.
DropShadow ShadowStyle Вид тіні.
