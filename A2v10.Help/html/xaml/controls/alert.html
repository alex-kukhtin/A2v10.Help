Элемент Alert

Все элементы управления

Представляет собой панель сообщения.

Наследует: UIElement : UIElementBase

Свойство содержимого: Content

Свойства

Имя Тип Описание
Content Object Свойство содержимого. Содержимое панели. Может быть как простым текстом, так и элементом управления (UIElementBase).
Style AlertStyle Стиль отображения. Возможные значения (конкретный вид зависит от темы пользовательского интерфейса):
  • Default - (по умолчанию) Обычная панель.
  • Success - Успех (зеленая).
  • Warning - Предупреждение (оранжевая).
  • Info - Информация (голубая).
  • Danger - Ошибка (красная).
  • Dark - Темная.
  • Light - Светлая.
Icon Icon Иконка, которая будет отображаться перед содержимым.
DropShadow ShadowStyle Вид тени.
