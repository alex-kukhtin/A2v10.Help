Елемент Alert

Все елементи керування

Являє собою панель повідомлення.

Наследует: UIElement : UIElementBase

Властивість вмісту: Content

Властивості

Ім'я Тип Опис
Content Object Властивість вмісту. Вміст панели. Може бути як простым текстом, так і елементом керування (UIElementBase).
Style AlertStyle Стиль відображення. Можливі значення (конкретный вид зависит від темы пользовательского інтерфейса):
  • Default - (за замовчанням) Обычная панель.
  • Success - Успех (зеленая).
  • Warning - Предупреждение (оранжевая).
  • Info - Информация (голубая).
  • Danger - Ошибка (красная).
  • Dark - Темная.
  • Light - Светлая.
Icon Icon Иконка, яка буде отображаться перед содержимым.
DropShadow ShadowStyle Вид тени.
