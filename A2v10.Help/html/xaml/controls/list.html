Элемент List

Все элементы управления

Представляет собой список элементов (c возможностью выделения). Содержимое обычно представляет собой список элементов ListItem.

Наследует: Control : UIElement : UIElementBase

Свойство содержимого: Content

Свойства

Имя Тип Описание
Content UIElementCollection Свойство содержимого. Содержимое списка
ItemsSource Array Всегда привязка - Bind. Источник данных для отображения в списке.
Style ListStyle Стиль отображения списка.
Background BackgroundStyle Стиль фона для списка.
Border Boolean Отображать ли рамку вокруг списка.
BorderStyle BorderStyle Стиль рамки вокруг элемента.
Striped Boolean Раскраска списка "в полоску" с выделением четных и нечентых элементов.
Flush Boolean Убирает линию после последнего элемента в списке (только для стиля Underlined).
AutoSelect AutoSelectMode Автоматически выделять элемент после загрузки или обновления списка.
Height Length Высота списка.
MaxHeight Length Максимальная высота списка. Если список больше, появляется вертикальный скроллер.
Select Boolean? Разрешено ли выделение элементов в списке.
Mark Object Только привязка (Bind). Маркировка элемента списка. Маркер представляет собой вертикальную линию слева от элемента. Подробности ниже.
MarkerStyle RowMarkerStyle Стиль маркировки элемента в списке. Возможные значения:
  • None - (по умолчанию) нет маркировки.
  • Row - фоновый цвет.
  • Marker - маркер в виде полоски слева.
  • Both - оба вида маркировки.
EmptyPanel UIElement Панель, которая отображается, если содержимое пустое. Обычно представляет собой объект EmptyPanel.
DoubleClick BindCmd Команда, которая будет выполнена при двойном щелчке на элементе списка. Не забудьте передать в команду аргумент.
Свойства базового элемента

Маркировка элементов списка

Выражение привязки для свойства Mark должно возвращать текстовую строку, которая представляет собой дополнительный css-класс, добавляемый к элементу списка. В теме по умолчанию поддерживаются следующие варианты маркировки элементов списка:

значение Mark"Marker""Row" "danger", "red", "error" #fc7c7c #fff5fa "warning", "yellow" #ffd700 #fffff0 "success", "green" #89e489 #f5fff5 "info", "cyan" #72eaea #f0ffff
