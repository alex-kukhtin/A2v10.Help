Елемент List

Всі елементи керування

Являє собою перелік елементів (c можливістю виділення). Вміст зазвичай являє собою перелік елементів ListItem.

Наслідує: Control : UIElement : UIElementBase

Властивість вмісту: Content

Властивості

Ім'я Тип Опис
Content UIElementCollection Властивість вмісту. Вміст переліку
ItemsSource Array Завжди прив'язка - Bind. Джерело даних для відображення в переліку.
Style ListStyle Стиль відображення переліку.
Background BackgroundStyle Стиль фону для переліку.
Border Boolean Чи відображати рамку навколо переліку.
BorderStyle BorderStyle Стиль рамки навколо елементу.
Striped Boolean Розфарбування переліку "у смужку" з виділенням парних і непарних елементів.
Flush Boolean Прибирає лінію після останнього елементу в переліку (тільки для стилю Underlined).
AutoSelect AutoSelectMode Автоматично виділяти елемент після завантаження або оновлення переліку.
Height Length Висота переліку.
MaxHeight Length Максимальна висота переліку. Якщо список вищий, з'являється вертикальний скроллер.
Select Boolean? Чи дозволено виділення елементів в переліку.
Mark Object Тільки прив'язка (Bind). Маркування елементу переліку. Маркер являє собою вертикальну лінію ліворуч від елементу. Підробиці нижче.
MarkerStyle RowMarkerStyle Стиль маркування елементу в списку. Можливі значення:
  • None - (за замовчуванням) нема маркування.
  • Row - фоновий колір.
  • Marker - маркер у вигляді смужки ліворуч.
  • Both - обидва види маркування.
EmptyPanel UIElement Панель, що відображається, коли вміст пустий. Зазвичай являє собою об'єкт EmptyPanel.
DoubleClick BindCmd Команда, яку буде виконано при подвійному кліку на елементі переліку. Не забудьте передати в команду аргумент.
