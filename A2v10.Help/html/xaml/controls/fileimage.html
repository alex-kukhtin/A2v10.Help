Елемент FileImage

Все елементи керування

Представляет собой изображение с можливістью привязки к данным.

Детальніше про работе с двоичными об'єктами...

Наследует: UIElementBase

Властивості

Им'я Тип Опис
Url String URL изображения. Должен ссылаться на раздел files файла описания моделі.
Value Bind Связывается с полем, которое визначає ідентифікатор картинки.
Width Length Ширина изображения.
Height Length Высота изображения.
Властивості базового елементу

Особливості

Частіше за всі задається тільки одно з властивостей Width або Height. В цьому випадкут картинка буде сохранять пропорции.
