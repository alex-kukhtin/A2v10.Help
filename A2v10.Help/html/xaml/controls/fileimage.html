Елемент FileImage

Все елементи керування

Являє собою зображення з можливістю прив`язки к данным.

Детальніше про роботі з бінарними об'єктами...

Наследует: UIElementBase

Властивості

Ім'я Тип Опис
Url String URL изображения. Должен посилатися на раздел files файла описания моделі.
Value Bind Связывается з полем, которое визначає ідентифікатор зображення.
Width Length Ширина изображения.
Height Length Высота изображения.
Властивості базового елементу

Особливості

Частіше за всі задається тільки одно з властивостей Width або Height. В цьому випадкут картинка буде зберігати пропорции.
