Елемент ComboBox

Все елементи керування

Представляет собой выпадающий список выбора.

Наследует: ValuedControl : Control : UIElement : UIElementBase

Властивість содержимого: Children

Властивості

Им'я Тип Опис
Children List<ComboBoxItems> Содержимое списка. Представляет собой список елементів ComboBoxItem. Може бути статическим або динамическим.
ItemsSource Array Всегда привязка - Bind. Джерело даних для отображения в списке.
DisplayProperty String Им'я властивості, которое буде отображаться при связывании елементів списка с об'єктами.
ShowValue Boolean Отображать в списке значення вмісце содержимого.
Align TextAlign Выравнивание текста.
Style ComboBoxStyle Стиль отображения. Можливі значення:
  • Default - (за замовчанням) звичайний список.
  • Hyperlink - отображать список у вигляді гиперссылки
Size ControlSize Размер елементу. Можливі значення:
  • Default - (за замовчанням) список звичайного розміру.
  • Large - увеличенный размер списка.
(остальные значення перечисления ControlSize не поддерживаются).
Властивості базового елементу

Опис

Содержимое выпадающйого списка може задаватися тремя различными способами:

  • Статическое содержимое. В цьому випадку, елемент повинен включать перечень елементів ComboBoxItem, для каждого з которых задаются властивості Content і Value. Властивість ItemsSource не зазначається. При выборе елементу властивості Value присваивается соответствующее значення Value з елементу. В списке отображается властивість Content (или Value, якщо установлено ShowValue = true).
  • Прив'язка к списку об'єктів. В цьому випадку привязка ItemsSource связывается с масивом об'єктів. Значенням буде выбранный об'єкт. В списке отображается властивість связанного об'єкта, заданное властивістю DisplayProperty.
  • Прив'язка к списку об'єктів і значению. В цьому випадку привязка ItemsSource також связывается с масивом об'єктів. Но для отображения і выбора значений використовується единственный вложенный елемент ComboBoxItem. Властивості Content і Value цього елементу связываются с необходимыми значениями елементу масива.
