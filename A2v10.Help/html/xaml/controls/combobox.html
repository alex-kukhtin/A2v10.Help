Елемент ComboBox

Всі елементи керування

Являє собою спадний список вибору.

Наслідує: ValuedControl : Control : UIElement : UIElementBase

Властивість вмісту: Children

Властивості

Ім'я Тип Опис
Children List<ComboBoxItems> Вміст списку. Являє собою список елементів ComboBoxItem. Може бути статичним або динамічним.
ItemsSource Array Завжди прив'язка - Bind. Джерело даних для відображення в списку.
DisplayProperty String Ім'я властивості, яке буде відображатися при зв'язуванні елементів списку з об'єктами.
ShowValue Boolean Відображати в списку значення замість вмісту.
Align TextAlign Вирівнювання тексту.
Style ComboBoxStyle Стиль відображення. Можливі значення:
  • Default - (за замовчуванням) звичайний список.
  • Hyperlink - список у вигляді гіперпосилання
Size ControlSize Розмір елементу. Можливі значення:
  • Default - (за замовчуванням) список звичайного розміру.
  • Large - збільшений розмір переліку.
(всі решта значень переліку ControlSize не підтримуються).
Властивості базового елементу

Опис

Вміст спадного списку може задаватися трьома різними способами:

  • Статическое вміст. У цьому випадку, елемент повинен містити перелік елементів ComboBoxItem, для кожного з яких задаються властивості Content і Value. Властивість ItemsSource не зазначається. При обранні елементу властивості Value присвоюється відповідне значення Value з елементу. В списку відображається властивість Content (або Value, якщо встановлено ShowValue = true).
  • Прив'язка до списку об'єктів. У цьому випадку прив'язка ItemsSource пов'язується з масивом об'єктів. Значенням буде обраний об'єкт. В списку відображається властивість зв'язаного об'єкта, задана властивістю DisplayProperty.
  • Прив'язка до списку об'єктів і значень. У цьому випадку прив'язка ItemsSource також пов'язується з масивом об'єктів. Але для відображення і вибору значень використовується єдиний вкладений елемент ComboBoxItem. Властивості Content і Value цього елементу зв'язуються з необхідними значеннями елементу масиву.
