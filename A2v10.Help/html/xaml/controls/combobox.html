Елемент ComboBox

Все елементи керування

Являє собою выпадающий список выбора.

Наследует: ValuedControl : Control : UIElement : UIElementBase

Властивість вмісту: Children

Властивості

Ім'я Тип Опис
Children List<ComboBoxItems> Вміст списка. Являє собою список елементів ComboBoxItem. Може бути статическим або динамическим.
ItemsSource Array Завжди прив'язка - Bind. Джерело даних для відображення в списке.
DisplayProperty String Ім'я властивості, которое буде отображаться при связывании елементів списка з об'єктами.
ShowValue Boolean Отображать в списке значення вмісце вмісту.
Align TextAlign Выравнивание текста.
Style ComboBoxStyle Стиль відображення. Можливі значення:
  • Default - (за замовчанням) звичайний список.
  • Hyperlink - отображать список у вигляді гиперпосилання
Size ControlSize Размер елементу. Можливі значення:
  • Default - (за замовчанням) список звичайного розміру.
  • Large - увеличенный размер списка.
(остальные значення перечисления ControlSize не підтримуються).
Властивості базового елементу

Опис

Вміст выпадающйого списка може задаватися тремя различными способами:

  • Статическое вміст. В цьому випадку, елемент повинен включать перечень елементів ComboBoxItem, для кожного з которых задаются властивості Content і Value. Властивість ItemsSource не зазначається. При выборе елементу властивості Value присваивается соответствующее значення Value з елементу. В списке отображается властивість Content (або Value, якщо установлено ShowValue = true).
  • Прив'язка к списку об'єктів. В цьому випадку прив'язка ItemsSource связывается з масивом об'єктів. Значенням буде выбранный об'єкт. В списке отображается властивість связанного об'єкта, заданное властивістю DisplayProperty.
  • Прив'язка к списку об'єктів і значению. В цьому випадку прив'язка ItemsSource також связывается с масивом об'єктів. Но для відображення і выбора значень використовується єдиний вложенный елемент ComboBoxItem. Властивості Content і Value цього елементу связываются з необходимыми значеннями елементу масива.
