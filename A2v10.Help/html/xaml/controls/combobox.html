Элемент ComboBox

Все элементы управления

Представляет собой выпадающий список выбора.

Наследует: ValuedControl : Control : UIElement : UIElementBase

Свойство содержимого: Children

Свойства

Имя Тип Описание
Children List<ComboBoxItems> Содержимое списка. Представляет собой список элементов ComboBoxItem. Может быть статическим или динамическим.
ItemsSource Array Всегда привязка - Bind. Источник данных для отображения в списке.
DisplayProperty String Имя свойства, которое будет отображаться при связывании элементов списка с объектами.
ShowValue Boolean Отображать в списке значение вместо содержимого.
Align TextAlign Выравнивание текста.
Style ComboBoxStyle Стиль отображения. Возможные значения:
  • Default - (по умолчанию) обычный список.
  • Hyperlink - отображать список в виде гиперссылки
Size ControlSize Размер элемента. Возможные значения:
  • Default - (по умолчанию) список обычного размера.
  • Large - увеличенный размер списка.
(остальные значения перечисления ControlSize не поддерживаются).
Свойства базового элемента

Описание

Содержимое выпадающего списка может задаваться тремя различными способами:

  • Статическое содержимое. В этом случае, элемент должен включать перечень элементов ComboBoxItem, для каждого из которых задаются свойства Content и Value. Свойство ItemsSource не указывается. При выборе элемента свойству Value присваивается соответствующее значение Value из элемента. В списке отображается свойство Content (или Value, если установлено ShowValue = true).
  • Привязка к списку объектов. В этом случае привязка ItemsSource связывается с массивом объектов. Значением будет выбранный объект. В списке отображается свойство связанного объекта, заданное свойством DisplayProperty.
  • Привязка к списку объектов и значению. В этом случае привязка ItemsSource также связывается с массивом объектов. Но для отображения и выбора значений используется единственный вложенный элемент ComboBoxItem. Свойства Content и Value этого элемента связываются с необходимыми значениями элемента массива.
