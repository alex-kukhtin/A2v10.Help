Елемент PropertyGridItem

Все елементи керування

Являє собою один елемент таблиці властивостей PropertyGrid.

Наследует: UIElementBase

Властивість вмісту: Content

Властивості

Ім'я Тип Опис
Content Object Властивість вмісту. Значення властивості. Може бути рядком або елементом UIElementBase.
Name String Найменування властивості.
Bold Boolean? Значення властивості буде выводиться полужирным шрифтом.
HideEmpty Boolean Скрывать строки властивостей с пустыми значениями.
