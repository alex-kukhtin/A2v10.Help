Элемент Label

Все элементы управления

Представляет собой текстовую метку элемента управления (такую же, которая выводится в элементе с указанным свойством Label).

Наследует: UIElement : UIElementBase

Свойство содержимого: Content

Свойства

Имя Тип Описание
Content String Свойство содержимого. Текст метки.
Align TextAlign Выравнивание текста
Required Boolean Обязательное значение для ввода. Обратите внимание, что это только визуальное представление обязательного ввода в поле ввода (конкретный вид зависит от темы пользовательского интерфейса). Для управления проверками используйте валидаторы шаблонов.
Size ControlSize Размер элемента. Возможные значения:
  • Default, Normal - (по умолчанию) элемент обычного размера.
  • Large - увеличенный размер элемента.
(остальные значения перечисления ControlSize не поддерживаются).
