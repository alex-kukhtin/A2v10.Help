Елемент Label

Всі елементи керування

Являє собою текстовую метку елементу керування (такую же, яка виводиться в елементе з вказаним властивістю Label).

Наслідує: UIElement : UIElementBase

Властивість вмісту: Content

Властивості

Ім'я Тип Опис
Content String Властивість вмісту. Текст метки.
Align TextAlign Вирівнювання текста
Required Boolean Обов'язковое значення для введення. Зверніть увагу, що це тільки визуальное вид обов'язкового введення в поле введення (конкретный вид залежить від теми користувацького інтерфейса). Для керування перевірками використовуйте валідатори шаблонов.
Size ControlSize Розмір елементу. Можливі значення:
  • Default, Normal - (за замовчанням) елемент звичайного розміру.
  • Large - увеличенный розмір елементу.
(остальные значення перечисления ControlSize не підтримуються).
