Елемент Label

Все елементи керування

Являє собою текстовую метку елементу керування (такую же, яка выводится в елементе с вказаним властивістю Label).

Наследует: UIElement : UIElementBase

Властивість вмісту: Content

Властивості

Им'я Тип Опис
Content String Властивість вмісту. Текст метки.
Align TextAlign Выравнивание текста
Required Boolean Обов'язковое значення для ввода. Зверніть увагу, що це тільки визуальное представление обов'язкового ввода в поле ввода (конкретный вид зависит від темы пользовательского інтерфейса). Для керування проверками используйте валидаторы шаблонов.
Size ControlSize Размер елементу. Можливі значення:
  • Default, Normal - (за замовчанням) елемент звичайного розміру.
  • Large - увеличенный размер елементу.
(остальные значення перечисления ControlSize не підтримуються).
