Елемент DataGridRowDetails

Все елементи керування

Представляет собой елемент детализации строки для таблиці DataGrid.

Властивість содержимого: Content

Властивості

Им'я Тип Опис
Content UIElementBase Содержание елементу.
Visible Boolean Видим ли елемент.
Activate RowDetailsActivate Когда активировать детализацию. Можливі значення:
  • ActiveRow - (за замовчанням) показывать для выделенной строки.
  • Cell - показывать специальную ячейку с можливістью раскрывать детализацию.
  • Always - показывать завжди.
