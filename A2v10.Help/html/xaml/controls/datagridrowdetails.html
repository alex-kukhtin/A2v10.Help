Элемент DataGridRowDetails

Все элементы управления

Представляет собой элемент детализации строки для таблицы DataGrid.

Свойство содержимого: Content

Свойства

Имя Тип Описание
Content UIElementBase Содержание элемента.
Visible Boolean Видим ли элемент.
Activate RowDetailsActivate Когда активировать детализацию. Возможные значения:
  • ActiveRow - (по умолчанию) показывать для выделенной строки.
  • Cell - показывать специальную ячейку с возможностью раскрывать детализацию.
  • Always - показывать всегда.
