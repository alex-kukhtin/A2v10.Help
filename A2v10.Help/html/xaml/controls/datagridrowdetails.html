Елемент DataGridRowDetails

Всі елементи керування

Являє собою елемент деталізації рядку для таблиці DataGrid.

Властивість вмісту: Content

Властивості

Ім'я Тип Опис
Content UIElementBase Вміст елементу.
Visible Boolean Чи видимий елемент.
Activate RowDetailsActivate Коли активувати деталізацію. Можливі значення:
  • ActiveRow - (за замовчуванням) показувати для виділеного рядку.
  • Cell - показувати спеціальну клітинку з можливістю розкривати деталізацію.
  • Always - показувати завжди.
