Елемент PeriodPicker

Все елементи керування

Представляет собой селектор периода.

Властивість Value всегда повинно бути связано с экземпляром класса TPeriod.

Наследует: ValuedControl : Control : UIElement : UIElementBase

Властивості

Им'я Тип Опис
Align TextAlign Выравнивание текста.
Placement DropDownPlacement Направление "выпадения" селектора. Можливі значения:
  • BottomLeft - (за замовчанням) внз с привязкой левого края.
  • BottomRight - внз с привязкой правого края.
  • TopLeft - вверх с привязкой левого края.
  • TopRight - вверх с привязкой правого края.
Style PeriodPickerStyle Вид селектора. Можливі значения:
  • Default - (за замовчанням) обычный селектор.
  • Hyperlink - отображение у вигляді гиперссылки.
Display DisplayMode Отображение периода в заголовке селектора. Можливі значения:
  • Date - (за замовчанням) тільки даты.
  • Name - название периода.
  • NameDate - название периода и даты.
ShowAllData Boolean Показывать ли елемент меню "За все время".
Size ControlSize Размер елементу. В поточної версії підтримуєся тільки для стиля Hyperlink Можливі значения:
  • Default - (за замовчанням) гиперссылка обычного розміру.
  • Large - увеличенный размер гиперссылки.
(остальные значення перечисления ControlSize не поддерживаются).
