Елемент PeriodPicker

Всі елементи керування

Являє собою селектор періоду.

Властивість Value завжди повинна бути зв'язана з екземпляром класа TPeriod.

Наслідує: ValuedControl : Control : UIElement : UIElementBase

Властивості

Ім'я Тип Опис
Align TextAlign Вирівнювання тексту.
Placement DropDownPlacement Напрямок "спадання" селектору. Можливі значення:
  • BottomLeft - (за замовчанням) донизу з прив'язкою лівого краю.
  • BottomRight - донизу з прив'язкою правого краю.
  • TopLeft - вверх з прив'язкою лівого краю.
  • TopRight - вверх з прив'язкою правого краю.
Style PeriodPickerStyle Вид селектора. Можливі значення:
  • Default - (за замовчанням) звичайний селектор.
  • Hyperlink - відображення у вигляді гіперпосилання.
Display DisplayMode Відображення періоду в заголовку селектора. Можливі значення:
  • Date - (за замовчанням) тільки дати.
  • Name - назва периода.
  • NameDate - назва періоду і дати.
ShowAllData Boolean Чи показувати елемент меню "За весь час".
Size ControlSize Розмір елементу. В поточній версії підтримується тільки для стиля Hyperlink Можливі значення:
  • Default - (за замовчанням) гіперпосилання звичайного розміру.
  • Large - збільшений розмір гіперпосилання.
(інші значення переліки ControlSize не підтримуються).
