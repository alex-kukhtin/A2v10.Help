Елемент PeriodPicker

Всі елементи керування

Являє собою селектор периода.

Властивість Value завжди повинно бути связано з екземпляром класса TPeriod.

Наслідує: ValuedControl : Control : UIElement : UIElementBase

Властивості

Ім'я Тип Опис
Align TextAlign Вирівнювання текста.
Placement DropDownPlacement Напрямок "выпадения" селектора. Можливі значення:
  • BottomLeft - (за замовчанням) внз з прив'язкою лівого края.
  • BottomRight - внз з прив'язкою правого края.
  • TopLeft - вверх з прив'язкою лівого края.
  • TopRight - вверх з прив'язкою правого края.
Style PeriodPickerStyle Вид селектора. Можливі значення:
  • Default - (за замовчанням) звичайний селектор.
  • Hyperlink - відображення у вигляді гіперпосилання.
Display DisplayMode Отображение периода в заголовке селектора. Можливі значення:
  • Date - (за замовчанням) тільки даты.
  • Name - название периода.
  • NameDate - название периода і даты.
ShowAllData Boolean Показывать ли елемент меню "За всі время".
Size ControlSize Розмір елементу. В поточної версії підтримується тільки для стиля Hyperlink Можливі значення:
  • Default - (за замовчанням) гіперпосилання звичайного розміру.
  • Large - увеличенный розмір гіперпосилання.
(остальные значення перечисления ControlSize не підтримуються).
