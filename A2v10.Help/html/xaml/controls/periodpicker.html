Элемент PeriodPicker

Все элементы управления

Представляет собой селектор периода.

Свойство Value всегда должно быть связано с экземпляром класса TPeriod.

Наследует: ValuedControl : Control : UIElement : UIElementBase

Свойства

Имя Тип Описание
Align TextAlign Выравнивание текста.
Placement DropDownPlacement Направление "выпадения" селектора. Возможные значения:
  • BottomLeft - (по умолчанию) вниз с привязкой левого края.
  • BottomRight - вниз с привязкой правого края.
  • TopLeft - вверх с привязкой левого края.
  • TopRight - вверх с привязкой правого края.
Style PeriodPickerStyle Вид селектора. Возможные значения:
  • Default - (по умолчанию) обычный селектор.
  • Hyperlink - отображение в виде гиперссылки.
Display DisplayMode Отображение периода в заголовке селектора. Возможные значения:
  • Date - (по умолчанию) только даты.
  • Name - название периода.
  • NameDate - название периода и даты.
ShowAllData Boolean Показывать ли элемент меню "За все время".
Size ControlSize Размер элемента. В текущей версии поддерживается только для стиля Hyperlink Возможные значения:
  • Default - (по умолчанию) гиперссылка обычного размера.
  • Large - увеличенный размер гиперссылки.
(остальные значения перечисления ControlSize не поддерживаются).
