Элемент ListItem

Все элементы управления

Представляет собой один элемент списка. Обычно используется внутри контейнера List.

Наследует: UIElement : UIElementBase

Свойство содержимого: Content

Свойства

Имя Тип Описание
Content Object Свойство содержимого. Текст элемента списка. Может быть как простым текстом, так и элементом управления (UIElementBase).
Header Object Заголовок элемента списка. Может быть как простым текстом, так и элементом управления (UIElementBase).
Footer Object Футер элемента списка. Может быть как простым текстом, так и элементом управления (UIElementBase).
Icon Icon Иконка, которая будет отображаться в элементе списка.
CommandBar CommandBar Панель команд, связанная с элементом списка.
Command BindCmd Команда для элемента списка. Она будет выполнена при нажатии на элемент списка. Обратите внимание, что использование команды вместе с CommandBar не имеет смысла. Нажать кнопку внутри панели команд будет невозможно.
