Елемент ListItem

Все елементи керування

Являє собою один елемент списка. Зазвичай використовується внутри контейнера List.

Наследует: UIElement : UIElementBase

Властивість вмісту: Content

Властивості

Ім'я Тип Опис
Content Object Властивість вмісту. Текст елементу списка. Може бути як простым текстом, так и елементом керування (UIElementBase).
Header Object Заголовок елементу списка. Може бути як простым текстом, так и елементом керування (UIElementBase).
Footer Object Футер елементу списка. Може бути як простым текстом, так и елементом керування (UIElementBase).
Icon Icon Иконка, яка буде отображаться в елементе списка.
CommandBar CommandBar Панель команд, связанная с елементом списка.
Command BindCmd Команда для елементу списка. Она буде выполнена при нажатии на елемент списка. Зверніть увагу, що использование команди вместе с CommandBar не має смысла. Нажать кнопку внутри панели команд буде невозможна.
