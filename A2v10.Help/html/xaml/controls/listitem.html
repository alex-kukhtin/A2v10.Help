Елемент ListItem

Всі елементи керування

Являє собою один елемент переліку. Зазвичай використовується всередині контейнера List.

Наслідує: UIElement : UIElementBase

Властивість вмісту: Content

Властивості

Ім'я Тип Опис
Content Object Властивість вмісту. Текст елементу переліку. Може бути як звичайним текстом, так і елементом керування (UIElementBase).
Header Object Заголовок елементу переліку. Може бути як звичайним текстом, так і елементом керування (UIElementBase).
Footer Object Футер елементу переліку. Може бути як звичайним текстом, так і елементом керування (UIElementBase).
Icon Icon Іконка, яка буде відображатися в елементі переліку.
CommandBar CommandBar Панель команд, пов'язана з елементом переліку.
Command BindCmd Команда для елементу переліку. Вона буде виконана при натисканні на елемент переліку. Зверніть увагу, що використання команди разом з CommandBar не має сенсу. Натиснути кнопку всередині панелі команд буде неможливо.
