Елемент Graphics

Все елементи керування

Являє собою місце для отрисовки графики. Сама отрисовка виконується программно с использованием делегата.

Наследует: UIElementBase

Властивості

Ім'я Тип Опис
Delegate String Ім'я делегата, который буде викликатися для отрисовки графики.
Argument Object Аргумент (обычно це прив'язка Bind), который буде передан в делегат.
CenterContent Boolean Центрировать ли вміст.
Watch WatchMode Режим отслеживания изменений при изменении значення аргумента. Можливі значення:
  • None - (за замовчанням) не отслеживать.
  • Watch - викликати делегат тільки при изменении аргумента.
  • Deep - викликати делегат при изменении аргумента або при изменении любых його вложенных властивостей (рекурсивно).
Height Length Высота елементу. За замовчанням Auto.
Властивості базового елементу

Формат делегата

draw(this:IRoot, chart:object, arg:any, node:DOMElement): void

Аргументи:

  • chart - об`єкт библиотеки d3.js.
  • arg - аргумент, описанный в разметке.
  • node - узел DOM, связанный с елементом.

Зверніть увагу! Зазвичай требуется щоб отрисовка графики выполнялась однократно. Однако, в некоторых случаях елемент, связанный с графикой може изменяться.
Установить режим отслеживания изменений можна за допомогою властивості Watch. Не устанавливайте режим Deep без необхідности, поскольку це приводит к снижению скорости роботи.
