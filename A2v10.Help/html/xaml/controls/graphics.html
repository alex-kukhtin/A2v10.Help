Элемент Graphics

Все элементы управления

Представляет собой место для отрисовки графики. Сама отрисовка выполняется программно с использованием делегата.

Наследует: UIElementBase

Свойства

Имя Тип Описание
Delegate String Имя делегата, который будет вызываться для отрисовки графики.
Argument Object Аргумент (обычно это привязка Bind), который будет передан в делегат.
CenterContent Boolean Центрировать ли содержимое.
Watch WatchMode Режим отслеживания изменений при изменении значения аргумента. Возможные значения:
  • None - (по умолчанию) не отслеживать.
  • Watch - вызывать делегат только при изменении аргумента.
  • Deep - вызывать делегат при изменении аргумента или при изменении любых его вложенных свойств (рекурсивно).
Height Length Высота элемента. По умолчанию Auto.
Свойства базового элемента

Формат делегата

draw(this:IRoot, chart:object, arg:any, node:DOMElement): void

Аргументы:

  • chart - объект библиотеки d3.js.
  • arg - аргумент, описанный в разметке.
  • node - узел DOM, связанный с элементом.

Обратите внимание! Обычно требуется чтобы отрисовка графики выполнялась однократно. Однако, в некоторых случаях элемент, связанный с графикой может изменяться.
Установить режим отслеживания изменений можно с помощью свойства Watch. Не устанавливайте режим Deep без необходимости, поскольку это приводит к снижению скорости работы.
