Елемент Selector

Всі елементи керування

Являє собою универсальный селектор.

Наслідує: ValuedControl : Control : UIElement : UIElementBase

Властивості

Ім'я Тип Опис
Delegate String Ім'я делегата, який буде викликатися при введенні текста в поле селектора.
SetDelegate String Ім'я делегата, який буде викликатися при выборе елементу в выпадающей панелі. Якщо указан цей делегат, то жодних присваиваний зв'язаного елементу виконуватися не буде.
TextValue Object Тільки прив'язка (Bind). Текстовая привязка. Якщо вона задана, то при выборе елементу в списке в нее буде передаваться текст, а не сам елемент.
Align TextAlign Вирівнювання текста.
Placeholder String Подсказка. Відображається в пустом поле введення. Исчезает при получении полем фокуса введення.
Style SelectorStyle Стиль відображення селектора. Можливі значення:
  • Default - (за замовчанням) звичайний селектор.
  • ComboBox - селектор у вигляді комбинированного выпадающйого списка. Фильтрация в цьому випадку не підтримується.
  • Hyperlink - селектор у вигляді гіперпосилання. Фильтрация в цьому випадку не підтримується.
Size ControlSize Розмір елементу. Можливі значення:
  • Default - (за замовчанням) селектор звичайного розміру.
  • Large - увеличенный розмір селектора.
(остальные значення перечисления ControlSize не підтримуються).
ItemsSource Array Завжди прив'язка - Bind. Джерело даних для відображення в списке.
DisplayProperty String Ім'я властивості основного елементу, которое буде відображатися в селекторе.
PanelPlacement DropDownPlacement Напрямок "выпадения" панелі вибору селектора. Можливі значення:
  • BottomLeft - (за замовчанням) внз з прив'язкою лівого края.
  • BottomRight - внз з прив'язкою правого края.
  • TopLeft - верх з прив'язкою лівого края.
  • TopRight - вверх з прив'язкою правого края.
ListSize Size Размеры выпадающей панелі селектора.
ItemsPanel UIElementBase Опис выпадающей панелі. В поточної версії в якості выпадающей панелі підтримується тільки елемент DataGrid. Якщо елемент не указан, використовується простий список елементів.
NewPane UIElementBase Опис панелі создания нового елементу.
CreateNewCommand BindCmd Команда, яка буде виконана при створенні нового елементу.
ShowCaret Boolean? Показывать значок открывания вмісту. Якщо стиль селектора ComboBox, то значення автоматично встановлюєся в True.
ShowClear Boolean Показывать кнопку очистки вмісту. Нажатие на эту кнопку викликає метод $empty для зв'язаного елементу.
Властивості базового елементу

Опис

Селектор може працювати в двух режимах. Выбор з списка і пошук в списке/на сервері.

В первом просто эмулируется комбинированный список (ComboBox) с додатковими возможнастями стилизации вікна вибору. В цьому випадку задається колекція для вибору (властивість ItemsSource) и ім'я властивості зв'язаного елементу (властивість DisplayProperty), которое повинно відображатися в селекторе.

Во втором случае потрібно задати делегат (властивість Delegate), який буде периодически викликатися при введенні текста в поле селектора.

Є можливість создания нового елементу у випадку, якщо в списке нічого не найдено. Для цього потрібно задати панель создания елементу NewPane і команду CreateNewCommand.

Формат делегата пошуку (Delegate)

fetch(this:IRoot, elem:IElement, text:string, all?: boolean): [] | Promise

Аргументи:

  • this - корінь моделі даних.
  • elem - елемент, пов'язаний з селектором.
  • text - текст, набранный в поле введення.
  • all - признак того, що надо выводить всі значення з списка. Будет дорівнює true, если установлено властивість ShowCaret і користувач открывает вікно вибору кнопкой, а не вводом текста в поле.

Значення, що повертається:

Делегат повинен повернути масив елементів або Promise, якщо елементи выбираются з сервера.

То що вернул делегат буде показано в панелі вибору.

Формат делегата установки (SetDelegate)

fetch(this:IRoot, elem:IElement): void

Аргументи:

  • this - корінь моделі даних.
  • elem - устанавливаемый (выбранный в списке) елемент.
