Представляет собой универсальный селектор.

Властивості

Delegate String Им'я делегата, который буде викликатися при вводе текста в поле селектора.
SetDelegate String Им'я делегата, который буде викликатися при выборе елементу в выпадающей панели. Якщо указан этот делегат, то жодних присваиваний связанного елементу виконуватися не буде.
TextValue Object Тільки привязка (Bind). Текстовая привязка. Якщо она задана, то при выборе елементу в списке в нее буде передаваться текст, а не сам елемент.
Align TextAlign Выравнивание текста.

  • ComboBox - селектор у вигляді комбинированного выпадающйого списка. Фильтрация в этом случае не підтримується.
  • Hyperlink - селектор у вигляді гиперссылки. Фильтрация в этом случае не підтримується.
Size ControlSize Размер елементу. Можливі значення:
  • Default - (за замовчанням) селектор звичайного розміру.
  • Large - увеличенный размер селектора.
(остальные значення перечисления ControlSize не поддерживаются).
ItemsSource Array Всегда привязка - Bind. Джерело даних для отображения в списке.
DisplayProperty String Им'я властивості основного елементу, которое буде отображаться в селекторе.
PanelPlacement DropDownPlacement Направление "выпадения" панели выбора селектора. Можливі значення:
  • BottomLeft - (за замовчанням) внз с привязкой левого края.
  • BottomRight - внз с привязкой правого края.
  • TopLeft - верх с привязкой левого края.
  • TopRight - вверх с привязкой правого края.
ListSize Size Размеры выпадающей панели селектора.
ItemsPanel UIElementBase Опис выпадающей панели. В поточної версії в качестве выпадающей панели підтримуєся тільки елемент DataGrid. Якщо елемент не указан, використовується простой список елементів.
NewPane UIElementBase Опис панели создания нового елементу.
CreateNewCommand BindCmd Команда, яка буде выполнена при создании нового елементу.
ShowCaret Boolean? Показывать значок открывания содержимого. Якщо стиль селектора ComboBox, то значення автоматически устанавливается в True.
ShowClear Boolean Показывать кнопку очистки содержимого. Нажатие на эту кнопку викликає метод $empty для связанного елементу.
Властивості базового елементу

Опис

Селектор може работать в двух режимах. Выбор з списка і пошук в списке/на сервере.

В первом просто эмулируется комбинированный список (ComboBox) с додатковими возможнастями стилизации вікна выбора. В цьому випадку задається коллекция для выбора (властивість ItemsSource) и ім'я властивості связанного елементу (властивість DisplayProperty), которое должно отображаться в селекторе.

Во втором случае потрібно задать делегат (властивість Delegate), который буде периодически викликатися при вводе текста в поле селектора.

Є можливість создания нового елементу у випадку, якщо в списке ничйого не найдено. Для цього потрібно задать панель создания елементу NewPane і команду CreateNewCommand.

Формат делегата пошуку (Delegate)

fetch(this:IRoot, elem:IElement, text:string, all?: boolean): [] | Promise

Аргументи:

  • this - корень моделі даних.
  • elem - елемент, связанный с селектором.
  • text - текст, набранный в поле ввода.
  • all - признак того, что надо выводить всі значення з списка. Будет равен true, если установлено властивість ShowCaret і користувач открывает окно выбора кнопкой, а не вводом текста в поле.

Возвращаемое значення:

Делегат повинен повернути масив елементів або Promise, якщо елементи выбираются с сервера.

То что вернул делегат буде показано в панели выбора.

Формат делегата установки (SetDelegate)

fetch(this:IRoot, elem:IElement): void

Аргументи:

  • this - корень моделі даних.
  • elem - устанавливаемый (выбранный в списке) елемент.
