Элемент Radio

Все элементы управления

Представляет собой радиокнопку.

Наследует: CheckBoxBase : ValuedControl : Control : UIElement : UIElementBase

Набор радиокнопок связывается с одним значением. Обычно такой набор располагается внутри контейнера FieldSet, хотя это и не обязательно.

Свойства

Имя Тип Описание
CheckedValue Скалярный тип (String, Number, и т.д.) Обязательно. Значение, при котором кнопка будет отмечена. Может быть привязкой.
Style RadioButtonStyle Вид кнопки. Может отображаться как стандартная радиокнопка, так и в виде флажка. Отображение в виде флажка чаще всего используется в отчетах.
