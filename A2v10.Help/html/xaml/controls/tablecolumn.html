Элемент TableColumn

Все элементы управления

Представляет собой одну колонку таблицы.

Свойства

Имя Тип Описание
Width Length Ширина колонки.
Fit Boolean Устанавливать ширину колонки по содержимому.
Background ColumnBackgroundStyle Цвет фона колонки.
If Boolean? Может быть привязкой. Включать ли колонку в таблицу. Обратите внимание, что исключение колонки исключает и все ячейки, которые в ней находятся.
