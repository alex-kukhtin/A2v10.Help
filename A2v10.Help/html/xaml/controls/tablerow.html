Елемент TableRow

Всі елементи керування

Являє собою рядок таблиці.

Наслідує: UIElement : UIElementBase

Властивість вмісту: Cells

Властивості

Ім'я Тип Опис
Cells Колекція TableCell Колекція клітинок в рядку.
Align TextAlign? Вирівнювання по горизонталі для всіх клітинок рядку. Може бути перевизначено в кожній клітинці індивідуально.
VAlign VerticalAlign Вертикальне вирівнювання текста. Може бути перевизначено в кожній клітинці індивідуально.
Mark Object Тільки прив'язка. Маркування рядку. Підробиці нижче.
Властивості базового елементу

Маркування рядків

Вираз прив`язки для властивості Mark повинен повертати текстовий рядок, який являє собою додатковий css-клас, що додається до рядку. В темі за замовчанням підтримуються наступні варіанти маркування рядків:

"danger", "error", "red" Вміст рядку
"warning", "orange" Вміст рядку
"success", "green" Вміст рядку
"info", "cyan" Вміст рядку

Для додавання додаткового елементу маркування можна використовувати клітинку TableMarkerCell.

Зверніть увагу! Маркування просто додає css-клас до рядку. Це означає, що ви не обмежені тільки вказаними варіантами. Наприклад, якщо властивість прив`язки поверне "danger bold", то рядок буде маркирований червоним фоном і буде відображатися напівжирним шрифтом.
