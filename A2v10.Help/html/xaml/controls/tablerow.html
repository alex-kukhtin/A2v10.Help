Свойства базового элемента

Маркировка строк

Выражение привязки для свойства Mark должно возвращать текстовую строку, которая представляет собой дополнительный css-класс, добавляемый к строке. В теме по умолчанию поддерживаются следующие варианты маркировки строк:

"danger", "error", "red" Содержимое строки
"warning", "orange" Содержимое строки
"success", "green" Содержимое строки
"info", "cyan" Содержимое строки

Для добавления дополнительного элемента маркировки можно использовать ячейку TableMarkerCell.

Обратите внимание! Маркировка просто добавляет css-класс к строке. Это означает, что вы не ограничены только указанными вариантами. Например, если свойство привязки вернет "danger bold", то строка будет маркирована красным фоном и будет показываться полужирным шрифтом.
