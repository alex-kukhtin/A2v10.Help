Елемент TableRow

Всі елементи керування

Являє собою строку таблиці.

Наслідує: UIElement : UIElementBase

Властивість вмісту: Cells

Властивості

Ім'я Тип Опис
Cells Колекція TableCell Колекція ячеек в строке.
Align TextAlign? Вирівнювання по горизонтали для всех ячеек рядки. Може бути переопределено в каждой ячейке индивидуально.
VAlign VerticalAlign Вертикальное вирівнювання текста. Може бути переопределено в каждой ячейке индивидуально.
Mark Object Тільки привязка. Маркировка рядки. Подробности ниже.
Властивості базового елементу

Маркировка строк

Вираз прив`язки для властивості Mark повинно возвращать текстовую строку, яка являє собою додатковий css-класс, добавляемый к строке. В теме за замовчанням поддерживаются наступні варианты маркировки строк:

"danger", "error", "red" Вміст рядки
"warning", "orange" Вміст рядки
"success", "green" Вміст рядки
"info", "cyan" Вміст рядки

Для додавання додаткового елементу маркировки можна використовувати ячейку TableMarkerCell.

Зверніть увагу! Маркировка просто добавляет css-класс к строке. Це означає, що вы не ограничены тільки вказаними вариантами. Наприклад, якщо властивість прив`язки вернет "danger bold", то рядок буде маркирована красным фоном і буде відображатися полужирным шрифтом.
