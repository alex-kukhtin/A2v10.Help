Элемент MenuItem

Все элементы управления

Представляет собой элемент меню. Обычно находится в контейнере DropDownMenu.

Наследует: CommandControl : ContentControl : Control : UIElement : UIElementBase

Свойство содержимого: Content

Свойства

Имя Тип Описание
Icon Icon Иконка элемента.
Separator SeparatorMode Связанный разделитель. Возможные значения:
  • None - (по умолчанию) нет.
  • Before - добавить разделитель перед элементом меню.
  • After - добавить разделитель после элемента меню.
