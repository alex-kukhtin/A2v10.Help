Елемент MenuItem

Всі елементи керування

Являє собою елемент меню. Зазвичай знаходиться в контейнері DropDownMenu.

Наслідує: CommandControl : ContentControl : Control : UIElement : UIElementBase

Властивість вмісту: Content

Властивості

Ім'я Тип Опис
Icon Icon Іконка елементу.
Separator SeparatorMode Зв'язаний роздільник. Можливі значення:
  • None - (за замовчуванням) нема роздільника.
  • Before - додати роздільник перед елементом меню.
  • After - додати роздільник після елементу меню.
