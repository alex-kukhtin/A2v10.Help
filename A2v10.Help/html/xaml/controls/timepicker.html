Елемент TimePicker

Все елементи керування

Являє собою селектор времени.

Наследует: ValuedControl : Control : UIElement : UIElementBase

Властивості

Им'я Тип Опис
Align TextAlign Выравнивание текста.
Placement DropDownPlacement Направление "выпадения" селектора. Можливі значення:
  • BottomLeft - (за замовчанням) внз с привязкой левого края.
  • BottomRight - внз с привязкой правого края.
  • TopLeft - верх с привязкой левого края.
  • TopRight - вверх с привязкой правого края.
Size ControlSize Размер елементу. Можливі значення:
  • Default, Normal - (за замовчанням) елемент звичайного розміру.
  • Large - увеличенный размер елементу.
(остальные значення перечисления ControlSize не підтримуються).
Властивості базового елементу

Приклад

Зверніть увагу! При изменении времени, дата меняться не буде. Це дозволяє вводить дату і время с помощью двух елементів керування - DatePicker і TimePicker.
