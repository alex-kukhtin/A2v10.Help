Елемент TreeGridColumn

Всі елементи керування

Являє собою один стовпчик таблиці TreeGrid.

Властивість вмісту: Content

Властивості

Ім'я Тип Опис
Content Bind Властивість вмісту. Тільки прив'язка (Bind). Вміст клітинки таблиці.
ShowButton Boolean Показувати в цьому стовпчику елементи керування деревом.
Header String Заголовок стовпчика.
Width Length Ширина стовпчика.
Fit Boolean Встановлювати ширину стовпчика по вмісту.
Wrap WrapMode Режим перенесення для вмісту клітинки і для заголовку.
Align TextAlign Горизонтальне вирівнювання вмісту клітинки.
Bold Boolean Відображати вміст клітинки напівжирним шрифтом
