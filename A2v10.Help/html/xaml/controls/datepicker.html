Елемент DatePicker

Всі елементи керування

Являє собою селектор дати.

Наслідує: ValuedControl : Control : UIElement : UIElementBase

Властивості

Ім'я Тип Опис
Align TextAlign Вирівнювання тексту.
View DatePickerView Вид селектора. Можливі значення:
  • Day - (за замовчанням) вибір дати.
  • Month - вибір місяцю та року.
Placement DropDownPlacement Напрямок "спадання" селектора. Можливі значення:
  • BottomLeft - (за замовчанням) донизу з прив'язкою лівого краю.
  • BottomRight - донизу з прив'язкою правого краю.
  • TopLeft - вверх з прив'язкою лівого краю.
  • TopRight - ввверх з прив'язкою правого краю.
Size ControlSize Розмір елементу. Можливі значення:
  • Default, Normal - (за замовчанням) елемент звичайного розміру.
  • Large - збільшений розмір елементу.
(всі решта значень переліку ControlSize не підтримуються).
YearCutOff String Тільки для ручного введення. Обмеження для двохзначних значень року. Двозначне число. Якщо починається з "+" або "-", то число розглядається відносно поточного року. Якщо введене значення менше або дорівнює обмеженню, то це 21-й вік, інакше 20-й.
Властивості базового елементу

Приклад

Зверніть увагу! При зміні дати час змінюватися не буде. Це дозволяє вводити дату і час за допомогою двох елементів керування - TimePicker і DatePicker.
