Элемент DatePicker

Все элементы управления

Представляет собой селектор даты.

Наследует: ValuedControl : Control : UIElement : UIElementBase

Свойства

Имя Тип Описание
Align TextAlign Выравнивание текста.
View DatePickerView Вид селектора. Возможные значения:
  • Day - (по умолчанию) выбор даты.
  • Month - выбор месяца и года.
Placement DropDownPlacement Направление "выпадения" селектора. Возможные значения:
  • BottomLeft - (по умолчанию) вниз с привязкой левого края.
  • BottomRight - вниз с привязкой правого края.
  • TopLeft - верх с привязкой левого края.
  • TopRight - вверх с привязкой правого края.
Size ControlSize Размер элемента. Возможные значения:
  • Default, Normal - (по умолчанию) элемент обычного размера.
  • Large - увеличенный размер элемента.
(остальные значения перечисления ControlSize не поддерживаются).
YearCutOff String Только для ручного ввода. Предел для двухзначных значений года. Двузначное число. Если начинается с "+" или "-", то число рассматривается относительно текущего года. Если введенное значение меньше или равно пределу, то это 21-й век, иначе 20-й.
