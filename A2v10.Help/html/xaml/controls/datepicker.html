Елемент DatePicker

Все елементи керування

Являє собою селектор даты.

Наследует: ValuedControl : Control : UIElement : UIElementBase

Властивості

Ім'я Тип Опис
Align TextAlign Выравнивание текста.
View DatePickerView Вид селектора. Можливі значення:
  • Day - (за замовчанням) выбор даты.
  • Month - выбор месяца і года.
Placement DropDownPlacement Направление "выпадения" селектора. Можливі значення:
  • BottomLeft - (за замовчанням) внз з прив'язкою левого края.
  • BottomRight - внз з прив'язкою правого края.
  • TopLeft - верх з прив'язкою левого края.
  • TopRight - вверх з прив'язкою правого края.
Size ControlSize Размер елементу. Можливі значення:
  • Default, Normal - (за замовчанням) елемент звичайного розміру.
  • Large - увеличенный размер елементу.
(остальные значення перечисления ControlSize не підтримуються).
YearCutOff String Тільки для ручного ввода. Предел для двухзначных значень года. Двузначное число. Якщо починається з "+" або "-", то число розглядається відносно поточного года. Якщо введенное значення меньше або дорівнює пределу, то це 21-й век, иначе 20-й.
Властивості базового елементу

Приклад

Зверніть увагу! При изменении даты, время меняться не буде. Це дозволяє вводить дату і время с помощью двух елементів керування - TimePicker і DatePicker.
