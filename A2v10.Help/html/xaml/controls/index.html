Элементы управления

Элементы управления (Controls) используются для ввода, изменения или отображения данных.

Элементы управления

  • Alert - панель предупреждения
  • Button - кнопка.
  • ChatItem - элемент чата.
  • CheckBox - флажок.
  • Code - текст программы.
  • ComboBox - выпадающий список.
  • ComboBoxItem - элемент выпадающего списка.
  • DataGrid - таблица данных.
  • DataGridColumn - колонка таблицы данных.
  • DataGridRowDetails - детализация строки таблицы данных.
  • DatePicker - выбор даты.
  • EmptyPanel - панель для пустого содержимого.
  • FileImage - картинка.
  • Graphics - контейнер для графического изображения.
  • Header - заголовок.
  • Image - картинка.
  • Label - метка.
  • List - список.
  • ListItem - элемент списка.
  • MenuItem - элемент меню.
  • Pager - пейджер (разбивка на страницы).
  • PeriodPicker - выбор периода.
  • PropertyGrid - таблица свойств.
  • PropertyGridItem - элемент таблицы свойств.
  • Radio - радиокнопка.
  • Selector - селектор.
  • Separator - разделитель.
  • StateCard - карточка.
  • Static - статический текст.
  • TabButton - кнопка панели закладок.
  • Table - таблица.
  • TableCell - ячейка таблицы.
  • TableColumn - колонка таблицы.
  • TableMarkCell - ячейка маркера в таблице.
  • TableRow - строка таблицы.
  • TextBox - текстовый редактор.
  • TimePicker - селектор выбора времени.
  • TreeView - иерархический список (дерево).
  • TreeViewItem - элемент дерева.
  • UploadFile - загрузка файла на сервер.
