Елементи керування

Елементи керування (Controls) використовуються для введення, змінення або відображення даних.

Елементи керування

  • Alert - панель попередження
  • Button - кнопка.
  • ChatItem - елемент чату.
  • CheckBox - прапорець.
  • Code - текст програми.
  • ComboBox - спадний список.
  • ComboBoxItem - елемент спадного списку.
  • DataGrid - таблиця даних.
  • DataGridColumn - стовпчик таблиці даних.
  • DataGridRowDetails - деталізація рядку таблиці даних.
  • DatePicker - вибір дати.
  • EmptyPanel - панель для пустого вмісту.
  • FileImage - зображення.
  • Graphics - контейнер для графічного зображення.
  • Header - заголовок.
  • Image - зображення.
  • Label - мітка.
  • List - список.
  • ListItem - елемент списку.
  • MenuItem - елемент меню.
  • Pager - пейджер (розбиття на сторінки).
  • PeriodPicker - вибір періоду.
  • PropertyGrid - таблиця властивостей.
  • PropertyGridItem - елемент таблиці властивостей.
  • Radio - радіокнопка.
  • Selector - селектор.
  • Separator - роздільник.
  • StateCard - картка.
  • Static - статичний текст.
  • TabButton - кнопка панелі закладок.
  • Table - таблиця.
  • TableCell - клітинка таблиці.
  • TableColumn - стовпчик таблиці.
  • TableMarkCell - клітинка маркера в таблиці.
  • TableRow - рядок таблиці.
  • TextBox - текстовий редактор.
  • TimePicker - селектор вибору часу.
  • TreeView - ієрархічний список (дерево).
  • TreeViewItem - елемент дерева.
  • UploadFile - завантаження файлу на сервер.
