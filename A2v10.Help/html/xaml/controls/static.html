Элемент Static

Все элементы управления

Представляет собой статическую метку, которая выглядит как запрещенный текстовый элемент.

Наследует: ValuedControl : Control : UIElement : UIElementBase

Свойства

Имя Тип Описание
Align TextAlign Выравнивание текста.
Size ControlSize Размер элемента. Возможные значения:
  • Default, Normal - (по умолчанию) элемент обычного размера.
  • Large - увеличенный размер элемента.
(остальные значения перечисления ControlSize не поддерживаются).
