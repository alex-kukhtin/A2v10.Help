Елемент DataGridColumn

Все елементи керування

Являє собою одну колонку таблиці DataGrid.

Властивість вмісту: Content

Властивості

Им'я Тип Опис
Content Bind Властивість вмісту. Тільки прив'язка (Bind). Вміст ячейки таблиці.
Header String Заголовок колонки.
Width Length Ширина колонки.
Fit Boolean Устанавливать ширину колонки по содержимому.
Wrap WrapMode Режим переноса для вмісту ячейки і для заголовка.
Align TextAlign Горизонтальное выравнивание вмісту ячейки.
Bold Boolean Отображать вміст ячейки полужирным шрифтом
Small Boolean Отображать вміст ячейки уменьшенным шрифтом.
Mark Object Тільки прив'язка (Bind). Маркировка ячейки таблиці.
Icon Icon Иконка, яка буде отображаться в таблице перед содержимым.
Editable Boolean Разрешено ли редактирование ячейки.
ControlType ColumnControlType Тип елементу керування в ячейке таблиці.
Sort Boolean? Разрешена ли сортировка в данной колонке. Позволяет индивидуально перевизначити сортировку по каждой колонке. Общее разрешение сортировки задається властивістю Sort всей таблиці DataGrid.
SortProperty String Им'я властивості в джерелі даних, по которому буде виконуватися сортировка. Якщо не вказано, буде виконуватися сортировка по властивості Content.
If Boolean? Включать колонку в поток рендеринга. Для того, щоб показывать/скрывать колонки в залежності від умов використовується прив'язка (Bind).
Command BindCmd Команда для ячейки таблиці. Якщо команда указана, то вміст (Content) буде выводится у вигляді гиперссылки со связанной командой.
Role ColumnRole Роль колонки. Встановлює определенный набір властивостей для колонок предопределенных типов. Можливі значення:
  • Default - (за замовчанням). Ничйого не устанавливается.
  • Number - Отображение идентификатора. Ширина по содержимому, выравнивание вправо, нет переноса (Fit="True" Align="Right" Wrap="NoWrap").
  • Number - Отображение числа. Ширина по содержимому, выравнивание вправо, нет переноса (Fit="True" Align="Right" Wrap="NoWrap").
  • Date - Отображение даты. Ширина по содержимому, выравнивание по центру, нет переноса (Fit="True" Align="Center" Wrap="NoWrap").
