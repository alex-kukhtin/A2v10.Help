Елемент TextBox

Все елементи керування

Являє собою поле для ввода текста.

Наследует: ValuedControl : Control : UIElement : UIElementBase

Властивості

Им'я Тип Опис
Multiline Boolean Многострочный елемент.
AutoSize Boolean Автоматически устанавливать высоту елементу в залежності від вмісту. Тільки для многострочных елементів.
Rows Int32? Количество строк. Тільки для многострочных елементів.
Align TextAlign Выравнивание текста
Password Boolean Поле ввода пароля. Замість текста отображаются заполнители. Копіювання текста з поля невозможна.
Number Boolean В поле разрешено вводить тільки цифры і знак минус ('-').
Placeholder String Подсказка. Отображается в пустом поле ввода. Исчезает при получении полем фокуса ввода.
SpellCheck Boolean? Чи перевіряти орфографические ошибки. За замовчанням - зависит від налаштувань браузера.
UpdateTrigger UpdateTrigger Способ оновлення значення моделі даних, связанных с этим елементом.
EnterCommand BindCmd Команда, яка буде выполнена при нажатии Enter (для многострочного поля - Ctrl+Enter).
MaxHeight Length Максимальная высота елементу. Має сенс тільки для многострочных елементів, у которых установлено властивість AutoSize.
Size ControlSize Размер елементу. Можливі значення:
  • Default, Normal - (за замовчанням) елемент звичайного розміру.
  • Large - увеличенный размер елементу.
(остальные значення перечисления ControlSize не підтримуються).
ShowClear Boolean Показывать кнопку очистки вмісту. Нажатие на эту кнопку очищает связанное значення моделі. Якщо значення пустое, кнопка буде запрещена.
Accel Accel Акселератор (горячая клавиша). При нажатии этой клавиши фокус буде установлен на цей елемент.
