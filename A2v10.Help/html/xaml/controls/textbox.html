Елемент TextBox

Всі елементи керування

Являє собою поле для введення текста.

Наслідує: ValuedControl : Control : UIElement : UIElementBase

Властивості

Ім'я Тип Опис
Multiline Boolean Многорядковий елемент.
AutoSize Boolean Автоматически устанавливать высоту елементу в залежності від вмісту. Тільки для многострочных елементів.
Rows Int32? Кількість строк. Тільки для многострочных елементів.
Align TextAlign Вирівнювання текста
Password Boolean Поле введення пароля. Замість текста відображаються заполнители. Копіювання текста з поля невозможна.
Number Boolean В поле разрешено вводить тільки цифры і знак минус ('-').
Placeholder String Подсказка. Відображається в пустом поле введення. Исчезает при получении полем фокуса введення.
SpellCheck Boolean? Чи перевіряти орфографические ошибки. За замовчанням - залежить від налаштувань браузера.
UpdateTrigger UpdateTrigger Способ оновлення значення моделі даних, зв'язаних з цим елементом.
EnterCommand BindCmd Команда, яка буде виконана при нажатии Enter (для многострочного поля - Ctrl+Enter).
MaxHeight Length Максимальна висота елементу. Має сенс тільки для многострочных елементів, у которых установлено властивість AutoSize.
Size ControlSize Розмір елементу. Можливі значення:
  • Default, Normal - (за замовчанням) елемент звичайного розміру.
  • Large - увеличенный розмір елементу.
(остальные значення перечисления ControlSize не підтримуються).
ShowClear Boolean Показывать кнопку очистки вмісту. Нажатие на эту кнопку очищает пов'язане значення моделі. Якщо значення пустое, кнопка буде запрещена.
Accel Accel Акселератор (горячая клавиша). При нажатии этой клавиши фокус буде установлен на цей елемент.
