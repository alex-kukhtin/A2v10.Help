Елемент TextBox

Всі елементи керування

Являє собою поле для введення тексту.

Наслідує: ValuedControl : Control : UIElement : UIElementBase

Властивості

Ім'я Тип Опис
Multiline Boolean Елемент з кількома рядками.
AutoSize Boolean Автоматично встановлювати висоту елементу в залежності від вмісту. Тільки для елементів з кількома рядками.
Rows Int32? Кількість рядків. Тільки для елементів з кількома рядками.
Align TextAlign Вирівнювання тексту
Password Boolean Поле введення пароля. Замість текста відображаються заповнювачі. Копіювання тексту з поля неможливе.
Number Boolean В поле дозволено вводити тільки цифри і знак мінус ('-').
Placeholder String Підказка. Відображається в пустому полі введення. Зникає при отриманні полем фокусу введення.
SpellCheck Boolean? Чи перевіряти орфографічні помилки. За замовчуванням - залежить від налаштувань браузера.
UpdateTrigger UpdateTrigger Спосіб оновлення значення моделі даних, зв'язаних з цим елементом.
EnterCommand BindCmd Команда, яка буде виконана при натисканні Enter (для поля з багатьма рядками - Ctrl+Enter).
MaxHeight Length Максимальна висота елементу. Має сенс тільки для елементів з кількома рядками, у яких встановлено властивість AutoSize.
Size ControlSize Розмір елементу. Можливі значення:
  • Default, Normal - (за замовчуванням) елемент звичайного розміру.
  • Large - збільшений розмір елементу.
(інші значення переліку ControlSize не підтримуються).
ShowClear Boolean Показувати кнопку очистки вмісту. Натискання на цю кнопку очищує пов'язане значення моделі. Якщо значення пусте - кнопку буде заборонено.
Accel Accel Акселератор (гаряча клавіша). При натисканні цієї клавіші фокус буде встановлено на цей елемент.
