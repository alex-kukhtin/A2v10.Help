Элемент TextBox

Все элементы управления

Представляет собой поле для ввода текста.

Наследует: ValuedControl : Control : UIElement : UIElementBase

Свойства

Имя Тип Описание
Multiline Boolean Многострочный элемент.
AutoSize Boolean Автоматически устанавливать высоту элемента в зависимости от содержимого. Только для многострочных элементов.
Rows Int32? Количество строк. Только для многострочных элементов.
Align TextAlign Выравнивание текста
Password Boolean Поле ввода пароля. Вместо текста отображаются заполнители. Копирование текста из поля невозможно.
Number Boolean В поле разрешено вводить только цифры и знак минус ('-').
Placeholder String Подсказка. Отображается в пустом поле ввода. Исчезает при получении полем фокуса ввода.
SpellCheck Boolean? Проверять ли орфографические ошибки. По умолчанию - зависит от настроек браузера.
UpdateTrigger UpdateTrigger Способ обновления значения модели данных, связанных с этим элементом.
EnterCommand BindCmd Команда, которая будет выполнена при нажатии Enter (для многострочного поля - Ctrl+Enter).
MaxHeight Length Максимальная высота элемента. Имеет смысл только для многострочных элементов, у которых установлено свойство AutoSize.
Size ControlSize Размер элемента. Возможные значения:
  • Default, Normal - (по умолчанию) элемент обычного размера.
  • Large - увеличенный размер элемента.
(остальные значения перечисления ControlSize не поддерживаются).
ShowClear Boolean Показывать кнопку очистки содержимого. Нажатие на эту кнопку очищает связанное значение модели. Если значение пустое, кнопка будет запрещена.
Accel Accel Акселератор (горячая клавиша). При нажатии этой клавиши фокус будет установлен на этот элемент.
