Елемент UploadFile

Все елементи керування

Являє собою елемент керування для выбора і завантаження файлів. Поддерживает drag'n'drop.

Детальніше про роботі с двоичными об'єктами...

Наследует: UIElementBase

Властивості

Им'я Тип Опис
Url String Путь к команде завантаження файла.
Argument Object Об'єкт, властивості якого будуть параметрами команди завантаження файла.
Accept String Рядок фильтра для діалога выбора файла. Mime-типы принимаемых файлів. Поддержка зависит від браузера. Соответствует атрибуту accept елементу <input type="file">. Можна указывать декілька типов через запятую. Например, для выбора зображень в форматі jpg або png можна вказати Accept="image/png, image/jpeg".
Зверніть увагу, що це тільки рекомендация браузеру. Запретить користувачу выбирать інші файли невозможна.
Limit Int32 Максимально допустимый размер загружаемого файла (в килобайтах). Якщо не указан, можна загружать файли любого розміру.
Delegate String Им'я делегата, который буде вызван після того, как файл буде загружен. Аргументом делегата буде полученный с сервера результат.
ErrorDelegate String Им'я делегата, который буде вызван при виникненні ошибок. Якщо не указан, выводится стандартне повідомлення об ошибке.
