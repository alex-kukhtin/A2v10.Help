Елемент UploadFile

Всі елементи керування

Являє собою елемент керування для вибору і завантаження файлів. Поддерживает drag'n'drop.

Детальніше про роботі з бінарними об'єктами...

Наслідує: UIElementBase

Властивості

Ім'я Тип Опис
Url String Шлях до команде завантаження файла.
Argument Object Об'єкт, властивості якого будуть параметрами команди завантаження файла.
Accept String Рядок фільтра для діалога вибору файла. Mime-типы принимаемых файлів. Поддержка залежить від браузера. Соответствует атрибуту accept елементу <input type="file">. Можна указывать декілька типів через запятую. Наприклад, для вибору зображень в форматі jpg або png можна вказати Accept="image/png, image/jpeg".
Зверніть увагу, що це тільки рекомендация браузеру. Заборонити користувачу выбирать інші файли невозможна.
Limit Int32 Максимально допустимый розмір загружаемого файла (в килобайтах). Якщо не указан, можна завантажувати файли будь-якого розміру.
Delegate String Ім'я делегата, який буде вызван після того, как файл буде загружен. Аргументом делегата буде полученный з сервера результат.
ErrorDelegate String Ім'я делегата, який буде вызван при виникненні ошибок. Якщо не указан, виводиться стандартне повідомлення про помилку.
