Элемент UploadFile

Все элементы управления

Представляет собой элемент управления для выбора и загрузки файлов. Поддерживает drag'n'drop.

Подробнее о работе с двоичными объектами...

Наследует: UIElementBase

Свойства

Имя Тип Описание
Url String Путь к команде загрузки файла.
Argument Object Объект, свойства которого будут параметрами команды загрузки файла.
Accept String Строка фильтра для диалога выбора файла. Mime-типы принимаемых файлов. Поддержка зависит от браузера. Соответствует атрибуту accept элемента <input type="file">. Можно указывать несколько типов через запятую. Например, для выбора картинок в формате jpg или png можно указать Accept="image/png, image/jpeg".
Обратите внимание, что это только рекомендация браузеру. Запретить пользователю выбирать другие файлы невозможно.
Limit Int32 Максимально допустимый размер загружаемого файла (в килобайтах). Если не указан, можно загружать файлы любого размера.
Delegate String Имя делегата, который будет вызван после того, как файл будет загружен. Аргументом делегата будет полученный с сервера результат.
ErrorDelegate String Имя делегата, который будет вызван при возникновении ошибок. Если не указан, выводится стандартное сообщение об ошибке.
