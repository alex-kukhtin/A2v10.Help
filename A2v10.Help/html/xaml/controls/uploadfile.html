Елемент UploadFile

Всі елементи керування

Являє собою елемент керування для вибору і завантаження файлів. Підтримує drag'n'drop.

Детальніше про роботі з бінарними об'єктами...

Наслідує: UIElementBase

Властивості

Ім'я Тип Опис
Url String Шлях до команди завантаження файла.
Argument Object Об'єкт, властивості якого будуть параметрами команди завантаження файла.
Accept String Рядок фільтра для діалогу вибору файла. Mime-типи файлів, що отримуються. Підтримка залежить від браузера. Відповідає атрибуту accept елементу <input type="file">. Можна вказувати декілька типів через кому. Наприклад, для вибору зображень в форматі jpg або png можна вказати Accept="image/png, image/jpeg".
Зверніть увагу, що це тільки рекомендація браузеру. Заборонити користувачу вибирати інші файли неможливо.
Limit Int32 Максимально припустимий розмір файлу, що завантажується (в килобайтах). Якщо не вказано, можна завантажувати файли будь-якого розміру.
Delegate String Ім'я делегата, який буде викликано після того, как файл буде завантажено. Аргументом делегата буде отриманий з сервера результат.
ErrorDelegate String Ім'я делегата, який буде викликано при виникненні помилок. Якщо не вказано, виводиться стандартне повідомлення про помилку.
