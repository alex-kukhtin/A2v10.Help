Елемент Table

Всі елементи керування

Являє собою таблицю.

Наслідує: Control : UIElement : UIElementBase

Властивість вмісту: Rows

Властивості

Ім'я Тип Опис
Rows Колекція TableRow Рядки тіла таблиці.
Header Колекція TableRow Рядки заголовку таблиці.
Footer Колекція TableRow Рядки підвалу таблиці.
Columns Колекція TableColumn Стовпчики таблиці.
ItemsSource Array Завжди прив'язка (Bind). Джерело даних для отримання рядків тіла таблиці.
Border Boolean Рамка навколо таблиці.
Compact Boolean Компактный стиль відображення таблиці.
Hover Boolean Виділення рядків при наведенні курсора миші.
Striped Boolean Розфарбування таблиці "у смужку" з виділенням парних і непарних рядків.
GridLines GridLinesVisibility Відображення сітки. Можливі значення:
  • None - за замовчанням.
  • Horizontal - горизонтальні лінії.
  • Vertical - вертикальні лінії.
  • Both - сітка.
Background TableBackgroundStyle Колір фону. Можливі значення:
  • None - за замовчанням. Прозорий фон.
  • Paper - колір паперу.
  • Yellow - блідий жовтий.
  • Cyan - блідий блакитний.
  • Rose - блідий рожевий.
  • WhiteSmoke - димчатий.
CellSpacing CellSpacingMode Відстань між клітинками. Якщо вказано, то кожна клітинка буде відображатися окремо. Можливі значення:
  • None - за замовчанням. Нет.
  • Small - маленьке.
  • Medium - среднє.
  • Large - велике.
