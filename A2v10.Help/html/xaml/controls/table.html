Элемент Table

Все элементы управления

Представляет собой таблицу.

Наследует: Control : UIElement : UIElementBase

Свойство содержимого: Rows

Свойства

Имя Тип Описание
Rows Коллекция TableRow Строки тела таблицы.
Header Коллекция TableRow Строки заголовка таблицы.
Footer Коллекция TableRow Строки подвала таблицы.
Columns Коллекция TableColumn Колонки таблицы.
ItemsSource Array Всегда привязка (Bind). Источник данных для получения строк тела таблицы.
Border Boolean Рамка вокруг таблицы.
Compact Boolean Компактный стиль отображения таблицы.
Hover Boolean Выделение строки при наведении курсора мыши.
Striped Boolean Раскраска таблицы "в полоску" с выделением четных и нечетных строк.
GridLines GridLinesVisibility Отображение сетки. Возможные значения:
  • None - по умолчанию.
  • Horizontal - горизонтальные линии.
  • Vertical - вертикальные линии.
  • Both - сетка.
Background TableBackgroundStyle Цвет фона. Возможные значения:
  • None - по умолчанию. Прозрачный фон.
  • Paper - цвет бумаги.
  • Yellow - бледно-желтый.
  • Cyan - бледно-голубой.
  • Rose - бледно-розовый.
  • WhiteSmoke - дымчатый.
CellSpacing CellSpacingMode Расстояние между ячейками. Если указано, то каждая ячейка будет отображаться отдельно. Возможные значения:
  • None - по умолчанию. Нет.
  • Small - маленькое.
  • Medium - среднее.
  • Large - большое.
