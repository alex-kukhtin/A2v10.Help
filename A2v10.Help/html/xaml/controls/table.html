Елемент Table

Все елементи керування

Являє собою таблицю.

Наследует: Control : UIElement : UIElementBase

Властивість вмісту: Rows

Властивості

Ім'я Тип Опис
Rows Колекція TableRow Строки тела таблиці.
Header Колекція TableRow Строки заголовка таблиці.
Footer Колекція TableRow Строки подвала таблиці.
Columns Колекція TableColumn Колонки таблиці.
ItemsSource Array Завжди прив'язка (Bind). Джерело даних для отримання рядків тела таблиці.
Border Boolean Рамка вокруг таблиці.
Compact Boolean Компактный стиль відображення таблиці.
Hover Boolean Выделение строки при наведении курсора мыши.
Striped Boolean Раскраска таблиці "в полоску" з выделением четных і нечетных строк.
GridLines GridLinesVisibility Отображение сетки. Можливі значення:
  • None - за замовчанням.
  • Horizontal - горизонтальные линии.
  • Vertical - вертикальные линии.
  • Both - сетка.
Background TableBackgroundStyle Цвет фона. Можливі значення:
  • None - за замовчанням. Прозрачный фон.
  • Paper - цвет бумаги.
  • Yellow - бледно-желтый.
  • Cyan - бледно-голубой.
  • Rose - бледно-розовый.
  • WhiteSmoke - дымчатый.
CellSpacing CellSpacingMode Расстояние между ячейками. Якщо указано, то кожна ячейка буде отображаться отдельно. Можливі значення:
  • None - за замовчанням. Нет.
  • Small - маленькое.
  • Medium - среднее.
  • Large - большое.
