Элемент TreeView

Все элементы управления

Представляет собой иерархический список (дерево). Поддерживает "ленивую" загрузку элементов.

Обратите внимание, что элемент используется для работы с иерархическими моделями данных.

Наследует: Control : UIElement : UIElementBase

Свойство содержимого: Children

Свойства

Имя Тип Описание
Children Коллекция TreeViewItem Свойство содержимого. Как правило состоит из одного элемента, свойства которого задают привязки к элементам дерева.
ItemsSource Array Всегда привязка (Bind). Источник данных для получения элементов дерева первого уровня. Привязка остальных уровней определяется в дочернем элементе TreeViewItem.
Style TreeViewStyle Стиль отображения дерева. Возможные значения:
  • Normal - по умолчанию.
  • SideBarMenu - выглядит как боковое меню.
AutoSelect AutoSelectMode Автоматический выбор элемента при загрузке. Возможные значения:
  • None - по умолчанию. Не выбирать ничего
  • FirstItem - первый элемент.
  • LastItem - последний элемент.
  • ItemId - элемент с заданным идентификатором. Это идентификатор основного объекта модели и, как правило, является последним элементом url страницы.
IconFolder Icon Иконка для папки. Обратите внимание, что это иконка "по умолчанию". Она может быть переопределена в элементе TreeViewItem.
IconItem Icon Иконка для элемента. Обратите внимание, что это иконка "по умолчанию". Она может быть переопределена в элементе TreeViewItem.
WrapLabel Boolean Разрешен ли перенос текста метки дерева. По умолчанию - False.
FolderSelect Boolean Разрешено ли выбирать папки. Если равно False, то выбрать можно только элемент дерева.
ExpandFirstItem Boolean Автоматически раскрывать первый элемент дерева после загрузки.
Height Length Высота элемента.
Indent Boolean? Отступы для вложенных элементов. По умолчанию - True.
DoubleClick BindCmd Команда, которая будет выполнена при двойном щелчке на элементе дерева. Не забудьте передать в команду аргумент.
ContextMenu DropDownMenu Контекстное меню, которое будет показыватьcя по правой кнопке мыши.
