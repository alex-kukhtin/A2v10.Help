Елемент SelectorSimple

Всі елементи керування

Являє собою спрощений селектор.

Фактично це скорочений запис для звичайного елемента Selector в якому встановлені деякі властивості.

Наслідує: Selector : ValuedControl : Control : UIElement : UIElementBase

Властивості

Ім'я Тип Опис
Url String URL кінцевої точки. В цій точці мають бути діалог "browse" та команда "fetch".
Data Object Завжди прив'язка - Bind. Додаткові параметри для вибору.
Властивості базового елементу

Зверніть увагу! Якщо в елементі задано властивість Data, вона буде використовуватися в якості параметрів, як для виклику діалогу вибору (browse), так і для отримання даних з сервера (fetch).
Фактично Data просто дублюється в FetchData базового селектора.
