Им'я Тип Опис

Властивості базового елементу

Особенности

Частіше за все задається тільки одно з властивостей Width або Height. В цьому випадкут картинка буде сохранять пропорции.

Приклад

Елемент керування Image використовується для одновременного показа и завантаження картинки з використанням упрощенного подхода.

Властивість Source должно ссылаться на властивість моделі, которое являє собою об`єкт с двумя полями (Id!!Id и Token!!Token). Им'я этого властивості буде определять суффикс для хранимых процедур, які будуть викликатися для отримання або записи реальных байтов изображения.

Властивість Base должно ссылаться на полную модель (ту, яка указана в корне файла model.json).

Получение изображения

Для отримання изображения система вызовет збережену процедуру a2.[Product.Image.Load], передав ей в качестве параметра @Id идентификатор об'єкта Image. Им'я Image в наименовании процедури соответствует имени властивості. Токен доступа буде проверен на сервере (в процедуру он не передається). Якщо он не соответствует правилам, то процедура вообще не вызовется.

Загрузка изображения

При завантаженні изображения на сервер, система вызовет збережену процедуру a2.[Product.Image.Update]. Процедура вернет идентификатор изображения и новый токен досупа, які будуть обновлены в моделі.
