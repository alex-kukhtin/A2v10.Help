Властивості базового елементу

Особливості

Частіше за всі задається тільки одно з властивостей Width або Height. В цьому випадкут картинка буде зберігати пропорции.

Приклад

Елемент керування Image використовується для одновременного показа і завантаження зображення з використанням упрощенного подхода.

Властивість Source повинно посилатися на властивість моделі, которое являє собою об`єкт с двумя полями (Id!!Id і Token!!Token). Ім'я цього властивості буде визначати суфікс для хранимых процедур, які будуть викликатися для отримання або записи реальных байтів изображения.

Властивість Base повинно посилатися на полную модель (ту, яка указана в корені файла model.json).

Получение изображения

Для отримання изображения система вызовет збережену процедуру a2.[Product.Image.Load], передав їй в якості параметра @Id ідентифікатор об'єкта Image. Ім'я Image в наименовании процедури відповідає імені властивості. Токен доступа буде проверен на сервере (в процедуру він не передається). Якщо він не відповідає правилам, то процедура вообще не вызовется.

Загрузка изображения

При завантаженні изображения на сервер, система вызовет збережену процедуру a2.[Product.Image.Update]. Процедура вернет ідентифікатор изображения і новий токен досупа, які будуть обновлены в моделі.
