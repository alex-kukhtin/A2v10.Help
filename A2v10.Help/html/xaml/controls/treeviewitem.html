Елемент TreeViewItem

Всі елементи керування

Являє собою один елемент ієрархічного переліку (дерева) TreeView.

Зверніть увагу, що елемент використовується для роботи з ієрархічними джерелами даних.

Наслідує: UIElement : UIElementBase

Властивість вмісту: Label

Властивості

Ім'я Тип Опис
ItemsSource Array Завжди прив'язка (Bind). Джерело даних для отримання дочірніх елементів.
Label String Властивість вмісту. Текст елементу.
Icon Icon Іконка, яка буде відображатися перед вмістом. Якщо іконка не визначена, то буде відображатися іконка, яка визначена у властивостях IconFolder (для папки) або IconItem (для елементу) батьківського елементу TreeView.
IsFolder Boolean? Завжди прив'язка. Чи є елемент папкою. Використовується для роботи з динамічно заповнюваними деревами.
IsGroup Boolean? Завжди прив'язка. Чи є елемент групою. Використовується для роботи з динамічно заповнюваними деревами.
