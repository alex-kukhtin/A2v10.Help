Элемент TreeViewItem

Все элементы управления

Представляет собой один элемент иерархического списка (дерева) TreeView.

Обратите внимание, что элемент используется для работы с иерархическими источниками данных.

Наследует: UIElement : UIElementBase

Свойство содержимого: Label

Свойства

Имя Тип Описание
ItemsSource Array Всегда привязка (Bind). Источник данных для получения дочерних элементов.
Label String Свойство содержимого. Текст элемента.
Icon Icon Иконка, которая будет отображаться перед содержимым. Если иконка не определена, то будет отображаться иконка, которая определена в свойствах IconFolder (для папки) или IconItem (для элемента) родительского элемента TreeView.
IsFolder Boolean? Всегда привязка. Является ли элемент папкой. Используется для работы с динамически заполняемыми деревьями.
IsGroup Boolean? Всегда привязка. Является ли элемент группой. Используется для работы с динамически заполняемыми деревьями.
