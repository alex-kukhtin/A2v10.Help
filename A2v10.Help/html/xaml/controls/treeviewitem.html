Елемент TreeViewItem

Все елементи керування

Являє собою один елемент иерархического списка (дерева) TreeView.

Зверніть увагу, що елемент використовується для роботи с иерархическими джерелами даних.

Наследует: UIElement : UIElementBase

Властивість вмісту: Label

Властивості

Ім'я Тип Опис
ItemsSource Array Завжди прив'язка (Bind). Джерело даних для отримання дочірніх елементів.
Label String Властивість вмісту. Текст елементу.
Icon Icon Иконка, яка буде отображаться перед содержимым. Якщо иконка не определена, то буде отображаться иконка, яка определена в властивостіх IconFolder (для папки) або IconItem (для елементу) батьківського елементу TreeView.
IsFolder Boolean? Завжди привязка. Чи є елемент папкой. Використовується для роботи с динамически заполняемыми деревами.
IsGroup Boolean? Завжди привязка. Чи є елемент группой. Використовується для роботи с динамически заполняемыми деревами.
