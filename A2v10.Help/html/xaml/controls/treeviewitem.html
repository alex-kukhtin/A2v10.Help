Елемент TreeViewItem

Все елементи керування

Представляет собой один елемент иерархического списка (дерева) TreeView.

Зверніть увагу, что елемент використовується для роботи с иерархическими джерелами даних.

Наследует: UIElement : UIElementBase

Властивість содержимого: Label

Властивості

Им'я Тип Опис
ItemsSource Array Всегда привязка (Bind). Джерело даних для отримання дочерних елементів.
Label String Властивість содержимого. Текст елементу.
Icon Icon Иконка, яка буде отображаться перед содержимым. Якщо иконка не определена, то буде отображаться иконка, яка определена в властивостіх IconFolder (для папки) або IconItem (для елементу) батьківського елементу TreeView.
IsFolder Boolean? Всегда привязка. Чи є елемент папкой. Використовується для роботи с динамически заполняемыми деревьями.
IsGroup Boolean? Всегда привязка. Чи є елемент группой. Використовується для роботи с динамически заполняемыми деревьями.
