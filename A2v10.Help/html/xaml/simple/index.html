Простые элементы

Простые элементы разметки, как правило, поддерживают конвертирование из строки. Это позволяет значительно сократить код разметки.

Простые элементы

  • Length - длина.
  • Size - размер.
  • Thickness - размер прямоугольной области.
  • GapSize - промежуток.
  • PrintPage - описание печатной страницы.
  • Accel - акселератор (горячая клавиша).
  • AccelCommand - команда акселератора.

Коллекции

  • AccelCommandCollection - коллекция акселераторов.
  • UIElementCollection - коллекция элементов.
  • ResourceDictionary - словарь ресурсов.
