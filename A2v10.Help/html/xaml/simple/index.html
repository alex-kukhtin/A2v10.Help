Прості елементи

Прості елементи розмітки, як правило, підтримують конвертування з рядку. Це дозволяє значно скоротити код розмітки.

Прості елементи

  • Length - довжина.
  • Size - розмір.
  • Thickness - розмір прясокутної області.
  • GapSize - проміжок.
  • PrintPage - опис друкованої сторінки.
  • Accel - акселератор (гаряча клавіша).
  • AccelCommand - команда акселератора.

Коллекции

  • AccelCommandCollection - колекція акселераторів.
  • UIElementCollection - колекція елементів.
  • ResourceDictionary - словник ресурсів.
