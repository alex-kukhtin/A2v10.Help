Простые елементи

Простые елементи розмітки, як правило, підтримують конвертирование з строки. Це дозволяє значительно сократить код розмітки.

Простые елементи

  • Length - длина.
  • Size - размер.
  • Thickness - размер прямоугольной области.
  • GapSize - промежуток.
  • PrintPage - опис печатной страницы.
  • Accel - акселератор (горячая клавиша).
  • AccelCommand - команда акселератора.

Коллекции

  • AccelCommandCollection - коллекция акселераторов.
  • UIElementCollection - коллекция елементів.
  • ResourceDictionary - словник ресурсов.
