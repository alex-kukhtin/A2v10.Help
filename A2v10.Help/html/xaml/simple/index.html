Простые елементи

Простые елементи розмітки, як правило, підтримують конвертирование з рядки. Це дозволяє значительно сократить код розмітки.

Простые елементи

  • Length - длина.
  • Size - размер.
  • Thickness - розмір прямоугольной области.
  • GapSize - промежуток.
  • PrintPage - опис печатной сторінки.
  • Accel - акселератор (горячая клавиша).
  • AccelCommand - команда акселератора.

Коллекции

  • AccelCommandCollection - колекція акселераторов.
  • UIElementCollection - колекція елементів.
  • ResourceDictionary - словник ресурсов.
