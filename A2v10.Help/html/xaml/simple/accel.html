Елемент Accel

Всі прості елементи

Представляет собой описание акселератора ("горячей" клавиши).

Використовується: TextBox, AccelCommand.

Властивості

Им'я Тип Опис
Key String Акселератор ("горячая" клавиша). Задається у вигляді наименования клавиши і необязательных (одного або нескольких) модификаторов. Модификаторы і названия клавиш разделяются знаком "+".

Названия клавиш

Единица Опис
A..Z, 0..9 Алфавитно-цифровые клавиши.
F0..F12 Функциональные клавиши.
Left, Right, Up, Down Клавиши со стрелками.
PageUp, PageDown, Insert, Delete,
Enter, Escape, Tab, Home, End,
Backspace, Delete		 Управляющие клавиши.

Модификаторы

Единица Опис
Control, Ctrl Клавиша Control.
Shift Клавиша Shift.
Alt Клавиша Alt.
Meta Windows - клавиша ⊞ Windows.
Mac - клавиша ⌘ Command.
Зверніть увагу, може работать не во всех браузерах.

Конвертор

Підтримується преобразование з текстовой строки. Рядок буде просто значенням властивості Key.
