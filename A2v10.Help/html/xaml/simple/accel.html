Элемент Accel

Все простые элементы

Представляет собой описание акселератора ("горячей" клавиши).

Используется: TextBox, AccelCommand.

Свойства

Имя Тип Описание
Key String Акселератор ("горячая" клавиша). Задается в виде наименования клавиши и необязательных (одного или нескольких) модификаторов. Модификаторы и названия клавиш разделяются знаком "+".

Названия клавиш

Единица Описание
A..Z, 0..9 Алфавитно-цифровые клавиши.
F0..F12 Функциональные клавиши.
Left, Right, Up, Down Клавиши со стрелками.
PageUp, PageDown, Insert, Delete,
Enter, Escape, Tab, Home, End,
Backspace, Delete		 Управляющие клавиши.

Модификаторы

Единица Описание
Control, Ctrl Клавиша Control.
Shift Клавиша Shift.
Alt Клавиша Alt.
Meta Windows - клавиша ⊞ Windows.
Mac - клавиша ⌘ Command.
Обратите внимание, может работать не во всех браузерах.

Конвертор

Поддерживается преобразование из текстовой строки. Строка будет просто значением свойства Key.
