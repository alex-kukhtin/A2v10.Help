Елемент Accel

Всі прості елементи

Являє собою опис акселератора ("гарячої" клавіші).

Використовується: TextBox, AccelCommand.

Властивості

Ім'я Тип Опис
Key String Акселератор ("гаряча" клавіша). Задається у вигляді найменування клавіші і необов'язкових (одного або кількох) модифікаторов. Модифікатори і назви клавіш розділяются знаком "+".

Названия клавіш

Одиниця Опис
A..Z, 0..9 Алфавітно-цифрові клавіші.
F0..F12 Функціональні клавіші.
Left, Right, Up, Down Клавіші зі стрілками.
PageUp, PageDown, Insert, Delete,
Enter, Escape, Tab, Home, End,
Backspace, Delete		 Керівні клавіші.

Модифікатори

Одиниця Опис
Control, Ctrl Клавіша Control.
Shift Клавіша Shift.
Alt Клавіша Alt.
Meta Windows - клавіша ⊞ Windows.
Mac - клавіша ⌘ Command.
Зверніть увагу, може працювати не у всіх браузерах.

Конвертор

Підтримується перетворення з текстового рядку. Рядок буде просто значенням властивості Key.
