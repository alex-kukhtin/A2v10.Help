Елемент Accel

Всі прості елементи

Являє собою опис акселератора ("горячей" клавіші).

Використовується: TextBox, AccelCommand.

Властивості

Ім'я Тип Опис
Key String Акселератор ("гаряча" клавіша). Задається у вигляді наименования клавіші і необов'язкових (одного або кількох) модифікаторов. Модификаторы і названия клавиш розділяются знаком "+".

Названия клавиш

Единица Опис
A..Z, 0..9 Алфавитно-цифровые клавіші.
F0..F12 Функциональные клавіші.
Left, Right, Up, Down Клавиши со стрелками.
PageUp, PageDown, Insert, Delete,
Enter, Escape, Tab, Home, End,
Backspace, Delete		 Управляющие клавіші.

Модификаторы

Единица Опис
Control, Ctrl Клавиша Control.
Shift Клавиша Shift.
Alt Клавиша Alt.
Meta Windows - клавіша ⊞ Windows.
Mac - клавіша ⌘ Command.
Зверніть увагу, може працювати не во всех браузерах.

Конвертор

Підтримується перетворення з текстовой рядки. Рядок буде просто значенням властивості Key.
