Елемент AccelCommand

Всі прості елементи

Являє собою об'єкт, связывающий команду і акселератор ("горячую" клавишу). При нажатии этой клавиши виконується команда.

Властивості

Ім'я Тип Опис
Accel Accel Акселератор ("горячая" клавиша).
Command BindCmd Команда. Она буде виконана при нажатии клавиши акселератора.
