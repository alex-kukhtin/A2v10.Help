Элемент AccelCommand

Все простые элементы

Представляет собой объект, связывающий команду и акселератор ("горячую" клавишу). При нажатии этой клавиши выполняется команда.

Свойства

Имя Тип Описание
Accel Accel Акселератор ("горячая" клавиша).
Command BindCmd Команда. Она будет выполнена при нажатии клавиши акселератора.
