Элемент Size

Все простые элементы

Задает размер по горизонтали (Width) и вертикали (Height). Хотя элемент представляет собой объект, чаше всего значение задается с помощью конвертора из текстовой строки.

Свойства

Имя Тип Описание
Width Length Ширина.
Height Length Высота.

Конвертор

Поддерживается преобразование из текстовой строки. Значения ширины и высоты указываются через запятую. Если указано только одно значение, то оно будет использовано и как ширина и как высота.
