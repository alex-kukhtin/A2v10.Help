Элемент GapSize

Все простые элементы

Задает промежуток по основной (X) и дополнительной (Y) оси. Основная и дополнительная ось могут меняться местами в зависимости от ориентации компонента. Хотя элемент представляет собой объект, чаще всего значение задается с помощью конвертора из текстовой строки.

Свойства

Имя Тип Описание
X Length Промежуток по основной оси.
Y Length Промежуток по дополнительной оси.

Конвертор

Поддерживается преобразование из текстовой строки. Значения X и Y указываются через запятую. Если указано только одно значение, то оно будет использовано и как X и как Y.
