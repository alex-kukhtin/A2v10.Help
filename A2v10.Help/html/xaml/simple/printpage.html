Элемент PrintPage

Все простые элементы

Представляет собой правила формирования печатной страницы.

Свойства

Имя Тип Описание
Orientation PageOrientation Ориентация страницы. Возможные значения:
  • Portrait - по умолчанию. Портретная.
  • Landscape - ландшафтная.
Margin Thickness Поля для страницы.
Zoom ZoomValue Масштаб. Возможные значения:
  • Auto - Автоматический. Система пытается напечатать содержимое на одной странице.
  • Значение - Значение. 1 - 100%.
