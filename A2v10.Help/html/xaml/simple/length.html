Елемент Length

Всі прості елементи

Задает довжину. Задається у вигляді числа з необов'язковім указанием единиц измерения або у вигляді кількох буквальных значень. Единицы измерения відповідають единицам в css.

Единицы измерения

Единица Опис
% Проценты.
px Пиксели (1px = 1/96in). Це значення за замовчанням, використовується якщо единица измерения не задана.
mm Миллиметры.
cm Сантиметры.
pt Пункты (1pt = 1/72in).
in Дюймы (1in = 2.54cm).
ch Символы (використовується ширина символа "0").
em Ширина символа "m" поточного шрифта.
rem Ширина символа "m" шрифта кореневого елементу (того, который задан в теге <html>).
ex Высота x-height поточного шрифта.
vh 1% від высоты вікна (viewport).
vw 1% від ширины вікна (viewport).
vmin Меньшее з vh, vw.
vmin Большее з vh, vw.

Буквальные значения

Значення Опис
Auto Автоматически. Соответствует css значению auto.
Fit Минимальное значення вмісту. Соответствует css значению fit-content.
