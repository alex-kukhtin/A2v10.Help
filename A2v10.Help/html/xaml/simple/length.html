Элемент Length

Все простые элементы

Задает длину. Задается в виде числа с необязательным указанием единиц измерения либо в виде нескольких буквальных значений. Единицы измерения соответствуют единицам в css.

Единицы измерения

Единица Описание
% Проценты.
px Пиксели (1px = 1/96in). Это значение по умолчанию, используется если единица измерения не задана.
mm Миллиметры.
cm Сантиметры.
pt Пункты (1pt = 1/72in).
in Дюймы (1in = 2.54cm).
ch Символы (используется ширина символа "0").
em Ширина символа "m" текущего шрифта.
rem Ширина символа "m" шрифта корневого элемента (того, который задан в теге <html>).
ex Высота x-height текущего шрифта.
vh 1% от высоты окна (viewport).
vw 1% от ширины окна (viewport).
vmin Меньшее из vh, vw.
vmin Большее из vh, vw.

Буквальные значения

Значение Описание
Auto Автоматически. Соответствует css значению auto.
Fit Минимальное значение содержимого. Соответствует css значению fit-content.
