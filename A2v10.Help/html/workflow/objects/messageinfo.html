Об'єкт MessageInfo

Все об'єкти

Представляет собой описание повідомлення.

Використовується общая подсистема роботи с повідомленнями.

Використовується в: Request, SendMessage

Властивості (C#)

Им'я Тип Опис
Template String Им'я файла шаблона. Им'я файла задається відносно кореня прикладного програми.
Key String Ключ повідомлення в шаблоне.
TargetId Int64 Идентификатор моделі даних. Сама модель (якщо потрібно) задається в шаблоне повідомлення.
Parameters Dictionary<String, Object> Параметри повідомлення. Для более удобной роботи есть статический метод CreateParameters

Методи (C#)

Опис

Создает словарь з произвольного об'єкта (в том числе анонимного). Імена властивостей становятся ключами, а значення - значениями.
