Объект MessageInfo

Все объекты

Представляет собой описание сообщения.

Используется общая подсистема работы с сообщениями.

Используется в: Request, SendMessage

Свойства (C#)

Имя Тип Описание
Template String Имя файла шаблона. Имя файла задается относительно корня прикладного приложения.
Key String Ключ сообщения в шаблоне.
TargetId Int64 Идентификатор модели данных. Сама модель (если нужно) задается в шаблоне сообщения.
Parameters Dictionary<String, Object> Параметры сообщения. Для более удобной работы есть статический метод CreateParameters

Методы (C#)

Описание

Создает словарь из произвольного объекта (в том числе анонимного). Имена свойств становятся ключами, а значения - значениями.
