Об'єкт MessageInfo

Всі об'єкти

Являє собою опис повідомлення.

Використовується загальна підсистема роботи з повідомленнями.

Використовується в: Request, SendMessage

Властивості (C#)

Ім'я Тип Опис
Template String Ім'я файлу шаблону. Ім'я файла задається відносно кореня прикладної програми.
Key String Ключ повідомлення в шаблоні.
TargetId Int64 Ідентифікатор моделі даних. Сама модель (якщо потрібно) задається в шаблоні повідомлення.
Parameters Dictionary<String, Object> Параметри повідомлення. Для більш зручної роботи є статичний метод CreateParameters

Методи (C#)

Опис

Створює словник з довільного об'єкта (в тому числі анонімного). Імена властивостей стають ключами, а значення - значеннями.
