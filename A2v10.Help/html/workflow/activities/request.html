Активність Request

Всі активності

Зупиняє процес і встановлює закладку. Ім'я закладки (задається в об'єкті Inbox) використовується для продовження процесу з цієї точки зупинки.

Властивості

Ім'я Тип Опис
Inbox InArgument<Inbox> Опис задачі.
TrackBefore InArgument<TrackRecord> Запис до протоколу процесу перед призупинкою процесу.
TrackAfter InArgument<TrackRecord> Запис до протоколу процесу після поновлення процесу.
SendBefore InArgument<MessageInfo> Повідомлення перед призупинкою процесу.
SendAfter InArgument<MessageInfo> Повідомлення після поновлення процесу.
