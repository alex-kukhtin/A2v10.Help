Активность Request

Все активности

Останавливает процесс и устанавливает закладку. Им'я закладки (задається в об'єкте Inbox) використовується для продолжения процесса з цією точки останова.

Властивості

Им'я Тип Опис
Inbox InArgument<Inbox> Опис задачи.
TrackBefore InArgument<TrackRecord> Запись в протокол процесса перед приостановкой процесса.
TrackAfter InArgument<TrackRecord> Запись в протокол процесса после возоновлення процесса.
SendBefore InArgument<MessageInfo> Сообщение перед приостановкой процесса.
SendAfter InArgument<MessageInfo> Сообщение после возоновлення процесса.
