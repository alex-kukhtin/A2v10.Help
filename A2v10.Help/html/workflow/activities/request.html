Активность Request

Все активности

Останавливает процесс і устанавливает закладку. Им'я закладки (задається в об'єкті Inbox) використовується для продолжения процесу з цією точки останова.

Властивості

Им'я Тип Опис
Inbox InArgument<Inbox> Опис задачи.
TrackBefore InArgument<TrackRecord> Запись в протокол процесу перед приостановкой процесу.
TrackAfter InArgument<TrackRecord> Запись в протокол процесу після возоновлення процесу.
SendBefore InArgument<MessageInfo> Повідомлення перед приостановкой процесу.
SendAfter InArgument<MessageInfo> Повідомлення після возоновлення процесу.
