Активность Request

Все активности

Останавливает процесс и устанавливает закладку. Имя закладки (задается в объекте Inbox) используется для продолжения процесса с этой точки останова.

Свойства

Имя Тип Описание
Inbox InArgument<Inbox> Описание задачи.
TrackBefore InArgument<TrackRecord> Запись в протокол процесса перед приостановкой процесса.
TrackAfter InArgument<TrackRecord> Запись в протокол процесса после возобновления процесса.
SendBefore InArgument<MessageInfo> Сообщение перед приостановкой процесса.
SendAfter InArgument<MessageInfo> Сообщение после возобновления процесса.
