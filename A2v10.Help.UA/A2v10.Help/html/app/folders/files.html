Спеціальна папка _files

Спеціальна папка _files містить довільні файли для завантаження користувачами.

Зазвичай для завантаження використовується комада Download.

Приклад завантаження файла

Зверніть увагу! Папка _files може містити вкладені папки. В команді шлях зазначається відносно цієї папки.
