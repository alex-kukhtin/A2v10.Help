Властивості и методы кореневого об'єкта

Модель на клиенте

Корневой об`єкт расширяется следующими служебными властивостіми и методами:

Властивості

readonly $ready: boolean

true, якщо в данный момент НЕ виконується жодних запитов к серверу. Может використовуватися для запрета/разрешения елементів интерфейса при выполнении серверных команд. 

readonly $template: Template

Ссылка на об`єкт шаблона (Template).

Методы

$defer(handler:()=>any): void

Выполняет функцию (handler) после завершения поточного цикла событий.
