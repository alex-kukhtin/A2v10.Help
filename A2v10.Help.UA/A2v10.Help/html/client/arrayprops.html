Властивості и методы масива

Модель на клиенте

Каждый массив расширяется следующими дополнительными служебными властивостіми и методами:

Каждый елемент в массиве реализует інтерфейс IArrayElement и містить неякі додаткові властивості и методы.

Властивості

readonly $root: IRoot

Ссылка на корневой об`єкт моделі (IRoot). 

readonly $parent: IElement

Ссылка на родительский об`єкт масива (IElement). 

readonly $ctrl: IController

Ссылка на об`єкт контроллера (IController). 

readonly $hasSelected: boolean

Есть ли в массиве выделенный елемент (тот, у якого властивість $selected установлено в true). 

readonly $selectedIndex: number

Индекс выделенного елементу або -1, якщо нет выделенного елементу. 

readonly $checked: IElementArray<T>

Массив отмеченных ($checked = true) елементів поточного масива. 

readonly $cross: { [prop: string]: string[] }

Об'єкт ключей для перекрестных моделей даних. Возвращает об`єкт с іменами властивостей, соответствующими перекрестным полям. Значеннями является массив строк - перекрестных ключей.

Методы

Опис

Перезагружает массив з базы даних. Має сенс тільки для дочерних и "ленивых" массивов.

Возвращает:

Удаляет заданный елемент з масива.

Аргументы:

  • elem - удаляемый елемент.

Возвращает:

  • Ссылку на сам массив.

Опис

Сбрасывает признак выделения со всех елементів масива.

Возвращает:

  • Ссылку на сам массив.

Опис

Вставляет елемент в конец масива. Якщо джерело указан и является об'єктом, то його властивості копируются во вновь вставленный елемент.

Аргументы:

  • src - джерело для вставки. Може бути обычным (не реактивным) об'єктом javascript.

Возвращает:

Опис

Вставляет елемент в начало масива. Якщо джерело указан и является об'єктом, то його властивості копируются во вновь вставленный елемент.

Аргументы:

  • src - джерело для вставки. Може бути обычным (не реактивным) об'єктом javascript.

Возвращает:

  • Вставленный елемент.

Опис

Вставляет в массив новый елемент.

Аргументы:

  • src - джерело. Може бути обычным (не реактивным) об'єктом javascript.
  • to - место вставки. Можливі значения:
    • 'start' - в начало масива (эквивалент $prepend).
    • 'end' - в конец масива (эквивалент $append).
    • 'above' - перед указанным (ref) елементом.
    • 'below' - после указанного (ref) елементу.
  • ref - Елемент, відносно якого буде виконуватися вставка для to = 'above' | 'below'.

Возвращает:

  • Вставленный елемент.

Опис

Копирует джерело в текущий массив.

Аргументы:

  • src - джерело для копіювання. Може бути массивом обычных (не реактивных) об'єктов javascript. Властивості джерела, которых нет в елементе игнорируются.

Возвращает:

  • Сам массив.

Опис

Вычисляет сумму для елементів масива.

Аргументы:

  • fn - функция обратного вызова, яка повинна повернути властивість елементу масива для суммирования.

Возвращает:

  • Вычисленную сумму.

Опис

Ищет в массиве первый подходящий елемент. Функция отличается від обычного поиска в массиве тем, что для иерархических массивов она виконує рекурсивный поиск по дереву.

Аргументы:

  • fn - функция обратного вызова, яка повинна повернути true, якщо елемент подходит по критериям выборки. Аргументы функции:
    • this? - значення аргумента thisArg.
    • item - елемент масива.
    • index? - индекс елементу.
    • array? - массив, в которором виконується поиск.
  • thisArg - значення аргумента this для функции обратного вызова

Возвращает:

  • Найденный елемент.

Опис

Чи є массив "ленивым".

Возвращает:

  • true, якщо массив "ленивый" (с отложенной загрузкой).

Опис

Сбрасывает признак завантаження "ленивого" масива. При следующем обращении массив заново буде загружен с сервера.
