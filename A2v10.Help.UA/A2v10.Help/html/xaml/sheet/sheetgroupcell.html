Елемент SheetGroupCell

Все елементи таблиц

Представляет собой ячейку для отображения елементів керування группировкой в таблице.

Ячейка не має содержимого, вместо этого она отображает елементи керування группировкой в таблице. Внешний вид зависит від темы пользовательского интерфейса.

Использование такой ячейки має смысл тільки для строк в иерархических секциях SheetTreeSection. Во всех остальных случаях ячейка просто буде пустой.

Наследует: SheetCell : UIContentElement : UIElementBase
