Елемент SheetCellGroup

Все елементи таблиц

Представляет собой коллекцию ячеек таблиці. Может спользоваться в любом месте, где допустимо использование елементу SheetCell.

Властивість содержимого: Cells

Властивості

Им'я Тип Опис
Cells Коллекция SheetCell Коллекция ячеек в строке.
ItemsSource Array Всегда привязка (Bind). Джерело даних для отримання дочерних ячеек.

Опис

Зазвичай этот елемент використовується для совместной роботи с перекрестными моделями даних.
