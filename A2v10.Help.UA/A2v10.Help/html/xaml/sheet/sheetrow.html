Елемент SheetRow

Все елементи таблиц

Представляет собой строку таблиці.

Наследует: UIElement : UIElementBase

Властивість содержимого: Cells

Властивості

Им'я Тип Опис
Cells Коллекция SheetCell Коллекция ячеек в строке.
RowStyle RowStyle Стиль отображения строки. Можливі значения
  • Default - (за замовчанням) обычная строка.
  • Title - Заголовок звіта.
  • Parameter - Параметр звіта.
  • LastParameter - Последний параметр в списке параметров.
  • Header - Рядок заголовка.
  • LightHeader - Рядок заголовка другого стиля.
  • Footer - Рядок подвала.
  • Total - Рядок итогов.
  • NoBorder - Рядок с ячейками без рамок.
Align TextAlign? Выравнивание по горизонтали для всех ячеек строки. Може бути переопределено в каждой ячейке индивидуально.
Mark Object Тільки привязка. Маркировка строки. Подробности ниже.
Властивості базового елементу

Маркировка строк

Выражение привязки для властивості Mark должно возвращать текстовую строку, яка являє собою дополнительный css-класс, добавляемый к строке. В теме за замовчанням поддерживаются наступні варианты маркировки строк:

"danger", "error", "red" Содержимое строки
"warning", "orange" Содержимое строки
"success", "green" Содержимое строки
"info", "cyan" Содержимое строки
"gray" Содержимое строки

Зверніть увагу! Маркировка просто добавляе css-класс к строке. Это означает, что вы не ограничены тільки указанными вариантами. Например, якщо властивість привязки вернет "danger bold", то строка буде маркирована красным фоном и буде показываться полужирным шрифтом.
