Елемент List

Все елементи керування

Представляет собой список елементів (c можливістью выделения). Содержимое обычно являє собою список елементів ListItem.

Наследует: Control : UIElement : UIElementBase

Властивість содержимого: Content

Властивості

Им'я Тип Опис
Content UIElementCollection Властивість содержимого. Содержимое списка
ItemsSource Array Всегда привязка - Bind. Джерело даних для отображения в списке.
Style ListStyle Стиль отображения списка.
Background BackgroundStyle Стиль фона для списка.
Border Boolean Отображать ли рамку вокруг списка.
BorderStyle BorderStyle Стиль рамки вокруг елементу.
Striped Boolean Раскраска списка "в полоску" с выделением четных и нечентых елементів.
Flush Boolean Убирает линию после последнйого елементу в списке (тільки для стиля Underlined).
AutoSelect AutoSelectMode Автоматически выделять елемент после завантаження або оновлення списка.
Height Length Высота списка.
MaxHeight Length Максимальная высота списка. Якщо список больше, появляется вертикальный скроллер.
Select Boolean? Разрешено ли выделение елементів в списке.
Mark Object Тільки привязка (Bind). Маркировка елементу списка. Маркер являє собою вертикальную линию слева від елементу. Подробности ниже.
MarkerStyle RowMarkerStyle Стиль маркировки елементу в списке. Можливі значения:
  • None - (за замовчанням) нет маркировки.
  • Row - фоновый цвет.
  • Marker - маркер у вигляді полоски слева.
  • Both - оба вида маркировки.
EmptyPanel UIElement Панель, яка отображается, якщо содержимое пустое. Зазвичай являє собою об'єкт EmptyPanel.
DoubleClick BindCmd Команда, яка буде выполнена при двойном щелчке на елементе списка. Не забудьте передать в команду аргумент.

Маркировка елементів списка
