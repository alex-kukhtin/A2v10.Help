Простые елементи

Простые елементи розмітки, как правило, поддерживают конвертирование з строки. Это дозволяє значительно сократить код розмітки.

Простые елементи

  • Length - длина.
  • Size - размер.
  • Thickness - размер прямоугольной области.
  • GapSize - промежуток.
  • PrintPage - описание печатной страницы.
  • Accel - акселератор (горячая клавиша).
  • AccelCommand - команда акселератора.

Коллекции

  • AccelCommandCollection - коллекция акселераторов.
  • UIElementCollection - коллекция елементів.
  • ResourceDictionary - словарь ресурсов.
