Елемент AccelCommand

Всі прості елементи

Представляет собой об'єкт, связывающий команду и акселератор ("горячую" клавишу). При нажатии этой клавиши виконується команда.

Властивості

Им'я Тип Опис
Accel Accel Акселератор ("горячая" клавиша).
Command BindCmd Команда. Она буде выполнена при нажатии клавиши акселератора.
