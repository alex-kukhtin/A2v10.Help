Елемент Page

Все контейнеры

Представляет собой страницу.

Страница має свой Url адрес и попадает в контекст переходов бразуера.

Наследует: RootContainer : Container : UIElement : UIElementBase

Властивість содержимого: Children

Властивості

Им'я Тип Опис
Title String Заголовок окна страницы.
Taskpad UIElementBase Панель задач страницы. Частіше за все це елемент Taskpad.
Toolbar UIElementBase Панель инструментов страницы. Частіше за все це елемент Toolbar.
CollectionView CollectionView Представление коллекции, яка буде отображаться на странице. Зазвичай це серверная коллекция. Властивість необходимо щоб вынести управление коллекцией "за дужки" содержимого страницы.
Pager Pager Пейджер для коллекции, яка буде отображаться на странице. Властивість необходимо щоб вынести управление коллекцией "за дужки" содержимого страницы.
Background BackgroundStyle Цвет фона страницы.
