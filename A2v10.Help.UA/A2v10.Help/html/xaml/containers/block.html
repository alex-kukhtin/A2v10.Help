Елемент Block

Все контейнеры

Представляет собой контейнер, які містить другие елементи керування або другие контейнеры.

Наследует: UIElement : UIElementBase

Властивість содержимого: Children

Властивості

Им'я Тип Опис
Children UIElementCollection Властивість содержимого. Коллекция дочерних елементів.
Align TextAlign Горизонтальное выравнивание содержимого контейнера.
Color TextColor Цвет текста.
Background BackgroundStyle Цвет фона.
Height Length Высота блока.
Width Length Ширина блока. За замовчанням блок занимает 100% ширины батьківського елементу.
MaxWidth Length Максимальная ширина блока. Якщо содержимое шире, то оно переносится.
Border Boolean Отображать рамку для блока.
Scroll Boolean? Допускает ли блок прокрутку. Має сенс тільки, якщо установлена высота блока. Якщо установлено в True, то буде появляться полоса прокрутки. Якщо установлено в False, то лишнее содержимое буде обрезано. За замовчанням высота блока буде такой, щоб включать все вложенные елементи.
DropShadow ShadowStyle Вид тени.
Relative Boolean Относительное позиционирование для блока. Все дочерние елементи с установленным властивістю Absolute будуть позиционироваться відносно этого блока.
