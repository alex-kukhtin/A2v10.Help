Шаблони. Опции

Опции шаблона задают поведение всей моделі.

Синтаксис

Властивості об'єкта options

  • noDirty - не взводить флаг изменений $isDirty при изменениях моделі даних.
  • bindOnce - массив рядків, определяющий какие корневые властивості моделі будуть связываться однократно.
  • persistSelect - массив рядків, определяющий массивы моделі, для которых потрібно сохранять позицию выделенного елементу в процессе сохранения або перезавантаження моделі.
  • skipDirty - массив рядків, определяющий пути к властивостім моделі, при изменении которых не потрібно взводить флаг изменений $isDirty.

Подробнее

Флаг $isDirty відповідає за вывод повідомлення о необходимости зберігти модель после внесения изменений, а также за поведение некоторых елементів керування. Например кнопка, связанная с командой Save буде автоматически разрешаться после изменения моделі. Якщо опция noDirty установлена, то флаг $isDirty взводиться не буде.

Властивість bindOnce дозволяє визначити корневые властивості моделі, які будуть связываться с данными тільки при первой завантаженні моделі. При дальнейшей перезавантаженні (наприклад командой Reload або после сохранения моделі) перепривязки этого властивості не буде. Частіше за все це використовується для привязки елементів справочников.

Зверніть увагу! В массиве bindOnce можна указывать тільки корневые властивості моделі (то есть тільки те, в іменах которых нет точки).

Властивість persistSelect дозволяє визначити массивы в моделі, для которых буде сохраняться позиция выделенного елементу ($selected). Позиция буде сохраняться при сохранении моделі (наприклад командой Save) и при ее перезавантаженні (наприклад командой Reload).
