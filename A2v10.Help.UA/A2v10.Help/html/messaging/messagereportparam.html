Елемент MessageReportParam

Надсилання сообщений

Представляет собой параметр для моделі даних звіта.

Коллекция параметров являє собою словарь с текстовым ключом. Им'я параметра задається атрибутом x:Key

Властивості

Им'я Тип Опис
Value Object Зазвичай привязка. Значення параметра. Зверніть увагу, что привязка работает відносно основоной моделі повідомлення (не моделі звіта!).
