Платформа

Система A2v10 реализована з використанням технологического стека .NET Framework корпорации Microsoft.

В качестве СУБД використовується Microsoft SQL-Server або облачный сервис Azure SQL Database (service).

Серверна часть системы реализована как web-приложение на платформе Microsoft ASP.NET.MVC. Підтримується развертываение системы в облачном сервисе Microsoft Azure как Web App Service.

Для реализации клиентской части web-програми використовується JavaScript фреймворк с открытым исходным кодом Vue.js. Клиентская часть реализована как SPA-приложение (Single Page Application). Поддерживаются все современные браузеры (Edge, Chrome, FireFox, Safari). Браузер Internet Expolorer НЕ підтримуєся.

Для построения звітів и печатных форм применяется звітная система Stimulsoft Reports.Web.

Зверніть увагу! Библиотеки компании Stimulsoft НЕ входят в состав платформи. Для роботи с звітами требуется приобрести отдельную лицензию.
